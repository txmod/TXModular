Using Wii Controllers


In order to use one or more Wii controller(s) with the TX Modular system, you need other software to connect to the Wii and send the Wii data to the TX Modular.

Currently the software you can use is:


1) DarwiinRemote OSC, which is available from:

http://code.google.com/p/darwiinosc/downloads/list

DarwiinRemote OSC is Open Source.

It can get data from only one Wii controller at any one time.

Nunchuck and IR camera data are read as well.

N.B. Every time you start DarwiinRemote OSC, you need to go into the Preferences and set the Port. Usually this should be set to 57120, but occasionally this can change: To see which Port the TX Modular System is currently using, go to the Notes page (use the Notes button near the top of the screen).


2) OSCulator, which is available from:

http://www.osculator.net/wiki/

OSCulator is Floatware and costs a minimum of $19. (You can download a trial version which stops working for 15 seconds every so often).

It can get data from up to 8 Wii controllers at the same time.

Nunchuck and IR camera data are read as well.

You will find a folder included with the TX Modular software called "TX OSCulator Files".

Inside this there a file called "OSCulator_WiiToTX".

When you start OSCulator, "OSCulator_WiiToTX" needs to be opened so that data gets sent to the TX Modular (on port 57120).


N.B. Don't forget to turn Bluetooth on on your computer before trying to connect the Wii !


(All the above info is current on 8 October 2008)
