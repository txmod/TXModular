Interference St


Module type: Audio Insert (stereo input, stereo output)


Introduction:

This module adds 'interference' to the input signal by using up to 3 regular or noisy waveforms to modulate a short delay line. The modulation depth and frequency of each waveform can also be modulated.



Controls:

Wave 1/2/3 - Waveform to use

Wave 1/2/3 Depth - how much interference is contributed

Wave 1/2/3 Freq - the frequency of the interference

Out Gain - the output level of the effect

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.
