Vocoder FX


Module type: Audio Insert (mono input, mono output)


Introduction:

This module applies the frequency spectrum of one mono audio signal (called the Modulator) onto another mono audio signal (called the Carrier). It also allows the spectrum to be stretched and shifted.

This module is an audio insert module which should be added to the channel of the audio signal that is the Carrier.

Then on another audio channel, the destination can be set to the Vocoder module's Modulator bus (see Audio Input Busses below).


See the example file VocoderFX.


Controls:

Stretch - this stretches the spectrum.

Shift - this shifts the spectrum.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Audio Input Busses:

Modulator - mono audio signal whose spectrum is applied to the Carrier audio signal.

Control Modulation Input Busses:

Stretch

Shift

Dry-Wet Mix
