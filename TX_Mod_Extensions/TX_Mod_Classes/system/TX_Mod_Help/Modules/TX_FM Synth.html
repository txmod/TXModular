FM Synth


Module type: Audio Source (mono output, triggered)


Introduction:

This module creates polyphonic synthesised notes using multi-layer Frequency Modulation (FM) Synthesis.

It uses 6 "operators" (which are the equivalent to sine oscillators in FM terminology), which can each modulate the frequency of any of the 6 operators, including themselves (self-modulation).

See help file Frequency Modulation Synthesis for details.

Notes can be triggered in various ways such as by MIDI notes, other modules (e.g. Step Sequencer, Audio Trigger) or action buttons on an interface screen.


Controls:


Operators button

See help file Frequency Modulation Synthesis for details about the operators.

Main level - this sets the overall output level of the synth

Amp Comp - select a preset for the type of amplitude compensation to use - see help file Amp Comp for details. Can be set to off when not needed.


Velocity Curves button

Op 1/Op 2/Op 3/Op 4/Op 5/Op6/Output - buttons to set which curve is being edited.
Each operator has it's own curve, which scales it's amplitude - by default this is flat curve set to 50% amplitude.
Output refers to the overall velocity curve which scales the output level of the synth.

Vel curve - sets a transfer curve which maps the input velocity to new output velocity. See the preset curves for examples.

See help file Curve Editing for details.

Grid rows - no of rows (1-99) separated by white lines in the grid behind the curve.

Grid cols - no of columns (1-99) separated by white lines in the grid behind the curve.


Note Levels button

Op 1/Op 2/Op 3/Op 4/Op 5/Op6 - buttons to set which curve is being edited

Note levels curve - sets the amplitude scaling for each individual note across the keyboard.

Copy curve/ Paste curve - for copying between layers

Smooth curve - evens out spikes or sudden changes in the curve. Useful when curve is drawn by the mouse.

Invert curve - turns the curve upside down.

Randomise curve - randomises the level of every note.

Default curve - sets the level of every note to maximum.


MIDI/ Note button

See help file MIDI Note Button for details.


Envelopes button

There are buttons to select one the following envelopes which are triggered when a note is played:

Volume Env - overall volume envelope for each note played

Operator 1 Env - optional envelope which is added to the amplitude of Operator 1

Operator 2 Env - optional envelope which is added to the amplitude of Operator 2

For Operator 1 Env and Operator 2 Env , the following controls are shown:

OFF/ON Operator 1/2 Env - turns the envelope on/off.

Env amount - sets how much the envelope will affect the amplitude of Operator 1/2. The range is -1 to +1. Negative values will mean the envelope moves the amplitude in a negative direction.

For other envelope controls, see help file Env DADSSR.
