Infinity


Module type: Audio Insert (mono input, stereo output)


Introduction:

This module adds a long, smeared wash to the input signal using multiple all-pass delays. Maximum feedback produces infinite sustain.


Controls:

Reverb type - various presets, with and without modulation

Feedback type - various presets, with and without negative feedback - which emphasises odd harmonics an octave lower.

Density -

Feedback- amount of feedback through the system. A setting of 1.0 produces infinite sustain.

Mod Speed - speed of delay-line modulation.

Mod Depth - depth of delay-line modulation.

Gain Boost

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.
