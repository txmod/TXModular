Changed Trig


Module type: Control Insert


Introduction:

This module analyses it’s input signal and outputs a trigger pulse whenever the value changes.


Controls:

Trigger - manually trigger a pulse

Threshold - the minimum change needed to trigger a pulse (default 0).

Trig Time - how long the trigger pulse will last. No further triggers will be output while the current pulse lasts, so setting this higher will stop fast retriggering.


Control Modulation Input Busses:

Trigger - this allows a modulation signal to trigger a pulse, even when there’s no change detected.
