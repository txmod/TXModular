Phaser St


Module type: Audio Insert (stereo input, stereo output)


Introduction:

This module creates a phasing effect on a stereo audio signal using allpass delay lines, with the phase time being modulated by a low frequency oscillator (LFO).

Note: see also the Notch Phaser module for a phasing effect using notch filters.


Controls:

Phase time - in milliseconds. A longer phase time focuses the phaser at a lower frequency.

Feedback - this intensifies the phaser.

Waveform - selects different kinds of waveforms for the LFO.

LFO rate - the frequency of the LFO.

LFO depth - the amount of modulation. If set to 0, the flange time will stay constant.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.
