Reverb St


Module type: Audio Insert (stereo input, stereo output)


Introduction:

This module adds a basic reverb to a stereo audio signal.


Controls:

Room size - sets the size of the modelled room

Decay - sets the reverb decay time.

Damping freq - the frequency where high frequency damping starts. Set higher for a brighter sounding reverb.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Control Modulation Input Busses:

Room size

Decay

Damping freq

Dry-Wet Mix
