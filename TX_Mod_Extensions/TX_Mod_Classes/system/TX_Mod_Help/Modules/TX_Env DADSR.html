Env DADSR


Module type: Control Source (triggered)


Introduction:

This module creates DADSR (Delay Attack Decay Sustain Release) envelopes as control signals.

They can be triggered in various ways such as by MIDI notes, by other modules (such as Step Sequencer, Audio Trigger) or by action buttons on an interface screen.


Controls:


Envelope button:

Env presets - presets for various standard envelope shapes.

Envelope display box - this shows each stage of the envelope by level (up-down) and time (left-right)..

Pre-Delay - the delay time before the envelope begins.

Attack - the attack time of the envelope

Decay - the decay time of the envelope

Sustain level - the level which the sustain part of the envelope is at.

Sustain time - this is only relevant if the Envelope type is fixed length.

Release - the release time of the envelope

Curve - this sets the shape of the curve.
Some options are for one curve that applies to all parts of the envelope.
Other options give an upwards curve for the attack stage, and a downwards curve for the decay and release stages.

Env. Type - this can be Sustain or Fixed Length. A Sustain envelope will remain at the sustain level until the end of the trigger or midi note. A Fixed Length envelope will remain at the sustain level according to the Sustain time slider.

Scale level to velocity - if this is set on, then the maximum level of the envelope will depend on the velocity of the trigger or midi note; if this is set off, the maximum level will always be100%.


MIDI button:

Midi Listen - this allows the module to be triggered by a midi note.

Midi Chan. - set the midi channels the module will listen to.

Note range - which notes will trigger the module.

Vel range - which velocities will trigger the module.

Polyphony - maximum number of simultaneous envelopes.


Trigger Envelope button - this will trigger an envelope for 1 second whenever it is pressed.


Control Modulation Input Busses:

Pre-Delay

Attack

Decay

Sustain level

Sustain time

Release


Output Busses:

Env Out - the envelope signal

Env End Trig - outputs a trigger when the envelope ends
