Mono env 16 stage


Module type: Control Source (triggered)


Introduction:

This module creates a mono 16-stage envelope as a control signal.

It can be triggered in various ways such as by modulation of the gate input, by other modules (such as an Action Sequencer) or by action buttons on an interface screen.


Controls:

No. Stages - the total number of stages in the envelope (3-16)

Trigger Envelope - this will trigger an envelope for 1 second whenever it is pressed.

Loop stage - which stage the loop stages - this is only relevant when the envelope type is "Sustain Looped".

Release stage - which stage the release part of the envelope begins from - this is only relevant when the envelope type is "Sustain Looped" or "Sustain Unlooped".

Env. Type - this can be "sustain looped", "sustain unlooped" or "fixed length". A "sustain looped" envelope will keep looping from the release stage to the loop stage until the end of the gate. A "sustain unlooped" envelope will remain at the release stage level until the end of the gate. A "fixed length" envelope will play through all its stages according to the stage times.

Envelope display box - this shows each stage of the envelope by level (up-down) and time (left-right). The stages can be moved by dragging them with a mouse.

Levels/Times/Curves - settings for each stage.
Curve types are: Linear, Exponential, Sine, Welch, Step, Slope +10, Slope +9, ... Slope -9, Slope -10

Total Time - for the whole envelope.

Use Curve 1 for all stages - to use one curve for the whole envelope.

Level scale - scales the level of the curve.

Gate - sets the gate to trigger or stop the envelope.

Gate Mode - This controls how the gate affects the envelope .

“Fixed length” means the gate will start the envelope and it will play its whole length. The final envelope value will be left at whatever value is at the end of the curve.

“Forced release” means the gate will start the envelope and it will play its whole length or, if the gate is turned off before then, it will fade the curve to its minimum value using Close time as the fade out time.


Control Modulation Input Busses:

Gate

Gate close time

Level scale


Output Busses:

Env Out - the envelope signal

Env End Trig - outputs a trigger when the envelope ends
