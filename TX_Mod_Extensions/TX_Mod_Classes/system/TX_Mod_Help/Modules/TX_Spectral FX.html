Spectral FX

   

Module type:  Audio Insert (mono input, mono output)


Introduction:

Spectral effects can change a mono audio signal in unique ways. Vocoding is an example of a well-known spectral effect, but there are many more which can be combined for further complexity.

This module uses a DSP process called the fast fourier transform (FFT) which analyzes the frequency spectrum of a signal. 

Once analysed, the signal can then be further processed, using up to 6 different processes (from over 50 choices - see below). 

In addition a side chain input can be used to introduce a second audio signal - this is only used in certain labelled processes. 

After all processing, a final process called the inverse fast fourier transform (IFFT) is run, which turns the analysed signal back into an audio signal (resynthesis).

Further info about FFT: To properly explain FFT and what the settings do is quite complex and technical. See online for further details (e.g. search for FFT or Phase Vocoder, & see:  http://en.wikipedia.org/wiki/Phase_vocoder &  http://www.cycling74.com/story/2006/11/2/113327/823).

Controls:


Settings button

There are various FFT settings which can be set. For best results, use the one of the default settings.  

Default 1 - this sets the FFT settings to one recommended default.

Default 2 - this sets the FFT settings to another recommended default.

(N.B. for best results, it is recommended to use one of these defaults.)

FFT size - sets the number of FFT bins used for analysis(default 512).

FFT window - sets the type of window used for analysis (Sine, Hanning, or Rectangular).

FFT hop - sets the amount of offset from one FFT analysis frame to the next.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal.  For send effects this will usually be set to 1 - i.e. 100% affected signal.


Process 1 button

Process 1 - sets which process will be used. A description of the process will be shown in the box below, along with details of what the parameters Modify 1/2/3 do. 

Modify 1 - sets the Modify 1 control. 

Modify 2 - sets the Modify 2 control. 

Modify 3 - sets the Modify 3 control. 


Process 2/3/4/5/6 buttons - see Process 1 button
