Mix Audio 16-2


Module type: Audio Source (stereo output)


Introduction:

This module mixes 8 stereo audio input signals.

Once the module has been added, it's inputs will become valid destinations for any audio Channel.


Controls:

Level 1+2 ... 15+16 - sets the level of inputs 1..16.

Auto scale (output reduced if mixed level headroom exceeds 1) - automatically scales the mixed level so it is within system limits.

Output Level - sets the output level of the mix.


Audio Input Busses:

Input 1+2 ... 15+16


Control Modulation Input Busses:

Level 1+2 ... 15+16
