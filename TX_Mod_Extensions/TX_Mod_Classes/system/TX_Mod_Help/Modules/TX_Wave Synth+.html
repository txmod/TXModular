Wave Synth+


Module type: Audio Source (mono output, triggered)


Introduction:

This module creates polyphonic synthesised waveform notes. It is like the Wave Synth module, but includes extra features including a filter and various modulation sources, such as LFOs.

Notes can be triggered in various ways such as by MIDI notes, other modules (e.g. Step Sequencer, Audio Trigger) or action buttons on an interface screen.


Controls:


Waveform button

See help file Waveform Synthesis for details.


MIDI/ Note button

See help file MIDI Note Button for details.


Filter button

See help file Filter for details.


Amp Envelope button

This is for the amplitude (volume) envelope for each note.

See help file Env DADSR for details.


Envelope 2 button

This is an envelope that can be used for modulation in the modulation matrix (see below).

See help file Env DADSR for details.


LFO 1/2 button

This is for 2 Low Frequency Oscillators that can be used for modulation in the modulation matrix (see below).

See help file LFO for details.


Mod Matrix button

This is for the modulation matrix which allows you to use various modulation sources to control the synth. The lower half of the window has 10 modulation rows for setting modulation parameters.

Note range - when using the note played as a modulation source, this sets the lowest and highest notes that will be tracked.

Vel range - when using the note velocity as a modulation source, this sets the lowest and highest velocities that will be tracked.

Slider 1/2 - these are free sliders that can be used as modulation sources

Source - sets the source of the modulation . This includes 2 random sources (Random val 1/2) which create random values for every note, and a source called Offset which allows you to set a fixed value of modulation.

Destination - sets the destination of the modulation (i.e. what is being modulated).

Modulation amount - This is a percentage and can be positive or negative from -100% to +100%. A modulation of 0% will have no effect.

Scale - This allow you to use note, velocity, slider1 and slider2 to scale the modulation amount for extra flexibility. If nothing is selected (the default), a scaling of 1 will be used - i.e. the modulation amount set by the slider will by used directly.
