Notch Phaser St


Module type:  Audio Insert (stereo input, stereo output)


Introduction:

This module creates a stereo phasing effect on a stereo audio signal using notch filters, with the filter frequency being modulated by a low frequency oscillator (LFO). 

Note: see also the Phaser module for a phasing effect using allpass delay lines, although there is no stereo version of Phaser.


Controls:

No. layers - the number of filter layers (1-5). Add more to thicken the sound. 

Centre freq - in hertz.  

Resonance -  how focussed the filter is around the centre frequency.

Waveform - selects different kinds of waveforms for the LFO.

LFO rate - the frequency of the LFO. 

LFO depth - the amount of modulation. If set to 0, the flange time will stay constant.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal.  For send effects this will usually be set to 1 - i.e. 100% affected signal.
