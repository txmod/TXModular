ReverbG


Module type: Audio Insert (mono input, mono output)


Introduction:

This module adds reverb to a stereo audio signal (based on GReverb LADSPA effect by Juhana Sadeharju).


Controls:

Presets - use this popup to load a preset. There are 2 versions of each preset, as a send effect or as an insert effect. The difference is that the send effect presets do not include any of the dry signal, only the early reflections and reverb tail.

Room size - sets the size of the modelled room

Reverb time - sets the reverb decay time.

Input damp - the amount of high frequency damping on the input signal. Set higher for a brighter sound.

Reverb damp - the amount of high frequency damping on the reverb signal. Set higher for a brighter sound.

Input level - level of input signal mixed with the early reflections and reverb tail. For send effects this will usually be set to 0.

Early refl's - level of early reflections

Reverb tail - level of reverb tail

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.
