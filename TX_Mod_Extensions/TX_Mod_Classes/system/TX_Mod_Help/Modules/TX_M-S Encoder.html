M-S Encoder


Module type:  Audio Source


Introduction:

This module takes a stereo audio signal and converts into mid and side signals.

Once an M-S Encoder module has been created, it will appear in the list of possible destinations for audio channels. This allows you to send a stereo audio signal to it from another channel.

It outputs mid and side signals.

Also check online for details about M-S techniques; e.g. http://emusician.com/mag/emusic_front_center/, see section Dial M-S for Mastering.


Controls:

Left/Right Level - level of output for each channel.


Audio Input Busses:

Audio in - the incoming stereo audio signal.
