Noise Gate St


Module type: Audio Insert (stereo input, stereo output)


Introduction:

This module can be used as a gate or ducker with an optional side-chain input.

It changes the level of an audio signal coming into it by detecting the level of the control signal (the main input or side-chain).

As a gate it reduces the level of the input signal when the control signal is below a certain threshold. For example it can be used to remove the low level noise in a signal.

As a ducker it reduces the level of the input signal when the control signal is above the threshold. This can be used for example to reduce the output of a send effect such as delay, when the original signal is present (known as "ducking delay" or "ducked delay").

It can optionally use a mono audio side-chain input to control the noise gate, instead of the main input signal. If the side-chain is used, then on another audio channel, the destination can be set to the noise gate module's Side-chain bus (see Audio Input Busses below).

An optional bracket EQ can be added to the control signal to focus on a specific range of frequencies (for example for de-essing, or ducking in response to a kick or bass hit).

A lookahead delay can also be added. Take care to set appropriate open and close times if using a lookahead delay.

The controls Hysteresis and Hold time (see below) can be used to reduce "gate chattering", where the control signal is hovering very close to the threshold and causes fast opening and closing.


Controls:

Control signal - sets whether the main input signal or the side-chain signal will be used to control the noise gate.

Monitor control signal + eq - when set on, this let's you hear the control signal, with the Bracket EQ, which makes it easier to set the EQ range.

Channel link - sets whether the left and right channels are linked together (the default setting) or independent of each other while being gated or ducked.

Bracket EQ - set's the type of EQ used: 12db or 24db, Low-cut, Hi-cut, both or none.

EQ range - sets the low and high frequencies used for the Bracket EQ.

Play mode - whether the module acts as a Gate or Ducker.

Lookahead - a lookahead delay can be added to input signal to allow the gate/ducker to act ahead of time. Set appropriate open and close times if using lookahead delay.

Threshold db - when the level of the control signal goes above the threshold, in Gate mode the gate will open, and in Ducker mode the gate will close.

Hysteresis db - this raises the opening Threshold for the Gate or Ducker.

Reduction db - sets how much the input signal will be reduced when the Gate is off, or the Ducker is on.

Open time ms - sets how long it takes for the Gate/Ducker to increase the audio level.

Hold time ms - sets the minimum time the Gate/Ducker will be on.

Close time ms - sets how long it takes for the Gate/Ducker to reduce the audio level.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Audio Input Busses:

Side-chain - mono audio signal which can be used to control the noise gate.
