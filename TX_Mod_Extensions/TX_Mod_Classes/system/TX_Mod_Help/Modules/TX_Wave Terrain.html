Wave Terrain


Module type: Audio Source (mono output)


Introduction:

This module outputs a synthesised waveform as a mono audio signal.

The synthesis is based on a mathematical model of an object moving over a 3D terrain. The vertical movement created by orbiting around the terrain at an audio frequency creates an output waveform.

The orbit can be varied (using sliders) which generates time varying waveforms. Different mathematical functions can be used to define the shape of the terrain and the orbit that is used to move around it.

The terrain size is 256 x 256 which is 65536 points. This resolution gives quite a lo-fi, aliased sound to the module which is part of it's character! There is an optional filter which can smooth out the signal and tame it's character a bit.

The edit pages for this module include text boxes which show the underlying computer code, written in SuperCollider, which define the Terrain and Orbit used for the synthesis.

If you are not familiar with SuperCollider code, you can ignore the computer code and use the various presets to select different Terrains and Orbits.

The terrain can also be created by importing a PNG image, or by using the Wave Terrain Builder, which is an experimental tool to create and visualise wave terrains.

A monochrome image of the terrain is shown to help visualise its shape.


For further details and technical info about Wave Terrain Synthesis, see:

Controls:


Orbit Controls button

Freq - sets the pitch or frequency of the orbit

Modify 1 - sets a value which can be used to change the orbit

Modify 2 - sets a value which can be used to change the orbit

Modify 3 - sets a value which can be used to change the orbit

Modify 4 - sets a value which can be used to change the orbit

Filter - 2 types of filter are available, a Median filter or a Smoothing filter which use the previous 3-31 audio samples to smooth the output. Each has a different sound.

Out level - sets the output level.

Amp Comp - select a preset for the type of amplitude compensation to use - see help file Amp Comp for details. Can be set to off when not needed.


Orbit button

Presets - select an orbit preset to load it

Orbit Code Box - This contains the SuperCollider code used to define the Orbit.

Evaluate text - use this button after you have edited the code


Terrain button

Terrain Mode - Three modes are available showing different controls:


Terrain Mode: Terrain Code

Presets - select a terrain preset to load it

Terrain Code Box - This contains the SuperCollider code used to define the Terrain.

Evaluate text - use this button after you have edited the code

Range adjust - if the generated values of a terrain are outside the range -1/+1, they will be adjusted using the selected function.
