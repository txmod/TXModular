Harmoniser


Module type: Audio Insert (mono input, mono output)


Introduction:

This module listens to the pitch and volume of the audio signal coming and creates a synthesised waveform with the same pitch and volume.

The output sound can also be transposed for creating harmonies.


Controls:

Pitch detector - this gives 2 options, slower & more accurate, or faster and less accurate. Both can be tried to see which works best.

Waveform/Modify 1/ Modify 2 - see help file Waveform Synthesis

Transpose - transposes the sythesised waveform up or down, by semitones or octaves.

Smooth time - this smooths changes in pitch - useful when the incoming audio changes quickly.

Att. + Decay - this adds an attack and decay time to the synthesised sound.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.
