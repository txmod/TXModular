Quartz Player


Module type: Action


Introduction:

This module creates a new window which can contain any Quartz Composer Composition.

Quartz Composer is a very powerful visual engine that comes free with OS X.

This module can be used with other modules to create interactive and/or automated audio-visual systems.

Quartz Compositions can have inputs that can be used to control the composition. (They can also have outputs, but these aren't currently used in the TX Modular).

The Quartz Player module allows you to send values to any of the inputs of a Quartz Composition. For example, to change the colour of an object, to change the folder used to select photos or movies, to change which photo is shown, to change the speed of a visual effect, to rotate an object or change its position, etc.

All inputs for a Quartz Composition are displayed. Then you can decide what data to send it.


About Quartz Composer

Quartz Composer is a visual programming environment for processing and rendering graphical data, which is distributed free of charge as part of Apple's XCode Development Tools. QC is highly optimised to work with the OSX graphics system.

For more information on QC see: http://developer.apple.com/documentation/GraphicsImaging/Conceptual/QuartzComposer/index.html

If you want to create your own Quartz compositions you must install Quartz Composer on your computer by installing XCode. XCode comes on a CD when you buy a Mac.

See online for Quartz Compositions to download and modify (for examplel, see: http://www.quartzcompositions.com/phpBB2/upload/index.php).


Controls:

Controls are on 11 screens accessed by pressing buttons marked Global, 1, 2, 3 ...10.

The first screen is for global settings for the Quartz Composition, the other screens are used for up to 80 input controls. (Note, some screens will be empty if the composition has less then 80 inputs).


Global Controls:

Open file button - This is displayed the first time you open a Quartz Player module. Use this button to select a Quartz Composition file and open it.

Show Quartz Window - sets whether the Quartz Window is shown or not.

Quartz File - This displays the filename and path of the current Quartz Composition.

Replace current Quartz File - Use this button to select a new Quartz Composition file and open it as it was originally created.

Replace current Quartz File but keep current control settings - Use this button to select a new Quartz Composition file and open it keeping the control settings from the previous Quartz file.

Window width /Wind. height - To specify the size of the display window.

Wind. Pos X/ Wind. Pos Y - To specify the screen position where the window should be displayed.

The X and Y refer to the distance in pixels of the bottom left corner of the window from the bottom left corner of the computer screen.

Sample Current Position - this allows you to move the window to where you want it and then press this button to set Wind. Pos X/ Wind. Pos Y to the current position.

Default Screen Position - this sets Wind. Pos X/ Wind. Pos Y to near the bottom left corner of the screen.

Border X/ Border Y - To set the width of the black border around the edge of the window.

The X and Y refer to the distance in pixels of the left /right and top/bottom borders.

Max FPS - sets the maximum rendering rate (Frames Per Second). If set to 0, it will render as fast as possible. Low FPS values (e.g. 5) will make the playback seem jerky.

Window name - this will appear at the top of the window if Show Window Controls is on.

Show Window Controls - turn this off for a borderless window. However, if you want to be able to drag the window around or resize it, then turn the window controls on.


Controls - for inputs:

Up to 80 inputs can controlled using buttons 1-10 (eight inputs on each page). The name of each input is displayed.

When a Quartz Composition is first opened, all its settings will be exactly how they were when it was originally created.

For each input on the Quartz screen, you need to specify the Type of input. Depending on the type the following controls are shown:

. Type: Not used - No controls are shown. No values will be sent to the input.

. Type: Number - Float - A Float can be any number (including a decimal point), whereas an Integer must be a whole number (nothing allowed after the decimal point). The Min/Max controls set the minimum and maximum numbers allowed. The slider and number box below can then be used to set which number is sent to the input. This value can be modulated.

. Type: Number - Integer - An Integer must be a whole number (nothing allowed after the decimal point). The Min/Max controls set the minimum and maximum numbers allowed. The slider and number box below can then be used to set which number is sent to the input. This value can be modulated.

[N.B. One point to note is that the user will need to check that the Min and Max values for numeric inputs are suitable . The Quartz Player module can only see the current value for a composition when it opened. It won't know the min/max range expected by Quartz Composer for a particular input. Of course, you can always open up the composition in Quartz Composer to see what range of values work best.]

. Type: String - this is for text strings. Text can be entered into the text box and will be sent to the input.

. Type: Boolean number - 0 or 1 - This will send a value of "false" or "true" to the input, depending on if the number is 0 or 1. A button marked 0-False or 1-True sets the value sent to the input. This value can be modulated.

. Type: Colour - R, G, B, A - This will send a colour specified by four numbers, each in the range 0-1. These correspond to Red, Green, Blue and Alpha values. A box next to the numbers displays the colour that will be sent to the input.

. Type: Directory Name - This is for sending a string which is the name of a directory. To create the directory name, press the "Select a File" button and then choose any file that is in the directory you want. The directory name will then be shown in the white box and will be sent to the input.

. Type: File Name - This is for sending a string which is the name of a file. To create the file name, press the "Select a File" button and then choose the file that you want. The file name will then be shown in the white box and will be sent to the input.


Control Modulation Input Busses:

Up to 80 inputs can modulated. However, the modulation will only have an effect if the input Type is numeric. It must be one of the following Type: Number - Float, Number - Integer, or Boolean number.
