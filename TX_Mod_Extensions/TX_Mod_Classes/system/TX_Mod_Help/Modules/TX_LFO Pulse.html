LFO Pulse


Module type:Control Source


Introduction:

Low Frequency Oscillator that outputs a 'pulse' waveform, where a proportion of the signal will have a value of 1 and the rest a value 0. The proportion depends on the pulse width.


Controls:

Frequency - the pulse rate shown as frequency and bpm.

Pulse width - set the proportion of time the signal will be 1 in each cycle.

Output range - preset output ranges including negative values


Control Modulation Input Busses:

Frequency

Pulse width
