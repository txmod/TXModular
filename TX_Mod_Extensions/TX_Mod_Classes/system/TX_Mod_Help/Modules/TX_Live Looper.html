Live Looper


Module type: Audio Insert (mono input, mono output)


Introduction:

This module records the audio input signal into a loop recorder.


Controls:

Record - when this checkbox is switched on, the audio input signal is recorded into the loop.

Freeze Loop - this button does a few things at once to freeze the current loop: it turns record off, sets the input level to 0, and sets the feedback to 1.

Clear Loop - this button clears (silences) the current loop.

Tap tempo button - this displays measures the tempo when it is clicked with a mouse on each beat. Use the Command button to drag the BPM to the Loop time BPM.

LoopTime - sets the loop time in milliseconds or in beats per minute. For example, 1000 milliseconds (i.e. 1 second) is equal to 60 bpm.

time x 2 , time x 3 , time / 2 , time / 3 - buttons to change the loop time.

Input Level - set how much of the input audio signal is recorded into the loop.

Feedback - how much of the audio signal already in the loop will be added as the loop continues to record. If feedback is 1 (i.e. 100 %), the loop will never die away. Lower feedback will make it die away more quickly.

Speed factor - this sets the playback and record speed of the loop. It should always be set to 1(i.e. normal speed) if the live looper is recording or the signal will deteriorate.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.
