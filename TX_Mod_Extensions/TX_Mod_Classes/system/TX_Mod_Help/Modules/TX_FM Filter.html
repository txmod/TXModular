FM Filter


Module type: Audio Insert (mono input, mono output)


Introduction:

This module filters mono audio signals using a resonant filter.

Frequency modulation can be used to add complexity to the sound. The modulation can come from another module using the FM input, or from an internal sine wave.


Controls:

Centre Frequency - the centre frequency of the filter.

Ring time ms - the length of time the filter takes to decay (in milliseconds).

Frequency modulation - sets the frequency modulation mode which can be:
Internal - use sine wave, FM freq, FM depth
Internal - use sine wave, [FM ratio x Centre frequency], FM depth
External- use FM audio input, FM depth

FM freq - sets the frequency of the internal sine wave.

FM ratio - sets the ratio of the internal sine wave frequency relative to the Centre frequency.

FM depth - sets the depth of the frequency modulation.

Out gain - sets the output level of the filtered signal.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Audio Input Bus:

FM input - the incoming mono audio signal used for frequency modulation (if switched on).


Control Modulation Input Busses:

Centre frequency

Ring time ms

FM depth

FM freq

FM ratio

Dry-Wet Mix
