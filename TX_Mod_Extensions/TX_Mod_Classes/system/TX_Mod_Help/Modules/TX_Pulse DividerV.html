Pulse DividerV


Module type: Control Source


Introduction:

This module outputs clock pulses by dividing the number of triggers it receives by varying amounts. For example, if the number of divisions is 3, only the first of every 3 pulses will be output.

There are 4 outputs: Out A/B/C/D. For each one, the number of divisions can be changed and modulated.

Each output also has an offset, which offsets the pulse counting by a variable amount. For example, if the output has a division of 3 and an offset of 1, then the second of every 3 pulses will be output.

This module can be triggered in various ways such as by modulation of the Trigger input, by other modules (such as an Action Sequencer) or by action buttons on an interface screen.

See also Pulse Divider which is similar but different.


Controls:

Trigger Pulse - this is for manually triggering a pulse into the dividers.

Reset Dividers - this resets all dividers so they will reset their counters on the next trigger received. This means that if they have offset of 0, they will trigger on the next trigger received.

Probability - the chance that an input pulse will be sent into the dividers. 1 means always, 0 means never.

Output range - preset output ranges including negative values

Div A/B/C/D - the number of divisions for each output. Out C jumps divisions in steps of 2, and Out D jumps divisions in steps of 3.

Offset A/B/C/D - the offset for each output.
