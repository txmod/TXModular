T Random


Module type: Control Source (triggered)


Introduction:

This module outputs random values that change whenever a trigger is received.

If Length is set to “Infinite”, it will keep on outputting new values.

If Length is set to “Limited until reset”, it will only respond to a maximum number of triggers - Trigger count - before staying at it’s current value. If a reset is received, it will start counting from zero and will respond when the next trigger is received.

This module can be triggered in various ways such as by modulation of the Trigger input, by other modules (such as an Action Sequencer) or by action buttons on an interface screen.


Controls:

Trigger - triggers the next value

Reset - resets the trigger counter to zero

Trigger when reset - if set on, a reset will automatically trigger a new value

Length - “Infinite” or “Limited until reset”

Trigger count - how many new values will be triggered if length is limited.

Output range - preset output ranges including negative values


Control Modulation Input Busses:

Trigger

Reset
