Spring


Module type: Control Source


Introduction:

This module models the behaviour of a vibrating string that is hit with a triggered force.

The output control signal gives the current position of the spring. When the spring isnâ€™t vibrating, it rests in the centre of the output range. When a hit is triggered, it vibrates around the centre point.

This module can be triggered in various ways such as by modulation of the Trigger input, by other modules such as a Clock or Action Sequencer, or by action buttons on an interface screen.


Controls:

Trigger - manually trigger a hit

Trig force - how much force is used when a hit is triggered

Springiness - higher values increase the speed of vibration

Damping - how quickly the vibration dies away

Output scale - scales the depth of vibration


Control Modulation Input Busses:

Trigger

Trig force

Springiness

Damping
