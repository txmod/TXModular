Convolution


Module type:  Audio Insert (mono input, mono output)


Introduction:

"Convolution reverb is a process for digitally simulating the reverberation of a physical or virtual space. It is based on the mathematical convolution operation, and uses a pre-recorded audio sample of the impulse response of the space being modelled."  [from : http://en.wikipedia.org/wiki/Convolution_reverb]

This module convolves a mono audio signal with a mono sample. 

Convolution can also be used to simulate filters/EQ, microphones, speakers, amplifiers as well as to produce special effects - e.g. using a pitched vocal sample. 

Samples must be less than 10 seconds long.  The longer they are, the bigger the CPU load (take care not to overload it, this can cause crashes).

Many free impulse responses can be found online - see link above.


Controls:

Input level - sets the level of the input signal 

Pre-delay - set the delay time before the convolution (useful for creating a larger sense of space with reverbs).  

Dry-Wet Mix - this sets the mix of the original signal with the convolved signal.
