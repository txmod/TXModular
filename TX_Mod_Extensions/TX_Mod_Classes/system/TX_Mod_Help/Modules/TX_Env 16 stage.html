Env 16 stage


Module type: Control Source (triggered)


Introduction:

This module creates polyphonic 16-stage envelopes as control signals.

They can be triggered in various ways such as by MIDI notes, by other modules (such as Step Sequencer, Audio Trigger) or by action buttons on an interface screen.


Controls:

No. Stages - the total number of stages in the envelope (3-16)

Loop stage - which stage the loop stages - this is only relevant when the envelope type is "Sustain Looped".

Release stage - which stage the release part of the envelope begins from - this is only relevant when the envelope type is "Sustain Looped" or "Sustain Unlooped".

Env. Type - this can be "sustain looped", "sustain unlooped" or "fixed length". A "sustain looped" envelope will keep looping from the release stage to the loop stage until the end of the trigger or midi note. A "sustain unlooped" envelope will remain at the release stage level until the end of the trigger or midi note. A "fixed length" envelope will play through all its stages according to the stage times.

Trigger Envelope 1 sec - this will trigger an envelope for 1 second whenever it is pressed.

Envelope display box - this shows each stage of the envelope by level (up-down) and time (left-right). The stages can be moved by dragging them with a mouse.

Levels/Times/Curves - settings for each stage.
Curve types are: Linear, Exponential, Sine, Welch, Step, Slope +10, Slope +9, ... Slope -9, Slope -10

Total Time - for the whole envelope.

Use Curve 1 for all stages - to use one curve for the whole envelope.

Midi Listen - this allows the module to be triggered by a midi note.

Midi Chan. - set the midi channels the module will listen to.

Note range - which notes will trigger the module.

Vel range - which velocities will trigger the module.

Scale level to velocity - if this is set on, then the maximum level of the envelope will depend on the velocity of the trigger or midi note; if this is set off, the maximum level will always be 100%.

Polyphony - maximum number of simultaneous envelopes.


Output Busses:

Env Out - the envelope signal

Env End Trig - outputs a trigger when the envelope ends
