Balance


Module type: Audio Insert (mono input, mono output)


Introduction:

This sets the incoming audio to be at the same level as one of two options:

1) the Target level slider.

2) the level of the mono audio side-chain signal.

Once a Balance module has been created, it will appear in the list of possible destinations for audio channels. This allows you to send an audio signal to the side-chain input.


Controls:

Level follow - this sets which level will be followed - the Target level or the level of the side-chain signal.

Target level - this is only used if it is selected in Level follow.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Audio Input Busses:

Side-chain - mono audio signal.


Control Modulation Input Busses:

Target Level

Dry-Wet Mix
