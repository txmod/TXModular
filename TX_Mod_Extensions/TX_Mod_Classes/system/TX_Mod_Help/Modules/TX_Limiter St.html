Limiter St


Module type: Audio Insert (stereo input, stereo output)


Introduction:

This module limits the volume (amplitude) of a mono audio signal to a certain level (called the threshold). If the volume of the signal goes above the threshold level, it will be reduced to the threshold level.

This is useful to stop a signal from clipping which causes digital distortion.


Controls:

Lookahead - the time which the limiter takes to process the audio. This also imposes a delay on the output signal.

In Gain - this can be used to boost or cut the volume of the audio signal before limiting.

Threshold - the level where the signal is limited.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.
