Trigger Impulse


Module type: Control Source (triggered)


Introduction:

This module outputs a very short impulse when triggered. It can be used as a polyphonic trigger for other modules.

It can be triggered in by clicking the Trigger button or by MIDI notes, other modules (e.g. Step Sequencer, Audio Trigger) or action buttons on an interface screen.


Controls:

Midi Listen - this allows it to be triggered by a midi note.

Midi Chan. - set the midi channels it will listen to.

Note range - which notes will trigger it.

Vel range - which velocities will trigger it.


See the example file TriggerImpulse.
