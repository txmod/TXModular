ReverbA


Module type: Audio Insert (mono input, mono output)


Introduction:

This module adds a simple reverb to a mono audio signal (based on Moorer Lay Reverb).


Controls:

Pre-Delay - sets the size of the modelled room

Reverb time - sets the reverb decay time.

Damping - the amount of high frequency damping. Set higher for a darker sounding reverb.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Control Modulation Input Busses:

Pre-Delay

Reverb time

Damping

Dry-Wet Mix
