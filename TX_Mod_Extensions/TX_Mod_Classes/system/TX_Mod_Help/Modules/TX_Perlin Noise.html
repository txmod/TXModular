Perlin Noise


Module type: Control Source


Introduction:

A Random Low Frequency Oscillator based on 3 dimensional Perlin Noise (see http://en.wikipedia.org/wiki/Perlin_noise for more details).

This is very useful for adding non-repetitive variation (i.e. "organic movement") to static controls.


Controls:

Rate 1/2/3 - the rate of change of dimension 1/2/3/

Rate Multiply - multiplies all the rates by a specified amount.

Output range - preset output ranges including negative and absolute values.

The output will oscillate around the centre of the output range apart from the final option which uses absolute values (use a meter to see this clearly in the output signal).


Control Modulation Input Busses:

Rate 1/2/3

Rate Multiply
