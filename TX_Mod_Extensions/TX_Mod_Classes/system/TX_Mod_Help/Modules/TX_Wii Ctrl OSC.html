Wii Ctrl OSC


Module type: Control Source


Introduction:


To use this module, you will need the software OSCulator to be open with the file "OSCulator_WiiToTX", and connected by bluetooth to your Wii controllers.

Please see the help file Using Wii Controllers for more details.


This module outputs a control signal between 0 and 1 based on a selected Wii or Nunchuck control, including: acceleration, rotation, button and IR camera data.


Controls:

Wii no. - set which Wii controller will be used. OSCulator allows up to 8 Wii controllers to be connected.

Wii control - select which control on the Wii or Nunchuck will be used.
