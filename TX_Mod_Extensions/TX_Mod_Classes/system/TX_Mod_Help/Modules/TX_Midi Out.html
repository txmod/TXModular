Midi Out


Module type: Action


Introduction:

This module sends out various kinds of MIDI messages (see below).

To send a midi message out, you need to run an Action which tells the module to do it. To run the action, there are various ways:

1) Press the button Send MIDI Message Now on the Midi Out module itself.

2) Create an Action in another module or on a front interface screen (see help file What_Are_Actions for more detials).

Controls:

Port - sets the MIDI port messages are sent from.

MIDI Loop-Back - for sending MIDI output back into the TX Modular

Channel - sets the MIDI channel (1-16) messages are sent on.

MIDI Message - sets the type of MIDI message to be sent. These have extra controls as follows:

. Note On - has controls Note (0-127), Velocity (0-127).

. Note Off - has controls Note (0-127), Velocity (0-127).

. Note On & Off - has controls Note (0-127), Velocity (0-127), Gate Time (0.01-20) which is the time between the Note On and Note Off messages.

. Controller - has controls Control no (0-127), Control val (0-127) which is the value to be sent.

. Program - has controls Program no (0-127).

. Pitch Bend - has controls Pitch Bend (-8192 - +8192).

. All Notes Off - has controls.

. Program - has no extra controls Program no (0-127).

. Touch - has controls Touch (0-127).

. Poly Touch - has controls Poly Touch (0-127).

Send MIDI Message Now - this button sends the MIDI message out.



Control no - sets the MIDI controller number (0-127).

Control val - sets the controller value (0-127).
