Rotate St


Module type: Audio Insert (stereo input, stereo output)


Introduction:

This module rotates a stereo input signal in the stereo field, with both manual and automatic rotation available.

It can be used to add movement to a stereo signal, creating amplitude modulation effects at higher rotation frequencies.


Controls:

Rotation - this sets the rotation in degrees. For example, 90 degrees means rotate to the right, and 180 degrees means facing behind.

Auto-Rotate - this turns on automatic rotation.

Frequency - the speed of auto-rotation shown as frequency and bpm.

Reverse - this reverses the direction of rotation.

Bypass - this bypasses the rotation, so the input is unchanged.
