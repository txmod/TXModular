Strings


Module type: Audio Insert (mono input, mono output, triggered)


Introduction:

This module uses the input signal to excite Karplus-Strong models of a group of 1-8 strings. When a trigger is received, the strings are 'plucked' using a short burst of the input signal to excite the strings.

The controls for each string are frequency, filter, delay, decay and amplitude. Negative decay times emphasise odd harmonics.

This module can be triggered in various ways such as by modulation of the Trigger input, by other modules (such as an Action Sequencer) or by action buttons on an interface screen.


Controls:

Trigger - this is for manually triggering a string pluck.


Gain button

In gain

Out gain

Level control - options:

None - level unchanged

Balance - match level to dry signal level

Limiter - keep the signal level below 0db

Normalize - force the signal level to just below 0db

Tanh Clipping - uses hyperbolic tangent of signal to clip it (this adds some distortion)

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Parameters button

Freq range - the range of frequencies for all the strings shown as midi notes

Quantise - forces all frequencies to the nearest midi note

Pitch bend - to globally change all the string frequencies

Max delay - maximum delay time for all the strings

Random delay - adds a random scaling to each stringâ€™s delay time. It will have no effect if the delay is zero.

Max decay - maximum decay time for all the strings


Strings button

Complexity - the number of strings

MIDI learn notes & amps - when this is switched on, any MIDI notes and velocities received on any MIDI channels will be recorded into the strings starting at the Next String, which is shown next to it.
