Convert To Audio


Module type: Audio Source


Introduction:

This module converts a control signal into an audio signal.

Once a Convert To Audio module has been created, it will appear in the list of possible destinations for control channels. This allows you to send an control signal to it.

It outputs an audio signal from the control signal coming in.

You can set the input range of the control signal (the output range for audio is -1 to +1).

There is an option to Remove DC - this is recommended if you are outputting the audio to speakers, since DC offset can damage speakers. For some uses, for example outputting signals to control an analogue modular system, you might want to allow DC offset.


Controls:

Input range - sets the range of the input signal.

Remove DC - filters the signal to remove DC offset. This is on by default.


Control Input Busses:

Control input
