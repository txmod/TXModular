Smooth


Module type:  Control Insert


Introduction:

This module is for smoothing out a control signal over time. There are options to have a separate smooth times for when the control signal moves upwards or downwards.


Controls:

Smoothing - set what type of smoothing to use - linear or 3 types of exponential smoothing which get progressively smoother. Later options use Time 1 for upward smoothing and Time 2 for downward smoothing.

Time 1 - sets how long it takes (in seconds) for the control signal to reach a new value. 

Time 2 - sets how long it takes (in seconds) for the control signal to reach a new value. 


Control Modulation Input Busses:

Time 1

Time 2
