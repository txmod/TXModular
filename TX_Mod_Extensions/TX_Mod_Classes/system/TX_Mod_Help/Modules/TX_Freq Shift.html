Freq Shift


Module type: Audio Insert (mono input, mono output)


Introduction:

From the SuperCollider help file:

'Freq Shift implements single sideband amplitude modulation, also known as frequency shifting, but not to be confused with pitch shifting.'

'Frequency shifting moves all the components of a signal by a fixed amount but does not preserve the original harmonic relationships.'

Frequency modulation is optional, using another audio signal connected to the FM Input to modulate the phase of the signal.

Once a Freq Shift module has been created, it will appear in the list of possible destinations for audio channels. This allows you to send another audio signal to the FM Input.


Controls:

Freq shift - this sets the frequency that will be added or subtracted from the audio signal.

FM type - If set on, the FM Input audio bus will be used to modulate the phase of the waveform, scaled by FM depth.

FM depth - scales the amount of modulation.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Audio Input Busses:

FM Input - the incoming mono audio signal used for frequency modulation (if switched on).


Control Modulation Input Busses:

Freq Shift

FM Depth

Dry-Wet Mix
