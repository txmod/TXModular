Logic Bool


Module type: Control Source


Introduction:

This module combines 2/3/4 input signals using Boolean logic.

Input signals are quantised, since Boolean values are either False (0) or True (1).

Logic options for combining signals are:

AND - output is lowest active input

NAND - output is 1 - lowest active input

OR - output is highest active input

NOR - output is 1 - highest active input



Controls:

Logic Mode - selects between Boolean AND/ NAND/ OR/ NOR and which inputs to use.
