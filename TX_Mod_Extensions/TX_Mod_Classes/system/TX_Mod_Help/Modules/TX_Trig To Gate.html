Trig To Gate


Module type: Control Insert


Introduction:

This module converts a 'trigger' signal into a 'gate'. A trigger is when a control signal changes from zero (or below) to above zero.

When a trigger is received, it outputs a gate signal (level set to 1) for a specific length of time.


Controls:

Retrig mode - sets what will happen if a new trigger is received when the gate is already on. Options are:
Retrigger Off - won't retrigger when gate is on.
Retrigger On - retriggers when gate is on.
Slurrable - retrigger resets time but keeps gate open.

Hold time - how long the gate will be on.


Control Modulation Input Busses:

Hold time
