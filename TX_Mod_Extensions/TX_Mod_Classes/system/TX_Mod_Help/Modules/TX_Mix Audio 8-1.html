Mix Audio 8-1


Module type:  Audio Source (mono output)


Introduction:

This module mixes 8 mono audio input signals. 

Once the module has been added, it's inputs will become valid destinations for any audio Channel.


Controls:

Level 1 ... 8 - sets the level of inputs 1..8. 

Output Level - sets the output level of the mix.
