Clock Pulse


Module type: Control Source


Introduction:

This module outputs clock pulses.


Controls:

Play - start or stop the clock

Restart Trig - restart trigger

BPM & Multiply BPM - set the average clock speed

Jitter - adds random variation to the BPM

Probability - the chance that a pulse will be output

Output range
