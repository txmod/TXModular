Pluck Synth


Module type: Audio Source (mono output, triggered)


Introduction:

This module creates notes based on a physical model of a plucked string.

The initial pluck sound can be noise, a waveform or any other sound. To use an external sound for the pluck, there is a pluck input where an audio signal can be connected using another audio channel (see Audio Input Busses below)

Notes can be triggered in various ways such as by MIDI notes, other modules (e.g. Step Sequencer, Audio Trigger) or action buttons on an interface screen.


Controls:


Pluck/ Env button

Pick type - the sound used to start the pluck. Noise or waveform combinations can be selected or the Pluck input can be used to sample an external sound source.

Pick morph - morphs between different pluck sounds, depending on which Pick type has been selected.

Filtering - this affects the developing tone as the pluck decays. A value of 0 gives an unfiltered decay. Negative values make the pluck time much shorter and brighten the sound.

Boost odd partials - this changes the tone of the pluck to emphasize odd partials (harmonics) making it sound deeper and more 'hollow'.

Decay - the decay time of the pluck envelope

Release - the release time of the envelope.

Curve - this sets the shape of the curve for the release of the envelope.

Env. Type - this can be sustain or fixed length. A sustain envelope will use the release time at the end of the trigger or midi note. A fixed length envelope will decay completely according to the decay time, ignoring the release time.

Level - sets the output level.

Amp Comp - select a preset for the type of amplitude compensation to use - see help file Amp Comp for details. Can be set to off when not needed.


MIDI/ Note button

See help file MIDI Note Button for details.


Audio Input Busses:

Pluck input - mono audio signal which can be used for the pluck sound.
