Note Stacker


Module type: Action


Introduction:

This triggered module makes no sound itself but instead triggers notes in other polyphonic modules, with the option of modifying the pitch or velocity of the notes for various effects.


Up to 18 different modules can be triggered, arranged in 6 layers each with 3 output modules.

The modules on a layer can be played together, rotated each time the layers is played, or randomly chosen.

Each layer has curves for modifying the overall velocity response, as well as individual note level and tuning for each note across the keyboard range.


It allows many kinds of instruments to be designed, for example:

- detuning multiple layers to create rich, evolving tones.

- velocity cross-fading between layers (e.g. to trigger different drum samples for different velocity ranges, or to use distorted waveforms on louder notes - see the Curve Auto-Build controls below)

- multiple keyboard zones which each trigger different output modules (e.g. a plucked bass zone and a lead synth zone).

- complex multi-timbral instruments which have a range of different characters depending on the pitch and velocity of the note played.

- hybrid instruments (e.g. stacked drums, multi-tone organs, or "prepared" instruments).


Controls:


MIDI/Outputs button

Midi Listen - this allows the module to be triggered by midi Notes, and to respond to midi Pitchbend and midi Sustain Pedal (controller no. 64) messages.

Midi Chan. - set the midi channels the module will listen to.

Output Modules and triggering type for each layer:

Layer 1/2/3/4/5/6 Outs - 3 polyphonic source modules, such as wave synths, samplers, or envelopes, All valid modules will appear in the modules list. See the Help Index for details of which modules are polyphonic.

N.B. If Midi Listen is turned on make sure none of the individual output modules has their Midi Listen turned on, or there will be double triggering.

Trigger type - can be one of the following options:

trigger nothing - layer disabled/ trigger all output modules (default)/ trigger a random output module/ rotate trigger between outputs

Midi keyboard- this can be played by clicking with a mouse. It will trigger all layers of the note stacker module. The midi velocity of the note played is set by the Velocity slider. Middle C is tinted red.


Velocity Curve button

Layer 1/2/3/4/5/6 - buttons to set which layer is being edited

Layer 1/2/3/4/5/6 Outs - as above

Trigger type - as above

Vel curve - sets a transfer curve which maps the input velocity to new output velocity. See the preset curves for examples.

See help file Curve Editing for details.

Grid rows - no of rows (1-99) separated by white lines in the grid behind the curve.

Grid cols - no of columns (1-99) separated by white lines in the grid behind the curve.

Random velocity mix - sets the amount of random variation of the velocity for every note played.

Curve Auto-Build: - this row of controls is for spreading a velocity curve across multiple layers, with a settable cross-fade between layers.

Target - the overall velocity curve that is going to spread across multiple layers. This can be either Linear curve or Current Layer 1. If you want to draw a custom curve, draw it in Layer 1 before using the Curve Auto-Build.

Across - sets the number of layers the velocity curve will be spread across.

X- Fade - set the amount of overlap between the velocity layers.

Build now - only press this when you are ready to build. The selected layers' velocity curves will be rebuilt.


Note Levels button

Layer 1/2/3/4/5/6 - buttons to set which layer is being edited

Layer 1/2/3/4/5/6 Outs - as above

Trigger type - as above

Note levels curve - sets the volume level for each individual note across the keyboard.

Copy curve/ Paste curve - for copying between layers

Smooth curve - evens out spikes or sudden changes in the curve. Useful when curve is drawn by the mouse.

Invert curve - turns the curve upside down.

Randomise curve - randomises the level of every note.

Default curve - sets the level of every note to maximum.


Note Detune button

Layer 1/2/3/4/5/6 - buttons to set which layer is being edited

Layer 1/2/3/4/5/6 Outs - as above

Trigger type - as above

Detune range semitones +/- - sets the maximum amount of detune, up to +/- 1 semitone

Transpose semitones - sets the amount of note tranposition for the current layer. This can be positive or negative. This is useful, for example, for doubling notes in octaves.

Random detune mix - sets the amount of random detune for every note played.

Note detune curve - sets the detune, which can be positive or negative, for each individual note across the keyboard.

Copy curve/ Paste curve - for copying between layers

Smooth curve - evens out spikes or sudden changes in the curve. Useful when curve is drawn by mouse.

Invert curve - turns the curve upside down.

Randomise curve - randomises the detune of every note.

Default curve - sets the detune of every note to 0.
