Strum


Module type: Control Source


Introduction:

This module makes no sound itself but instead outputs triggers and triggers notes in up to 6 different modules. It can also output MIDI notes.

Whenever, the 'strum position' is moved or modulated, it triggers notes in the current scale - either a preset scale, such as 'Major' or 'Minor', or a Custom Scale which is editable.

The 'strum position' is modelled on hand position when playing a guitar - as the hand moves up or down, the guitar strings are strummed.

The notes in the current scale are spread evenly along the strum position - the more notes there are in the current scale, the closer they will be to each other when strumming.

By modulating the strum position with LFOs or envelopes, many different kinds of strumming patterns can be created. This can also be used to create rasping or scraping effects by triggering short noise sounds in wave synths or samplers.


Controls:

Notes in scale - shows total number of notes in the current scale.

Active - if switched off, no notes will be triggered


Outputs button

Modules - up to 6 polyphonic source modules, such as wave synths, samplers or polyphonic envelopes, can be triggered. All valid modules will appear in the modules list. See the Help Index for details of which modules are polyphonic.

Output MIDI notes - on/off.

MIDI Port - select from available ports

MIDI Loop-Back - sends MIDI notes back into the TX Modular system on the selected channel, so other modules can respond

Note: Take care not to output MIDI loop-back to a channel that the module itself is listening to - otherwise the system will crash with a MIDI feedback loop.

MIDI Channel - output channel.


Strum button

Strum mode - sets which movement direction (up or down) will trigger notes when the strum position moves.
Options: 1. Strum up and down, 2. Strum up only, 3. Strum down only.

Strum position - when this moves it triggers notes in the current scale.

Smoothing - This allows you to add smoothing to the strum position.
See help file Smooth for smoothing options.

Velocity min - the minimum velocity for each note.

Random vel - adds an extra random velocity to each note.

Gate time - the gate time in seconds for each triggered note.


Scale & Transpose button

Keyboard- Middle C is tinted red. Notes that are part of the current scale are tinted yellow.

Use Custom Scale - When 'Use Custom Scale' is switched on, notes can be added to or removed from the current scale by clicking or dragging on the keyboard.
When 'Use Custom Scale' is switched off, the settings for 'Key/ root', 'Chord/ scale' and 'Note range' will be used to build the current scale.

Key/ root - C to B.

Chord/ scale - various options showing interval, chord, mode and scale names.

Note range - the range of notes used to create the current scale.

Transpose - transposes all played notes up or down (in semitones).

Quantise tsp - quantise the Transpose value to whole semitones or whole octaves (12 semitones).
