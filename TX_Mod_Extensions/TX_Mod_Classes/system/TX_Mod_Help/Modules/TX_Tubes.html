Tubes


Module type: Audio Insert (mono input, mono output)


Introduction:

This module uses the input signal to excite a physical model of a tube made up of 2-8 sections.

The controls are: length of each tube, scattering at junctions between sections, and loss at junctions between sections as well as at both ends.

So for example, for 2 tubes there are 2 length controls, 1 scatter control, and 3 loss controls.


Controls:

Gain button

In gain

Out gain

Level control - options:

None - level unchanged

Balance - match level to dry signal level

Limiter - keep the signal level below 0db

Normalize - force the signal level to just below 0db

Tanh Clipping - uses hyperbolic tangent of signal to clip it (this adds some distortion)

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Parameters button

Loss range - the range of values for loss

Max length - maximum length of each tube

Scatter range - the range of values for scatter


Tubes button

Complexity - the number of tubes

Loss 1-9

Length 1-8

Scatter 1-7
