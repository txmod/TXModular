Sub Harm


Module type: Audio Insert (mono input, mono output)


Introduction:

This module adds 4 pulse-wave sub-harmonics to a mono audio signal. Each sub-harmonic is progressively lower.

A selectable filter can be used to cut or boost frequencies to shape the output signal.


Controls:


Subs button

Sub 1/2/3/4 Level - level of each sub-harmonic. If all the levels add up to more than 1, they will automatically be scaled down to stop overloading the output.

Out Gain - to adjust the mixed output level.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Filter button

Filter - type of filter. The name will indicate whether the Resonance and Saturation controls are used - they are only used by some filters.

Frequency - the centre or cut-off frequency of the filter.

Smoothing - for smoothing out changes to the filter frequecy, this sets what type of smoothing to use - linear or 3 types of exponential smoothing which get progressively smoother.

Smooth Time - sets how long it takes (in seconds) for the frequency to reach a new value.

Resonance - only certain filter types have resonance controls (see filter names). Resonance is how focussed the filter is around the centre frequency.

Saturation - only certain filter types have saturation controls (see filter names). Saturation boosts the filter signal and can introduce mild distortion.
