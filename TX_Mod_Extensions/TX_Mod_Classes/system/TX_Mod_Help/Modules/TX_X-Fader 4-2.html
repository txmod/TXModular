X-Fader 4-2


Module type: Audio Source (stereo output)


Introduction:

This module crossfades between two stereo audio input signals.

Once a module has been created, it will appear in the list of possible destinations for audio channels, so you can send audio signals to it.


Controls:

Level 1+2 - sets the level of inputs 1+2.

Level 3+4 - sets the level of inputs 3+4.

X-Fade - the mix between inputs 1+2 and inputs 3+4.


Fade 1 & 2 Curves

See help file Curve Editing for details.
