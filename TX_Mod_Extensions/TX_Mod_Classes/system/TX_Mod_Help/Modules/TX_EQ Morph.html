EQ Morph


Module type: Audio Insert (mono input, mono output)


Introduction:

This module can morph between 2-5 EQ presets using 6 switchable bands of EQ: Lo-shelf, Parametric, or Hi-shelf, with +/- 15db gain per band.

These bands can be used to sculpt a very precise EQ curve. All bands can cut or boost frequencies in an audio signal by a specified amount, expressed in decibels (db).

There are 5 buttons for EQ settings (EQ A-E) which can be set to create different EQ presets. Then you can morph between 2-5 EQ presets using the morph controls.

The morph can be modulated.


Controls:


EQ Morph button

Output - sets the outputs to use the EQ Morph settings or else Solo one of the EQ presets A/B/C/D/E. In normal use, it should be set to EQ Morph. The others choices are only used when changing the settings for a particular EQ preset.

EQ Bands 1-6 - sets the type of filter for each band: OFF, Parametric, Lo-shelf, or Hi-shelf

Morph Type - select between different types of morph, 2-D or Linear, using 2 to 5 EQ presets.

Morph X-Y / Morph X - depending on the Morph Type chosen either a 2D slider or a normal slider will be displayed for morphing between different EQ presets.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


EQ A/B/C/D/E buttons

Output - sets the outputs to use the EQ Morph settings or else Solo one of the EQ presets A/B/C/D/E. In normal use, it should be set to EQ Morph. The others choices are only used when changing the settings for a particular EQ preset.

EQ Bands 1-6 - sets the type of filter for each band: OFF, Parametric, Lo-shelf, or Hi-shelf

Frequency - the centre or cut-off frequency of each filter.

Resonance - only parametric filters have resonance, which is how focussed the filter is around the centre frequency.

Gain - the amount of cut or boost (in db).

Store Snapshot - sets which snapshot number is stored.
Snapshots store all the Freq, Res & Gain settings, for easy copying and pasting between EQ presets A-E

Name - sets the snapshot name

Load Snapshot - sets which snapshot number is loaded

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.
