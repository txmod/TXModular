PolyNote Trigger


Module type: Control Source


Introduction:

This module can be used to trigger notes in up to 6 polyphonic modules using a control signal from another module, such as Clock Pulse, LFO or Step Sequencer, or using an Action Button on an Interface Screen.

When the Gate is switched on, notes are triggered in selected polyphonic modules - such as synths, envelopes & other 'note tools' modules.

If Use fixed time is on, the fixed time will be used for note length.

If Use fixed time is off, the notes will play until the the Gate is switched off. This can be used to create drones (continuous notes) from polyphonic modules.

The 3 output control signals - Gate, Note and Velocity - can be used to modulate other modules.


Controls:

Modules to be triggered - up to 6 modules can be selected. Only polyphonic modules currently in the system will be shown.

Gate - this will trigger modules when switched on.

Use fixed time - if set on, Fixed time will be used for note lengths.

Fixed time - length of the gate if Use fixed time is switched on.

Note - this sets the note (shown as a midi note number).

Velocity - this sets the note velocity.


Control Modulation Input Busses:

Gate

Fixed time

Note

Velocity
