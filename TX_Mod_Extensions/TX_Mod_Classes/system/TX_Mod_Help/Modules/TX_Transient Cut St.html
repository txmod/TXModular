Transient Cut St


Module type: Audio Insert (stereo input, stereo output)


Introduction:

This module limits the slope of a stereo audio signal.

The upward and downward slopes are expressed in units per second.

This module has the effect of removing transients and higher frequencies from the audio signal.


Controls:

Up Slope - sets the maximum upward slope.

Down Slope - sets the maximum downward slope.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.
