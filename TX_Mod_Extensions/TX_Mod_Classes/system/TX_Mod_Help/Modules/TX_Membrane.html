Membrane


Module type: Audio Insert (mono input, mono output)


Introduction:

This module uses the input signal to excite a physical model of a drum-like membrane, based on triangular waveguide meshes.

There are 2 shapes to choose from: Hexagon, which uses less CPU, and Circle which is more complex.

Controls:

Shape - Hexagon or Circle

In gain

Tension

Loss

Out gain

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.
