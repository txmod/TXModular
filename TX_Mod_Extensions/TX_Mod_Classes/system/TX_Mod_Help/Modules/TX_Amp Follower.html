Amp Follower


Module type: Control Source


Introduction:

This module follows the amplitude (volume level) of a mono audio signal coming into it.

Once an amp follower has been created, it will appear in the list of possible destinations for audio channels. This allows you to send an audio signal to it.

It sends out a control signal at the same level as the audio coming in.


Note: Also in the TX Modular system is an audio insert module - Env Follow - which matches the level of a mono audio signal to another mono audio signal that is being fed into the side-chain input.


Controls:

Follow time - time taken in milliseconds to follow the amplitude. Longer times will tend to smooth the output signal.


Audio Input Busses:

Audio in - the incoming mono audio signal for amp following.


Control Modulation Input Busses:

Follow time
