Live Grain


Module type: Audio Insert (mono input, stereo output)


Introduction:

This module granulates a mono signal, turning it into 'grains' of sound which are panned in stereo.


Controls:


Grain button

Max grains - maximum number of grains at any time

Triggering - can be Internal or External.
Internal triggering uses an internal clock based on the grain time and density.
External triggering uses the Ext Trigger control input - whenever the signal goes above zero, a grain will be triggered. (Vary triggering and Grain density are ignored.)

Vary triggering - this adds a random variation to the start time of each grain. If set to zero, all grains will be at equally spaced times.

Grain density - the number of overlapping grains. Values below one will have gaps between grains.

Grain time ms - how long each grain is. Shorter times sound more fragmented.

Vary grain time - this adds a random variation to the length of each grain.

Grain pan - the pan position of the grain.

Vary panning - this adds a random variation to the pan of each grain.

Level - the output level of the granulator.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Grain Env button

Grain Env - the envelope to use: either the custom envelope shown below it, or a built-in Hann envelope.

Custom Env - editable envelope curve.

See help file Curve Editing for details.

Grid rows - no of rows (1-128) separated by white lines in the grid behind the curve.

Grid cols - no of columns (1-128) separated by white lines in the grid behind the curve.


Control Modulation Input Busses:

Grain time ms

Grain density

Grain pan

Vary triggering

Vary grain time

Vary panning

Dry-Wet Mix

Ext Trigger
