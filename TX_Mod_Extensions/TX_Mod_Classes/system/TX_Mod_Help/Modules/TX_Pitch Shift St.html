Pitch Shift St


Module type: Audio Insert (stereo input, stereo output)


Introduction:

This module shifts the pitch of an incoming stereo audio signal.

It does this using a granulation process which can also be used as a creative effect.

Adding some Stereo Shift moves the Left and Right channels' pitch in opposite directions. This can be used for gentle stereo detune or more dramatic effects.


Controls:

Pitch shift - shown in semitones. This changes the pitch of both Left & Right channels.

Stereo shift - shown in semitones. This changes the pitch of Left & Right channels in opposite directions.

Grain size - sets the size in seconds of the audio grains which are then pitch shifted. Grain size also affects the sound of the pitch shifting, so it's worth trying different sizes.

Random pitch - adds some random pitch shifting to every audio grain. A small amount can sound like chorusing. Larger amounts create more extreme effects.

Random time - adds some random variation to the timing of every audio grain. Can be used to add non-uniformity to the grain stream.

Gain boost - the effect of granulation, especially with lots of randomness, can make the sound seems quieter, so gain boost can be used to compensate for this.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Control Modulation Input Busses:

Pitch shift

Stereo shift

Dry-Wet Mix
