MultiTrack Seq


Module type: Action


Introduction:

This module can be used to record, edit, and play back multiple Controller and Note Tracks to control parameters of other modules or send them note triggers.

Each track can record from MIDI input and have up to 6 output modules, as well as MIDI output.

Controller Tracks can also record direct input - from a slider, for example on the interface, or from one of 16 available modulation inputs.

Note tracks can send output to polyphonic source modules, such as synths or polyphonic envelopes.

Basic MIDI file import/export is available with BPM on a separate tempo track, although time signature changes in MIDI files are ignored when importing.

There are several examples that come with TX Modular system which show various ways to use the MultiTrack Sequencer
(see TX_Mod_Examples/Specific_examples/MultTrackSeq...).

Other help that might be useful: Modulation Vs Automation



Global Controls:

Rewind - sets the current Time position to be at the start position.

Play - button to start the sequencer playing from the current Time position.

Stop - button to stop the sequencer.

Record - button to start the sequencer recording from the current Time position. Only tracks with Record set on will record.

Sync Start - this checkbox sets whether the sequencer will be started when the "Sync Start" button is pressed. The "Sync Start" button, at the top of the main screen, is used to start multiple modules at the same time.

Sync Stop - this checkbox sets whether the sequencer will be stopped when the "Sync Stop" button is pressed. The "Sync Stop" button, at the top of the main screen, is used to stop multiple modules at the same time.

Mute all tracks - this will not stop a sequencer if it is running, just temporarily mute all the tracks.

All Tracks button - quick way to show all tracks

BPM - sets the tempo in beats per minute.

Beats per bar

Tap tempo button - this displays measures the tempo when it is clicked with a mouse on each beat. Use the Command (Apple) key with a mouse drag to copy the BPM to the sequencer BPM.

Auto copy - when set on, this automatically copies the tap tempo to the sequencer BPM as long as the tap tempo is within the min-max range of BPM values.

Play From Start checkbox - when on this forces playback to start from the start position

Start & Stop positions, shown as bars : beats

Selection Start & End positions, shown as bars : beats - the selected area is shown as a yellow box on the timeline and in all selected tracks

Loop checkbox, Loop Start & End positions, shown as bars : beats - the looped area of the timeline is shown as a green line. If the loop is on, the green line is thicker and the word “Loop” will appear at the start of the loop.


Track Controls:
- there are 3 main views of Tracks:

- Multi-Track Edit - shows multiple tracks on the timeline. You have the ability to set many track properties here an run processes such as copying, cutting, pasting, multiple tracks.

- Note Track Edit - shows a single track on the timeline. notes are shown on a frequency grid with note velocity optionally shown below

- Controller Track Edit - shows a single track on the timeline. controller values are shown as connected points on a grid



Multi-Track Edit Controls:

Snap To Grid checkbox - this affects timeline selection

Snap beats - if Snap To Grid is on this sets the grid resolution

Bars - sets the range of bars that are displayed

Max button - zoom out to display the maximum no of bars

Max number box - sets the maximum no of bars that will be displayed

Time - current time position, shown as bars : beats. A red line is shown on the timeline and in all tracks at the current time position. This moves during recording & playback.

< > - Skips to previous or next marker - a quick way to move around the timeline

Add Marker - A marker is added at the current time. To name a marker, type the name in the white box to right before adding it.

Del - deletes a marker (if it exists) at the current time position

Timeline ruler - this shows numbered bars with thick lines and individual beats with thin lines. Clicking on on the Timeline with modifier keys works as follows:

click to set play position, shift drag to change selection boundaries, & command drag to change loop boundaries

—

Select All - turns select on in all tracks

Unselect All - turns select off in all tracks

Mute All - turns mute on in all tracks

Unmute All - turns mute off in all tracks

Unhide All - all hidden tracks will be expanded tracks

Markers ruler - this shows each marker with a line at the time, either named or as bars & beats.

Markers can be dragged forwards or backwards in time.

Individual Track Controls:

- / + - track hide/ unhide - this small button make a track hidden so it appears only as a thin grey line

Track Position popup - shows a track’s position in the track order (e.g. “Tr 3”) - to move the track, select a new position from the list

Track Name - text box to add name

Track Type & ID - shows the track type - note or controller - and an ID that is fixed (unlike the track position which can change)

Edit - changes the window to edit a particular track

Del - this will delete the track. This cannot be undone.

—

Sel - sets whether the track is selected - lots of processes only affect selected tracks

Mon - sets input monitoring on/off for the track

Rec - sets record mode on/off for the track

Offset - this is a timing offset for the track, shown in beats

Mute - if a track is muted, the recorded material on the track won’t be output

Dup - this makes a duplicate copy of the the track, both settings and recorded data and places it directly below in the track order

Track Data - along the timeline, notes are shown as coloured lines, and controller data as a shaded graph - darker below the controller value, white above it.

MultiTrack Process Controls:

Process - select a process and any required controls will appear to the right. Some processes require tracks to be selected first.

Run Process - runs the selected process

Multitrack processes include:

Copy, Cut, Past, Undo, Redo, Interpolation,

MIDI File Import - tempo changes will be imported into a track called “tempo track”. For this to have an effect, the track output must be connected to the multitrack’s BPM

MIDI File Export - if a controller track is named “tempo track”, it won’t be made into a controller track but will be used to control the tempo of the exported MIDI file.


Note Track Edit Controls:

Inputs & Outputs button - this shows controls for setting inputs & outputs.

Note: Note tracks can only use MIDI as their input source.To be able to record another sequencer module within the TX Modular, set it’s output to MIDI loopback to bring the MIDI data back in. Then set the recording track to listen to the correct MIDI input channel.

MIDI Learn - automatically selects the Midi Channel based on incoming MIDI data.

MIDI Channels - which MIDI channels the track will listen to (1-16).

Velocity Scale - multiplies the output note velocities (default 1)

Transpose - transposes the track output, shown in semitones (default 0)

Active - to activate specific outputs

Output Module 1..6 & Parameter - up to 6 output modules can be triggered. Only polyphonic modules can be triggered, such as wave synths, samplers, or envelopes. All valid modules will appear in the modules list.

MIDI Output Port - select from available ports

MIDI Loop-Back - sends MIDI notes back into the TX Modular system on the selected channel, so other modules can respond.

Note: Take care not to output MIDI loop-back to a channel that the module itself is listening to - otherwise the system will crash with a MIDI feedback loop!

MIDI Channel - output channel (1-16).

—

Draw Notes button - this shows all notes on a grid, with an optional velocity track below it.

To add a note, click in the grid at the start of the note and drag to the right to set the note length. Dragging up or down will change the velocity of the note, which is also represented by the colour of the note.

If the velocity track is shown, once you have drawn a note, you can drag the black dot at the top of the velocity line to change it.

Various Snap grid controls are taken into account when drawing or editing. They are specific for each track in the sequencer.

In the yellow box below the velocity track, note and velocity values are shown.Velocity is given as a % (range 0-100), with MIDI velocity (range 0-127) shown in brackets.

—

Edit Notes button - this looks similar to Draw Notes but with more controls. In this view you can select multiple notes and run processes on them (Copy, Cut, Paste, etc.).

To select notes, you can drag over notes in the note grid, or in the velocity track, or if the auto select checkbox is on (located above the MIDI keyboard graphic), by shift-dragging in the Timeline Ruler. Use shift and drag to add notes to the current selection.

If only a single note is selected, all the note properties (MIDI note, start/end position, velocity %) are shown in editable boxes below the velocity track for precise editing.

Selected notes can be altered by dragging the start position line, end position line, or main body of the note.

If the velocity track is shown, dragging the black dot at the top of the velocity line will change it. If many notes are selected, they will be scaled proportionately when changing velocities.

Process - select a process and any required controls will appear to the right.

Run Process - runs the selected process


Controller Track Edit Controls:

Inputs & Outputs button - this shows controls for setting inputs & outputs. A controller track’s input can be MIDI controller or pitch bend data, one of 16 modulation inputs (for example for recording the output of an LFO module), or a direct input slider or number box (which could for example be controlled by a front interface slider).

Note: if a modulation input is used, a popup for “MultiTrack Modulation Sample Rate” appears. This setting will affect all tracks that are using modulation inputs in the current MultiTrack module.

Active - to activate specific outputs

Output Module 1..6 & Parameter - up to 6 output module parameters can be used.

MIDI Output Port - select from available ports

MIDI Loop-Back - sends MIDI data back into the TX Modular system on the selected channel, so other modules can respond.

Note: Take care not to output MIDI loop-back to a channel that the module itself is monitoring - otherwise the system will crash with a MIDI feedback loop!

MIDI Channel - output channel (1-16).

MIDI Controller - output controller number (0-127).

—

Draw Points button - this shows all controller values as a connected line of points on a grid. The area below the line is shaded.

The left edge of the grid shows minimum & maximum values as well as quantise steps as text.

If Output Parameter 1 has been assigned (see Inputs & Outputs button), the equivalent value of that parameter is also shown on the left (i.e. some parameters have values that aren’t in the range 0-1, so they will be shown mapped to their relevant range).

Various Snap grid controls are taken into account when drawing or editing. They are specific for each track in the sequencer.

There are range (= zoom) controls for bars displayed and controller values displayed above and to the right of the grid.

Quantise Value - if set on, all drawn values will be quantised based on the value of Quantise Steps.

Mouse drag draws multiple points - if set off, a single point will be drawn at the final mouse position at the end of the drag. This makes it easier for fine control of a single point’s position.

—

Draw Lines button - similar to Draw Points, this is for drawing lines or curves with a specific shape.

Line Shape - various curve shapes can be selected.

—

Edit Selection button - select multiple controller values and run processes on them (Copy, Cut, Paste, etc.).

To select controller values, you can drag over points in the grid or shift-drag in the Timeline Ruler. Use shift and drag to add to the current selection.

If only a single value is selected, all the event properties (time position, value, & output 1 value) are shown in editable boxes below the grid for precise editing.

Process - select a process and any required controls will appear to the right.

Run Process - runs the selected process
