Granulator FM


Module type: Audio Source (stereo output, triggered)


Introduction:

This module uses Frequency Modulation Synthesis to create small "grains" of sound which are panned in stereo.

Frequency Modulation Synthesis uses 2 Sine waves: the Carrier wave has it's frequency modulated by the Modulator wave.

The Modulation Index sets the amount of modulation. Higher values create more harmonics.

The module can be triggered in various ways such as by MIDI notes, other modules (e.g. Step Sequencer, Audio Trigger) or action buttons on an interface screen.


Controls:


FM button

Carrier mode - selects how the carrier frequency will be set:
1) Use Carrier ratio multiplied by the MIDI note played.
2) Use Carrier freq

Carrier ratio

Carrier freq

Modulator mode - selects how the modulator frequency will be set:
1) Use Modulator ratio multiplied by the MIDI note played.
2) Use Modulator freq

Modulator ratio

Modulator freq

Modulation index - sets the amount of frequency modulation.

Level - the output level of the granulator.

Amp Comp - select a preset for the type of amplitude compensation to use - see help file Amp Comp for details. Can be set to off when not needed.


Grain button

Max grains - maximum number of grains at any time

Triggering - can be Internal or External.
Internal triggering uses an internal clock based on the grain time and density.
External triggering uses the Ext Trigger control input - whenever the signal goes above zero, a grain will be triggered. (Vary triggering and Grain density are ignored.)

Vary triggering - this adds a random variation to the start time of each grain. If set to zero, all grains will be at equally spaced times.

Grain density - the number of overlapping grains. Values below one will have gaps between grains.

Grain time ms - how long each grain is. Shorter times sound more fragmented.

Vary grain time - this adds a random variation to the length of each grain.

Grain pan - the pan position of the next grain.

Vary panning - this adds a random variation to the pan of each grain.


Grain Env button

Grain Env - the envelope to use: either the custom envelope shown below it, or an built-in Hann envelope.

Custom Env - editable envelope curve.

See help file Curve Editing for details.

Grid rows - no of rows (1-128) separated by white lines in the grid behind the curve.

Grid cols - no of columns (1-128) separated by white lines in the grid behind the curve.


MIDI/ Note button

See help file MIDI Note Button for details.

Note: unlike most other audio synth or sample modules, keyboard tracking is turned off by default in the Granulator module. Of course, you can turn it on if required.


Envelope button

See help file Env DADSSR for details.
