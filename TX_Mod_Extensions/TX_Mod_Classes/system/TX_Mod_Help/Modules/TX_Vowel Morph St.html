Vowel Morph St


Module type: Audio Source (stereo output)


Introduction:

This module generates a stereo audio signal based on Vowel sounds.

It can morph between 2-5 Vowel presets, where 5 formants create the character of each vowel preset, either by directly synthesising the formants, or by filtering a source waveform (using 5 band-pass filters).

Different source sounds are available which change the character of the overall sound. If Formant is chosen, the character is most like a human voice. Other pitched or noisy waveforms give very different characters, depending on the mix of frequencies they contain.

An external input is available for using this module as a vowel filter, which adds the vowel character to a signal sent to it from another module.

There are 5 buttons for Vowel settings (Vowel 1-5) which can be set to create different Vowel presets. Then you can morph between 2-5 Vowel presets using the morph controls, and the morph can also be modulated.

To create a beating effect, the left and right channels are shifted slightly up and down in pitch. The beats frequency is also the difference in frequency between the two channels.


Controls:

Output - sets the outputs to use the Vowel Morph settings or else Solo one of the Vowel presets 1-5. In normal use, it should be set to Vowel Morph. The others choices are only used when changing the settings for a particular Vowel preset.

Source sound - select between different source sounds including formant, external inputs, and various noise and pitched waveforms.
Note: For noise waveforms or external input, Frequency and Smoothing aren't relevant and will be ignored.


Vowel Morph button

Morph Type - select between different types of morph, 2-D or Linear, using 2 to 5 Vowel presets.

Morph X-Y / Morph X - depending on the Morph Type chosen either a 2D slider or a normal slider will be displayed for morphing between different Vowel presets.

Out Level - sets the output level of the module.

Amp Comp - select a preset for the type of amplitude compensation to use - see help file Amp Comp for details. Can be set to off when not needed.


Vowel 1/2/3/4/5 buttons

Settings for each of the 5 formants:

Frequency - the centre or cut-off frequency of each formant.

Width - the bandwith in cycles per second (cps), which is how focussed the formant is around the centre frequency.

Gain - the formant level (in db).

Presets - choose between various vowel presets.

Load - loads the vowel preset.

Out Level - sets the output level of the module.

Amp Comp - select a preset for the type of amplitude compensation to use - see help file Amp Comp for details. Can be set to off when not needed.


Frequency button

Frequency - sets the pitch or frequency of the source sound waveform (not relevant for noise waveforms or external input).

notes - this popup allows you to select a MIDI note name (e.g. D3) to set the frequency.

OFF/ON Use note list instead of variable frequency - if set on, this uses a specific note list which the waveform's frequency is chosen from. The controls below allow you to specify which notes will be in the note list.

Scale type - sets the interval, scale, mode or chord used for the note list.

Key / root - sets the key or root of the above scale.

Note range - sets the min and max of notes in the note list using MIDI note numbers.

Note count - shows the total number of notes in the current note list.

Note select - this slider selects one of the notes in the current note list. The lowest note is with the slider set to 0, with the highest at 1, and all others in between. The more notes there are in the note list, the more closely they will be packed along the length of the slider. This can be modulated.

Beats Freq - sets the pitch or frequency of the beating.


Freq Smoothing button

This allows you to add smoothing to the frequency parameter (not relevant for noise waveforms or external input).

See help file Smooth
