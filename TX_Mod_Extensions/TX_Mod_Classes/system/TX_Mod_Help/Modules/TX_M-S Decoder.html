M-S Decoder


Module type: Audio Source


Introduction:

This module takes mid and side signals and converts them into a stereo audio signal.

Once an M-S Decoder module has been created, it will appear in the list of possible destinations for audio channels. This allows you to send mid and side signals to it from other channels.


Also check online for details about M-S techniques; e.g. http://emusician.com/mag/emusic_front_center/, see section Dial M-S for Mastering.


Controls:

Left/Right Level - level of output for each channel.


Audio Input Busses:

Audio in - the incoming stereo audio signal.


Control Modulation Input Busses:

Left Level

Right Level


Note: an M-S Encoder module can be used to convert a stereo signal into mid and side signals. See help file M-S Encoder for details.
