Env Follow


Module type: Audio Insert (mono input, mono output)


Introduction:

This module matches the level of a mono audio signal to another mono audio signal that is being fed into the side-chain input.


Note: Also in the TX Modular system is a control source module - Amp Follower - which sends out a control signal at the same level as the audio coming into the side-chain input.


Controls:

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Audio Input Busses:

Side-chain - mono audio signal whose level is matched.

Control Modulation Input Busses:

Dry-Wet Mix
