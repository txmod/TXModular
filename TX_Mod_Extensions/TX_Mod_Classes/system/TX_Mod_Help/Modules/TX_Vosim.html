Vosim


Module type: Audio Source (mono output)


Introduction:

This module outputs a synthesised waveform as a mono audio signal. The name VOSIM comes from VOice SIMulation.

The following technical explanation is taken from:

http://dspwiki.com/index.php?title=Physical_Modeling_Synthesis :

"The core idea of VOSIM is the generation of a repeating tone-burst signal, producing a strong formant component. VOSIM was originally used to model vowel sounds, but has been extended to model fricatives and quasi-instrumental tones.The VOSIM waveform was derived by approximating the signal generated by the human voice. This approximation takes the form of a series of pulsetrains, where each pulse in the train is the square of a sine function...Two characteristics of the VOSIM signal are a fundamental corresponding to the repetition frequency of the entire signal, and a formant peak in the spectrum corresponding to the pulse width of the sinÂ² pulses. In order to create a sound with several formants, it is necessary to mix the outputs of several VOSIM oscillators."


Controls:


Vosim button

Formant freq - sets the formant frequency

Instability - adds instability to the timing of the pulsetrains

Wave cycles - sets the number of wave cycles in each pulsetrain

Decay Factor - sets the relative level of successive waves cycles in each pulsetrain

Level - sets the output level

Amp Comp - select a preset for the type of amplitude compensation to use - see help file Amp Comp for details. Can be set to off when not needed.


Frequency button

Freq - sets the pitch or frequency of the waveform

notes - this popup allows you to select a midi note name (e.g. D3) to set the frequency.

OFF/ON Use note list instead of variable frequency - if set on, this uses a specific note list which the waveform's frequency is chosen from. The controls below allow you to specify the which notes will be in the note list.

Scale type - sets the interval, scale, mode or chord used for the note list.

Key / root - sets the key or root of the above scale.

Note range - sets the min and max of notes in the note list using midi note numbers.

Note list - shows the total number of notes in the current note list.

Note select - this slider selects one of the notes in the current note list. The lowest note is with the slider set to 0, with the highest at 1, and all others in between.The more notes there are in the note list, the more closely they will be packed along the length of the slider. This can be modulated, for example with an lfo or midi controller.


Freq Smoothing button

This allows you to add smoothing to the frequency parameter.

See help file Smooth
