Spectral Delay


Module type: Audio Insert (mono input, mono output)


Introduction:

This module adds separate delays, of up to 1 second, to the input signal in 256 different frequency bands.

Controls:

Delay Curve - sets the delay time for every band, showing Frequency on the X-axis (low on the left, high on the right) and Delay Time on the Y-axis (short at the bottom, long at the top).

Max Delay - Maximum delay time for any freq

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Control Modulation Input Busses:

Max Delay

Dry-Wet Mix
