Scale Quantise


Module type: Control Insert


Introduction:

This module quantises the input signal to a set key, scale and note range.

This could be used, for example, to restrict the frequency of a Waveform or Filter module so that it is in a certain key and scale.

The output signal is in the range 0-1.


Controls:

Quantise - activates the quantise. If turned off, the input signal will be mapped to the output range 0-1 without quantisation.

Input range - preset ranges including negative values

Key - C to B.

Scale - various options including intervals, chords, modes and scales.

Intonation - various options including equal temparament, just intonation and other specific tunings.

Note ratios - for the Intonation selected.

Note range - lowest and highest notes.
