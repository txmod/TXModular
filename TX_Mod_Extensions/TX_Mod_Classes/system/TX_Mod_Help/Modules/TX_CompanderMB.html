CompanderMB (Multi-Band)


Module type: Audio Insert (mono input, mono output)


Introduction:

This module splits the input signal into 4 frequency bands and responds to the amplitude (volume level) of each band.

Each band's compander can function as a compressor or expander/gate.

A compressor reduces the level of the louder parts of signal. It has the effect of evening out the volume.

An expander/gate reduces the level of the quieter parts of signal. It can be used to remove the low level noise in a signal.

To make it act as an expander or noise gate, increase the expander ratio to greater than 1 and it will reduce the amplitude if it below the threshold.

To make it act as a compressor, increase the compressor ratio to greater than 1 and it will reduce the amplitude if it above the threshold. Then increase the Out gain to bring the level up to what you want.

There is the option to use a side-chain audio input to control the compander, instead of the main input signal. If the side-chain is used, then on another audio channel, the destination can be set to the Compander module's Side-chain bus (see Audio Input Busses below).

A lookahead delay can also be added. Take care to set appropriate clamp and relax times if using a lookahead delay.


Note: Split frequencies cannot be modulated or automated.


Controls:


All Bands & Band 1/2/3/4 buttons

Compander On/Off - sets whether the compander is active.

Side-chain - sets whether the main input signal or the side-chain signal will be used to control the compander.

Threshold - adjust to get required effect.

Presets - when using presets, make sure to adjust the threshold and set the out gain as needed.

Expand ratio - expander ratio.

Comp. ratio - compression ratio.

Clamp time - how quickly the module responds to amplitude changes past the threshold.

Relax time - how quickly it releases after responding.

Make-up gain - boosts the level of the companded signal.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal.

Band level - this sets the output level of the band.


Global button

Lookahead - a lookahead delay can be added to input signal to allow transients to be detected early.

S-chain mode - sets whether or not the side chain signal will be split into 4 bands.

Expand range - sets the range of values allowed for each band's expander ratio.

Comp range - sets the range of values allowed for each band's compression ratio.

Clamp max - sets the maximum value of each band's clamp time.

Relax max - sets the maximum value of each band's relax time.

Split Freq 1 - sets the frequency to split bands 1 & 2.

Split Freq 2 - sets the frequency to split bands 2 & 3.

Split Freq 3 - sets the frequency to split bands 3 & 4.


Audio Input Busses:

Side-chain - mono audio signal which can be used to control the companders.


Control Modulation Input Busses:

Threshold 1/2/3/4

Expand Ratio 1/2/3/4

Compress Ratio 1/2/3/4

Attack 1/2/3/4

Release 1/2/3/4

Out gain 1/2/3/4

Dry-Wet Mix 1/2/3/4

Band level 1/2/3/4
