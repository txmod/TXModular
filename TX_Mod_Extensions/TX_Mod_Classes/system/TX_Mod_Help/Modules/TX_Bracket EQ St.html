Bracket EQ St


Module type: Audio Insert (stereo input, stereo output)


Introduction:

This module can "bracket" the frequencies of a sound, removing frequencies that are below the low-cut frequency or above the high-cut frequency.

Both frequencies can be modulated.


Controls:

Bracket EQ - set's the type of EQ used: 12db or 24db, Low-cut, Hi-cut, both or none.

Low-cut Freq - frequencies below this will be removed.

High-cut Freq - frequencies above this will be removed.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. This will usually be set to 1 - i.e. 100% affected signal.


Control Modulation Input Busses:

Low-cut Freq

High-cut Freq

Dry-Wet Mix
