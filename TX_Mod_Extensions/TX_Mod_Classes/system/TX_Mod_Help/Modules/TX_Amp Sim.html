Amp Sim


Module type: Audio Insert (mono input, mono output)


Introduction:

Ampllifier Simulator. This module adds distortion to the input signal, similar to the sound of vacuum-tube amplifiers. It adds a roughness and warmth to the signal. The effect produced depends on what signal is being distorted.


Controls:

Tube factor - how much tube-type distortion is added to the signal

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Control Modulation Input Busses:

Tube factor

Dry-Wet Mix
