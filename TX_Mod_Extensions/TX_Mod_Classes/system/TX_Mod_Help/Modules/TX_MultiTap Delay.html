MultiTap Delay


Module type: Audio Insert (mono input, stereo output)


Introduction:

This module puts a mono audio signal through eight delay taps which have separate delay time ratio, level and pan controls.

This module outputs a stereo audio signal.


Controls:

Clear delay - this button clears the delay, so any sounds in the delay buffer are silenced.


All controls button

Max time - this allocates more computer memory if you need longer delay times. Up to 5 minutes delay time can be set.

Tap tempo button - this displays measures the tempo when it is clicked with a mouse on each beat. Use the Command (Apple) key with a mouse drag to copy the BPM to the Delay time BPM.

Delay time - sets the largest delay time for all the delay taps. This is shown in milliseconds as well as beats per minute (bpm).

time x 2 , time x 3 , time / 2 , time / 3 - buttons to change the delay time.

Smoothing - sets the smooth time, up to 1 second, for any changes to the delay time.

Ratio - knobs for all tap ratios.

Level - knobs for all tap levels.

Pan - knobs for all tap pans.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Ratios button

Ratio Presets - various presets for all taps.

Ratio 1/2/3/4/5/6/7/8 - sets the delay time ratio for each tap. The ratio can be set as a fraction (e.g. 1/2 or 5/8) or a number between 0 and 1.


Levels button

Level Presets - various presets for all taps.

Level 1/2/3/4/5/6/7/8 - sets the level of each tap.


Pans button

Pan Presets - various presets for all taps.

Pan 1/2/3/4/5/6/7/8 - sets the pan of each tap Note: Left is 0, Right is 1.
