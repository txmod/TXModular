Amp Comp


Module type: Audio Insert (mono input, mono output)


Introduction:

Psychoacoustic amplitude compensation - frequency-dependent amplitude reduction.

This module is based on the idea that in human audio perception, higher frequencies are normally perceived as louder. To compensate for this, the level of the input signal is reduced for higher frequencies.

For example, when playing a single waveform across the whole keyboard - the bass can sometimes feel weak when played against higher notes. By using one of the Amp Comp presets, a suitable curve can be found where the higher notes are reduced to a suitable level. This can make it feel "more natural" when playing.

Various presets are given with different amounts of frequency-dependent amplitude reduction, based on 2 main options:

1. A-weighted psychoacoustic amplitude compensation - using ANSI standard A-weighting curve - "following measurements by Fletcher and Munson, the ANSI standard describes a function for loudness vs. frequency" (from SuperCollider help). The presets have various values for minAmp - the amplitude at the minimum point of the curve (around 2512 Hz). Lower values of minAmp mean more reduction.

2. Basic psychoacoustic amplitude compensation - this uses an exponential curve to reduce the level for higher frequencies. The presets have various values for exponent - how steep the curve decreases for increasing freq. Higher values of exponent mean more reduction.

Note - Amplitude compensation is found inside all Audio Source Modules which have a frequency parameter - for example Waveform, Vosim and Sample Player. On the other hand, this module can be inserted into any audio channel to be able to control its level based on the modulatable frequency.


Controls:

Amp Comp - select a preset for the type of compensation to use. Can be set to off when not needed.

Freq - this sets the frequency which controls the amount of amplitude reduction.
