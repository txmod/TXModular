Pitch Follower


Module type: Control Source


Introduction:

This module follows the pitch of a mono audio signal coming into it.

Once an pitch follower has been created, it will appear in the list of possible destinations for audio channels. This allows you to send an audio signal to it.

It outputs a control signal at the same level as the audio coming in.


Controls:

Pitch detector - this gives 2 options, slower & more accurate, or faster and less accurate. Both can be tried to see which works best.

Freq min-max - set the minimum and maximum frequencies that will be followed. There are presets for various musical instruments.

Follow time - time taken in milliseconds to follow the amplitude. Longer times will tend to smooth the output signal.

Cailbrate On - This button turns on the calibration process which automatically detects and set the minimum and maximum frequency values.

Cailbrate Off - This button turns of the calibration process.

Detect min & max [on/off] - to display the minimum and maximum values found in the current audio being analysed. (Note this uses more CPU power when detection is on, so only use when needed.)

Copy detected values to min/max - this copies the values found into the analysis controls above it.

Reset - resets the detector, so previous min/max values are ignored.

Automatically copy detected values to min/max [on/off] - automatically copies the values found into the analysis controls above it.

Detect Current Pitch - button to display current pitch.

Reset - resets the pitch displays.


Audio Input Busses:

Audio in - the incoming mono audio signal for amp following.


Control Modulation Input Busses:

Follow time
