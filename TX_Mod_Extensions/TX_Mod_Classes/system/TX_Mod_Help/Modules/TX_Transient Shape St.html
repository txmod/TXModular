Transient Shape St


Module type:  Audio Insert (stereo input, stereo output)


Introduction:

This module shapes the transients in a stereo audio signal.  

Attack and sustain of incoming audio can be separately controlled with the divide frequency affecting how attack and release are separated.

This module is useful with drums, plucked and percussive sounds which have strong transients.


Controls:

Divide freq - sets the frequency that divides the attack and sustain portions of sound. 

Attack - sets the attack level.  

Sustain - sets the sustain level.  

Dry-Wet Mix - this sets the mix of the original signal with the affected signal.  For send effects this will usually be set to 1 - i.e. 100% affected signal.
