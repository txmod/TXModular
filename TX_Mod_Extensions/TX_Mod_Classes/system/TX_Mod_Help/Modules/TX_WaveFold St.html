WaveFold St


Module type: Audio Insert (stereo input, stereo output)


Introduction:

This module distorts the input signal, by boosting the gain (amplitude) and adding harmonics using up to 4 layers of folding, wrapping, clipping and truncating (i.e. quantising).

Folding, wrapping, clipping methods change the signal if it goes below the Low Limit, or above the High Limit.

Truncating (quantising) rounds the signal using the High Limit value.

There are also high definition modes which use oversampling to reduce aliasing.

NOTE - the high definition modes HD4 or HD8 use much more computer resources. (If you need to check CPU usage, SuperCollider shows the average and peak CPU % for the Server at the bottom of the SC window).

An easy way to see the effects of this module is to show the signal on an oscilloscope while changing the controls.


Controls:


Layer 1/2/3/4 Mode - sets type of waveshaping for each layer - fold, wrap, clip, or truncate, with high definitions versions HD4 and HD8 (which use more computer CPU).

In Gain - cuts/boosts the signal before distorting it. This can affect how much distortion occurs.

Low Limit - signals below this level with be affected.

High Limit - signals above this level with be affected.

Harmonics - boosts the gain for each layer generating more harmonics.

Match Level of Input Signal - compensates for level changes caused by the distortion.

Out Gain - allows the dry and wet signals to be balanced in volume.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.
