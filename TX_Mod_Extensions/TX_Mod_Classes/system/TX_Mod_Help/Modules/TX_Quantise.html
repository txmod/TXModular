Quantise


Module type: Control Insert


Introduction:

This module quantises the input signal to a set number of equally placed steps.

For example, 5 steps would quantise the output signal to the 5 values: 0.0, 0.25, 0.5, 0.75, 1.0


Controls:

Input range - limits the range of value coming in. Values outside the range will be forced to be within range.

Output range - sets the range of values that the output signal will be mapped to.

Quantise - activates the quantise. If turned off, the input signal will be mapped to the output range without quantisation.

Steps - the number of steps in the output range.
