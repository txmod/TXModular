Table Drone


Module type: Audio Source (mono output)


Introduction:

This module outputs a synthesised waveform as a mono audio signal using wavetable synthesis, with optional sync, phase and frequency modulation.

Wavetables are single waveforms made up from a mixture of different harmonics. You can draw the harmonic levels for each wavetable or run various processes to create new wavetables.

This module can move between different wavetables to create smoothly changing waveforms.

There are 8 wavetables which can each be filled with a mix of up to 32 harmonics.

There are 5 banks which can be used to store and load sets of wavetables.

To modify the basic waveform, various phase processes can be optionally added:

Oscillator sync can reverse or reset the phase, using an internal oscillor or an external signal.

Phase modulation by biasing the phase rate can add distortion to the signal shape.

Frequency Modulation can also be added, using an external audio signal to modulate the phase.


Controls:


Wavetables button

Wavetables 1-8 - buttons to display the harmonics and waveforms of each wavetable.

Harmonics - a row of sliders which set the proportion of each of 8-32 harmonics. Drag across with a mouse to set values.
Behind the sliders, the wavetable signal is shown and it will change whenever the harmonic levels change.

Copy/Paste - for copying between different wavetables.

Reset - sets all harmonics to zero in current wavetable.

Load bank 1/2/3/4/5 - these buttons can be used to load wavetables stored in banks 1-5. Banks are places to store all 8 wavetables - they are useful for keeping wavetables during experimentation.

Store bank 1/2/3/4/5 - these buttons can be used to store wavetables in banks 1-5.

No. harmonics - sets the number of harmonics from a choice of 8, 16, 24 or 32.

Harmonic gap - normally the gap between each harmonic is 1, so the frequency of the 2nd harmonic is 2 times the first harmonic, and the 3rd harmonic is 3 times the first. Change this to other values to get different timbres (these are often sound more "buzzy" than with a gap of 1).The range of possible values is 0-3.

Scaling - This emphasises the difference between louder and quiter harmonics. Higher values create more of a difference - i.e quieter harmonics become much quieter. The default value is 1. The range of possible values is 1-5.

Table position - sets which wavetable, or mixture of wavetables, is being used for synthesis.

Stepping - if set to "None", there will be smooth changes in the waveform as the table position is moved. Otherwise, the changes will be stepped. The higher the value, the bigger the changes between steps. Stepping can be used for creative effects.

Level - sets the output level.

Amp Comp - select a preset for the type of amplitude compensation to use - see help file Amp Comp for details. Can be set to off when not needed.


Processes button - there are various processes which can be run on some or all of the wavetables to create a new set of wavetables.

Some processes transform the current wavetables, others generate completely new wavetables.

For all processes, you can decide how much of the original wavetable is mixed with the new one.

This can be useful, for example, if you want to add a small amount of randomness to the harmonic values of the wavetables. You could achieve this by using the process "Generate: random harmonic levels" and using a small mix percent (e.g. 5 percent).

Wavetables 1-8 - see above.

Harmonics - see above.

Copy/Paste - see above.

Reset - see above.

Load bank 1/2/3/4/5 - see above.

Store bank 1/2/3/4/5 - see above.

Process - sets the process

Processing parameters - start & end values - each processing parameter has two values which will be used for the first and last wavetables selected.

For example if wavetables 3-6 are selected for processing, then the start value will apply to wavetable 3 and the end value to wavetable 6.

All wavetables in between will have proportionate amounts in between the start and end values.

For examples of this, try running the various processes marked "Generate" using the default parameter values.

Run Process - button to run the process on the selected wavetables

Wavetables - select which wavetables will be processed.

Mix percent - of the original and processed wavetables will be mixed together. 100 percent mix will mean only processed wavetables.

other parameters - processes may have other parameters which will be labelled. These all have start and end values.


Base Frequency button

Base Freq - sets the pitch or frequency of the waveform

OFF/ON Use note list instead of variable frequency - if set on, this uses a specific note list which the waveform's frequency is chosen from. The controls below allow you to specify the which notes will be in the note list.

Chord/ Scale - sets the interval, scale, mode or chord used for the note list.

Key / root - sets the key or root of the above scale.

Note range - sets the min and max of notes in the note list using midi note numbers.

Note count - shows the total number of notes in the current note list.

Note select - this slider selects one of the notes in the current note list. The lowest note is with the slider set to 0, with the highest at 1, and all others in between.The more notes there are in the note list, the more closely they will be packed along the length of the slider. This can be modulated, for example with an lfo or midi controller.


Freq Smoothing button

This allows you to add smoothing to the frequency parameter.

See help file Smooth


Sync PD FM Smooth button

Sync mode - sets the type of sync, options are:
Sync Off
Reverse phase on cycle start & on sync trigger
Reset phase on cycle start, reverse phase on sync trigger
Reset phase on cycle start & on sync trigger

Sync trigger - sets the source of sync triggers, options are:
Internal - use [Sync ratio x Base freq] as sync frequency
External Sync audio input on upwards slope
External Sync audio input on downwards slope
External Sync audio input on upwards or downwards slope

Sync freq - sets the frequency of the internal sync oscillator

Sync ratio - sets the frequency ratio of the internal sync oscillator

Sync threshold - sets the minimum level which the phase must be at for a sync trigger to work.

Add Phase Distortion (PD) - introduces changes to the phase rate so the centre of the phase is earlier or later in the cycle

Phase Distort - sets the amount of forwards or backwards bias

Frequency Modulation using FM input - If set on, the FM Input audio side-chain input bus will be used to modulate the phase of the waveform, scaled by FM depth.

FM depth - scales the amount of modulation

Smooth filter - 2 types of filter are available, a Median filter or a Smoothing filter which use the previous 3-31 audio samples to smooth the output. Each has a different sound.


Audio Input Busses:

FM input - the incoming mono audio signal used for frequency modulation (if switched on).

Ext Sync input - the incoming mono audio signal used for oscillator sync (if switched on).
