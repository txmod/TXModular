Loop Player


Module type: Audio Source (mono output, triggered)


Introduction:

This module takes a mono loop from the loop bank and plays it back.

It can be triggered in various ways such as by MIDI notes, other modules (e.g. MultiTrack Seq, Step Sequencer, Action Sequencer, Audio Trigger) or action buttons on an interface screen.

Granular time-stretching and pitch-shifting can be applied to the loop.
Granulation allows independent control of pitch and timing, for example keeping the pitch steady while the bpm or "Groove" is changed.

Morphable curves can be used to change the "Feel" or "Groove" of the loop.

Feel curves change the level of the loop while it plays.

Groove curves change timing of the loop while it plays. They can also rearrange the order of the beats in a loop, or reverse them.
If time-stretch granulation isn't being used, Groove curves will also affect the pitch of the loop.

Note: Feel curves are horizontal (drawn left to right) & Groove curves are vertical (drawn top to bottom)

Controls:


Loop button

Mono loop - select from all the mono loops in the system's loop bank. Stereo loops from the loop bank won't be shown.

Loopbank Settings - the settings for original bpm and total beats of the loop are displayed, taken from the settings in the loopbank. These can be edited in the loopbank screen.

Add loops to loop bank - to add any audio loops stored on the hard disk.

Show/Hide - buttons for the display of the loop waveform. When the waveform is displayed, you can zoom in and out using the zoom buttons. You can also select which part of the waveform is played by using the select all/ select none buttons.

Start/End - the start and end of the loop section that will be played. These are shown as fractions (e.g. 7/8), as well as numbers between 0 and 1.
Fractions can be useful for selecting particular sections of rhythmic loops - such as 2/12 to 8/12.

Loop type - there are three options:
Looped - play while gated - play looped from start to end point while gate is on (default)
Single shot - play while gated - play once from start to end point while gate is on
Single shot - play whole loop - play once from start point to end point

Reverse - if on, loop playback will be reversed.

Level - the output level of the loop player.

Play BPM - The BPM that the loop will be played back at.
Note: If time-stretching is not being used, the pitch of the loop will go up or down compared to the original loop, depending the how different the play bpm is from the original bpm. If the play bpm is lower, the pitch will also be lower.

Speed factor - A quick way to change the playback speed.
Higher numbers will increase the speed, lower numbers will decrease the speed.
A setting of 0 will be normal speed, a setting of 1 will be double the speed, a setting of -1 will be half the speed,


Env, MIDI, Test button

Env Curve - this selects the curve for the loop envelope.

Attack - the attack time of the envelope.

Release - the release time of the envelope.

Midi Listen - this allows the module to be triggered by a midi note.

Midi Chan. - set the midi channels the module will listen to.

Note range - which notes will trigger the module.

Vel range - which velocities will trigger the module.

Test Loop Settings: - these set the velocity and play time of the test loop that is played when the Play test loop button is pressed.


Time Stretch button

Time Stretch - this sets the type of granular time-stretching used.

Window size - how long each grain is.

Overlaps - how many overlapping grains are used.

Interpolation - this sets the quality when pitch-shifting grains. Cubic is best though more computationally intensive.

Randomness - this randomises the timing of the grains which can help to blur the effect of the granulation.

Pitch-shift - this changes the pitch of the grains, shown in semitones.


Feel 1 button

Feel mode - sets which Feel curve (1 or 2 or both) will be used.

Feel multi - sets the number of times the Feel curve will be applied across the selected part of the loop.
For example if the loop was 8 beats long and the Feel multi was set to 2, the Feel curve would be applied twice - on each set of 4 beats.

Add Feel after Groove - if switched on, the Feel curve will be applied after the Groove curve.

Feel morph - if the Feel mode is set to morph, this will morph between curves 1 & 2.

Show Waveform - this displays the loop waveform behind the curve.
(Note: if Feel multi is more than 1, if the Waveform is shown it won't correspond to the Feel curve since they are in different timescales. This is also true if Add Feel after Groove is switched on since the Groove curve can change the playback order.)

Hide - this hides the loop waveform.

Grid rows - no of rows (1-128) separated by white lines in the grid behind the curve.

Grid columns - no of columns (1-128) separated by white lines in the grid behind the curve.

Feel 1 Curve - the Feel curve used with the loop. You can edit the curve with a mouse.
See help file Curve Editing for details.


Feel 2 button

Same as Feel 1.


Groove 1 button

Groove mode - sets which Groove curve (1 or 2 or both) will be used.

Groove multi - sets the number of times the Groove curve will be applied across the selected part of the loop.
For example if the loop was 8 beats long and the Groove multi was set to 2, the Groove curve would be applied twice - on each set of 4 beats. )

Add Feel after Groove - if switched on, the Feel curve will be applied after the Groove curve.

Groove morph - if the Groove mode is set to morph, this will morph between curves 1 & 2.

Show Waveform - this displays the loop waveform behind the curve.
(Note: if Groove multi is more than 1, if the waveform is shown it won't correspond to the Groove curve since they are in different timescales.)

Hide - this hides the loop waveform.

Grid rows - no of rows (1-128) separated by white lines in the grid behind the curve.

Grid columns - no of columns (1-128) separated by white lines in the grid behind the curve.

Groove 1 Curve - the Groove curve used with the loop. You can edit the curve with a mouse.
See help file Curve Editing for details.


Groove 2 button

Same as Groove 1.


Play button

This starts the test loop playing.

The velocity and time of the loop that is triggered are set using the Env, MIDI, Test button.


Stop button

Stops the test loop playing.
