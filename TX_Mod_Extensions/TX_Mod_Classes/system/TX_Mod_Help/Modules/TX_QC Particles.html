QC Particles


Module type: Action


Introduction:

This module creates a new window which contains a video screen running a 3D particle system animation (this is described below).


Quartz Composer 3D Particle System

The 3D particle system animation was created using Quartz Composer software which comes with OSX. It is a tool for creating simple interactive visual effects.

The controls affect how the particles in the animation behave:

Each particle is created at a position specified by the X/Y/Z position, with color (as hue, saturation, brightness) and lifetime defined. The initial velocity of the particle is randomly chosen in the range specified by the X/Y/Z Min/Max velocity inputs. Its initial size is randomly chosen in the range specified by the Min/Max size inputs.

The particle movement is then affected by gravity and by the attraction relative to its initial position, its size is reduced by "Size Delta" per second and its opacity is also reduced by "Opacity Delta" per second. When the particle reaches its lifetime, it is regenerated.
