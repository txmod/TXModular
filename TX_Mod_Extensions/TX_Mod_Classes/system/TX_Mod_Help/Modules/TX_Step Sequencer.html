Step Sequencer


Module type: Control Source


Introduction:

This module can trigger notes in other polyphonic modules as well as outputting control signals for note and velocity values and 3 extra controls.

The sequencer holds patterns of up to 64 steps. There are 100 pattern slots (numbered 0 - 99) which can be chained together to make longer patterns.

The sequencer BPM can be modulated by other modules.

It is possible to use a sequencer just for triggering notes in other modules without using the control signals that it outputs. In this case, it doesn't need to be in a channel (i.e. the channel that is automatically created when you create a new Step Sequencer can be deleted).

On the other hand, it is possible to use a sequencer just for it's control outputs, as a kind of step LFO. In this case you would need to use channels.

You can easily introduce randomisation to the sequence in various different ways.

It is also possible to manually trigger the sequencer step by step.

Note: In this module, the maximum note velocity is 100, whereas trigger modules use the MIDI standard of 127.

See also: Modulation Vs Automation

Controls:


Controls on all Step Sequencer screens:

Start -button to start the sequencer playing from the first step in the play range.

Stop - button to stop the sequencer.

Sync Start - this checkbox sets whether the sequencer will be started when the "Sync Start" button is pressed. The button is located at the top of the main screen and is used to start multiple modules at the same time.

Sync Stop - this checkbox sets whether the sequencer will be stopped when the "Sync Stop" button is pressed. The button is located at the top of the main screen and is used to stop multiple modules at the same time.


Controls on several Step Sequencer screens (Patterns, Steps, Groups, Processes):

Step - shows which steps are displayed .

Step on/off - if a step is turned off, no note is triggered for that step.

clone 1 - button which is at the end of lots of parameter rows in the sequencer. When pressed the value of the first step is copied to all the other steps.

Pattern - shows the current Pattern number followed by the MIDI note names (e.g. C3) of each step displayed.


Outputs button

Polyphonic modules to be triggered directly by the sequencer - up to 6 polyphonic source modules, such as wave synths, samplers or polyphonic envelopes, can be triggered. All valid modules will appear in the modules list. See the Help Index for details of which modules are polyphonic.

MIDI Port - for MIDI output

MIDI Loop-Back - for sending MIDI output back into the TX Modular

MIDI Channel - for MIDI output

Output MIDI notes for each step - turns on MIDI note output

Output MIDI control values for Control 1/2/3 values at each step - turns on MIDI control output

Control 1/2/3 MIDI - the MIDI control numbers used


BPM & Options button

BPM - sets the tempo in beats per minute.

Tap tempo button - this displays measures the tempo when it is clicked with a mouse on each beat. Use the Command (Apple) key with a mouse drag to copy the BPM to the sequencer BPM.

Auto copy tap tempo to BPM - when set on, this automatically copies the tap tempo to the sequencer BPM as long as the tap tempo is within the min-max range of BPM values.

Play Mode - this has 3 options: 1) Normal,

2) Random Trigger - steps are randomly triggered based on BPM. Step lengths are ignored. - when this is on, it changes the sequencer's rhythmic base; instead of using step lengths to created rhythms, a random trigger is used with an average trigger rate based on the current bpm.

3) Manual Trigger - steps are manually triggered. BPM & Step lengths are ignored.- when this is on, once the sequencer is started, the following will appear on the screen.

Manual Controls:

Trigger status - displays current info about manual triggering.

Rewind Sequencer button - rewinds to the start of the sequence.

Trigger Next Step button - manually triggers next play step.

Skip Next Step button - manually skips next play step.

Mute - sequencer plays but all steps are turned off - this stops all steps from triggering.

Randomise step order every cycle - when this is on, the step order will be scrambled every time a pattern plays, but only for selected parameters (the parameters are only shown when this button is switched on).



Patterns button

The Patterns button displays Pattern Slots (from 0 to 99) which can be used to store sequences and recall them.

Play steps - selects which steps will be played in the current Pattern.

Pattern time - displays the length of the current Pattern in beats and seconds. (Please note: the length in seconds is based on the BPM set inside the sequencer. But, since the BPM can be modulated, the length in seconds may not reflect the current modulated BPM.)

Pattern desc - space to add a description for each pattern.

Pattern slot - slot number of the current Pattern.

Copy pattern/ Paste pattern - buttons to copy and paste betweeen patterns.

Reset all values - will reset all the paramenters of the whole current pattern to default values.

Next pattern has various options:

- None will mean the current pattern only plays once;

- Repeat current pattern will loop the current pattern;

- Use pattern chain - in chain step order allows you to create your own chain of different patterns - up to 64 can be chained and you can loop the chain using the Loop button

- Use pattern chain - in random order randomises the order of the chain steps.


Note: Sequencer steps cannot be edited while Sequencer is playing a Pattern Chain.

You can still edit controls on the Outputs and BPM & Options screens, but the other screens will be disabled while the sequencer is playing.

To edit a Pattern while playing it, set Next pattern to None or Repeat current pattern.


Steps button

Here, you can edit the notes and other parameter of each step.

A piano keyboard shows all the possible notes.

The grid next to the keyboard can be clicked with a mouse to input notes and has scroll bars to move around it.

The green buttons allow you to edit various parameters for each step:

The notes button displays a 3 octave keyboard (it is only 2 octaves on the other buttons).

It also displays a checkbox: MIDI learn notes & velocities. When this is switched on, any MIDI notes and velocities received on any MIDI channels will be recorded into the steps of the sequencer starting at the Next Step, which is shown next to it. The new notes will be displayed on the screen.

This is useful adding a pattern played on a MIDI keyboard.

The velocity button displays the velocity values for each step, and the Velocity o/p range, which rescales the velocity to within a certain output range.

Note - the clone 1 button will copy the value in the first step to all other steps.

A random velocity can also be set for each note. This is useful for so-called "humanisation" of of patterns to make them sound less machine-like.

The step/note lengths button displays the length in beats of each step/note as a fraction: X over Y. So for example 1 over 2 would be a step length of 1/2 or 0.5 beats. Or 3 over 4 would be a step length of 3/4 or 0.75 beats. A checkbox shows various presets - select a preset to set all step & note lengths to the preset value.

The probability button displays the probablity of each step playing.

A probabililty of 100 would mean that the step would always be played.

A probabililty of 33 would mean that the step would be played 33 percent of the time - on average only every 3rd time the sequence is played. This is useful for creating sequences that change every cycle, with some steps playing every time, and others only sometimes.

The delay button displays the delay amount of each step. A step can be delayed by up to 50% of the step time.

random delay adds a random delay to each step. This is useful for so-called "humanisation" of the timing of patterns to make them sound less machine-like.

Delay can also be used to make a swung rhythm by delaying every second step.

random octave - if this is not zero, the note will be transposed by a random no of octaves between 0 and that value. For example a value of 2 will mean every time that step is played, it could be transposed by 0, 1, or 2 octaves.

The random octave is set every time a step is played. Negative values are allowed which will transpose downwards to lower octaves.

control 1/2/3 - these controls are not used for creating notes, but they are output as control signals at the same time as notes are played in each step.


Groups button

Group 1-8 - There are 8 groups and any step can belong to any number of groups.

Groups can used by to dynamically control a sequencer while it is running by sending an instruction to all the steps in a group.

You would do this by running an Action from another module.

For example, an Action Sequencer could instruct all the steps in Group 1 to have their probability set to 50%, or their velocity set to 60. You can also use widgets (such as sliders) on Interface screens to control group parameters.

- For more info on Actions in the TX Modular System, see help file: What_Are_Actions

(Example System: To see an example of how Groups can be used in a built system, see the file Drunken_Sailor in the Larger_Systems folder within the TX Mod Examples folder.)


Processes button

The Processes button allows you to apply all kinds of processes to change the pattern currently being displayed.

You can copy and paste steps from one part of the pattern to another, randomise the order of steps, or generate new patterns based on specific scales, chords or modes.

Each process has its own specific options and parameters to set. The best way to understand them is to try them out.

All processes have a parameter Target steps - this is the range of steps that will be affected when you run the process. Steps outside this range won't be changed in any way.

The following processes can be run on the current pattern:

Copy and paste steps once

Copy and paste steps repeatedly

Shift steps left

Shift steps right

Randomise step order

Reverse step order

Constrain notes to chord, mode, scale

Generate notes from chord, mode, scale

Generate notes from selected range

Transpose notes upwards

Transpose notes downwards

Randomise step on-off values

Randomise velocity values

Randomise step length X values

Randomise step length Y values

Randomise note length X values

Randomise note length Y values

Randomise probability values

Randomise delay values

Randomise control 1/2/3 values
