Filter Synth


Module type: Audio Source (mono output, triggered)


Introduction:

This module creates synth notes by filtering a mono audio signal coming into it and creating enveloped notes from the filtered audio.

Once a filter synth module has been created, it will appear in the list of possible destinations for audio channels. This allows you to send an audio signal to it from another audio channel.

Filter synth notes can be triggered in various ways such as by MIDI notes, other modules (e.g. Step Sequencer, Audio Trigger) or action buttons on an interface screen.

Controls:


Filter button

See help file Filter for details.

Level - sets the output level.

Amp Comp - select a preset for the type of amplitude compensation to use - see help file Amp Comp for details. Can be set to off when not needed.

See help file MIDI Note Button for details.
