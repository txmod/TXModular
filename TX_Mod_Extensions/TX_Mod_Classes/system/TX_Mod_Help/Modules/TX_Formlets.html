Formlets


Module type: Audio Insert (mono input, mono output)


Introduction:

This module uses a group of 1-8 resonant filters to add formants to the input signal. The filter impulse response is like that of a sine wave with an attack-decay envelope over it.

The controls for each filter layer are frequency, attack, decay and amplitude.

If the attacktime equals the decaytime then the signal cancels out. If the attacktime is greater than the decaytime then the impulse response is inverted.


Controls:

Gain button

In gain

Out gain

Level control - options:

None - level unchanged

Balance - match level to dry signal level

Limiter - keep the signal level below 0db

Normalize - force the signal level to just below 0db

Tanh Clipping - uses hyperbolic tangent of signal to clip it (this adds some distortion)

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Parameters button

Freq range - the range of frequencies for the filters layers shown as midi notes

Quantise - forces all frequencies to the nearest midi note

Pitch bend - to globally change all the layer frequencies

Max attack - maximum attack time for all the layers

Max decay - maximum decay time for all the layers


Layers button

Complexity - the number of layers

MIDI learn notes & amps - when this is switched on, any MIDI notes and velocities received on any MIDI channels will be recorded into the layers starting at the Next Layer, which is shown next to it.
