Split 2-Band


Module type: Audio Source (mono input & 2 mono outputs)


Introduction:

This module splits a mono audio input into 2 mono frequency bands.

Once a module has been created, it will appear in the list of possible destinations for audio channels, so you can send an audio signal to it.


Controls:


Split Freq - sets the frequency to split bands 1 & 2.

Output level - sets the overall output level.


Audio Input Busses:

Input


Audio Output Busses:

Band 1

Band 2
