Scale Chord


Module type: Action


Introduction:

This triggered module makes no sound itself but instead triggers notes in other polyphonic modules. Up to 6 different modules can be triggered. It can also output MIDI notes.

When it is triggered, it restricts the output notes to a particular key and scale (if the chromatic scale is chosen, then any notes played are be part of the scale).

Optionally, a chord can be played instead of just a single note . An open version of the chord can be chosen, with selectable octave transposition to spread the chord. Each note in the chord has both velocity and probability settings, allowing randomness to be introduced in various ways.


Controls:

Active - if off then no notes will be triggered

MIDI & Outputs button

MIDI Listen - this allows the module to be triggered by midi Notes, and to respond to midi Pitchbend.

Note: If Midi Listen is turned on make sure none of the individual output modules has their Midi Listen turned on, or there will be double triggering.

MIDI Channels - which midi channels the module will listen to.

Modules - up to 6 polyphonic source modules, such as wave synths, samplers or polyphonic envelopes, can be triggered. All valid modules will appear in the modules list. See the Help Index for details of which modules are polyphonic.

Output MIDI notes - on/off.

MIDI Port - select from available ports

MIDI Loop-Back - sends MIDI notes back into the TX Modular system on the selected channel, so other modules can respond.

Note: Take care not to output MIDI loop-back to a channel that the module itself is listening to - otherwise the system will crash with a MIDI feedback loop.

MIDI Channel - output channel.

MIDI keyboard- this can be played by clicking with a mouse.The midi velocity of the note played is set by the Velocity slider. Middle C is tinted red. Notes that are part of the selected scale are tinted yellow.


Key & Scale button

Key - C to B.

Ignore key, scale follows played note - for fixed chord shapes - when on, this means that any note played will be valid and the selected scale will be used as the key for all chords generated. For example, if the scale were Major, then all chords generated from notes played will be Major chords.

When this is off, then chords will depend on where the played note is within the scale in the selected key only. So for example, in the scale of C Major, when a D note is played a triad chord created would be a D Minor triad( i.e. D, F & A), since it is forced to be within the notes of the C Major scale.

Scale - various options showing scale names and notes per octave.

Silence notes played outside the scale - if off, outside notes will be transposed to the nearest note in the scale.

Transp Octs - transposes all played notes up or down number of octaves.

MIDI keyboard- this can be played by clicking with a mouse.The midi velocity of the note played is set by the Velocity slider. Middle C is tinted red. Notes that are part of the selected scale are tinted yellow.


Chord button

Play Chord - if on, chords instead of single notes will be output when the module is triggered.

Open Chord - spreads the chord by transposing selected notes.

Inversion - changes the note order by transposing the selected number notes up or down one octave

Randomise vel - adds random variation the velocity.

Velocity scale - if less than 1, all chord notes will have their velocity scaled down.

Scale note velocities to played velocity - this will scale all chord notes using the played note’s velocity.

Use played velocity for note 1 - this overrides the note settings. (This might be useful for example, if “Randomise velocity note order” is on, to make sure that the root of the chord is as played.)

Use probability 100% for note 1 - this overrides the note settings. (Again, this might be useful for example, if “Randomise probability note order” is on, to make sure that the root of the chord is always played.)

Randomise velocity note order - this randomly orders the velocity of all the selected notes.

Randomise probability note order- this randomly orders the probability of all the selected notes.

Chord presets - some standard chord shapes setting which notes are played.

Notes 1-12, Oct+, Oct- - which notes will play in the chord.

Open transp - when Open Chord is selected, certain notes in the chord will be transposed as selected.

Velocity - for each note.

Probability - for each note.
