Bowed


Module type: Audio Source (mono output, triggered)


Introduction:

This module creates polyphonic synthesised notes based on a digital wave guide physical model of a bowed instrument, which can optionally include torsional waves.

Notes can be triggered in various ways such as by MIDI notes, other modules (e.g. Step Sequencer, Audio Trigger) or action buttons on an interface screen.


To create the full sound of a bowed instrument, this module should be followed by Bowed Res which is an audio insert module that models a soundboard resonator for a bowed instrument.


Controls:


Envelope button

This is for the amplitude (volume) envelope for each note.

See help file Env DADSR for details.

Amp Comp - select a preset for the type of amplitude compensation to use - see help file Amp Comp for details. Can be set to off when not needed.

Bow Model button

Controls to change the model.

Torsional Waves button

Controls to add torsional waves to the model.

MIDI/ Note button

See help file MIDI Note Button for details.
