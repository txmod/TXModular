File Player


Module type: Audio Source (mono output)


Introduction:

This module plays mono audio files (such as wave or aiff files) from the hard disk.

Useful for long files that would be too large to play in a sampler (which uses RAM).


Controls:

Open new file - selects which audio file will be played.

Fade-in time - time taken for fade in.

Fade-out time - time taken for fade out.

Level - output level.

Loop file - This checkbox sets whether the file will loop when it reaches the end. N.B. If the file has been playing and reaches the end with the Loop file checkbox off, then if Loop file checkbox is switched on, the file will immediatedly continue playing from the start.

Play speed - sets the speed of playback, with 1 being normal speed.

Sync Start - sets whether the File Player will be started when the "Sync Start" button is pressed. This button is located at the top of the main screen and is used to start multiple modules at the same time.

Sync Stop - sets whether the File Player will be stopped when the "Sync Stop" button is pressed. This button is located at the top of the main screen and is used to stop multiple modules at the same time.

Rewind - button to rewind to start of file

Play from start - button to start playback from the start of the file.

Play - button to start playback. Playback will be from the current position in the file (wherever it last stopped).

Fade out - button to fade out and stop file playing

Stop - button to stop file playing immediately
