Distortion St


Module type:  Audio Insert (stereo input, stereo output)


Introduction:

This module can distort the input signal in various ways. Distortion adds a roughness and/or warmth to the signal. The effect produced depends on what signal is being distorted.


Controls:

Distortion - selects the type of distortion 

In Gain - cuts/boosts the signal before distorting it. This can affect how much distortion occurs. 

Depth - this is only used in the "Waveshape with Depth" type of distortion.

Out Gain - useful for reducing the volume of the distorted signal. This allows the dry and wet signals to be balanced in volume. 

Dry-Wet Mix - this sets the mix of the original signal with the affected signal.  For send effects this will usually be set to 1 - i.e. 100% affected signal.
