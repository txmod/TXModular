Mono env DADSR


Module type: Control Source (triggered)


Introduction:

This module creates a mono DADSSR (Delay Attack Decay Sustain1 Sustain2 Release) envelope as a control signal.

This is similar to a DADSR envelope but having start and end sustain levels allows many more envelope shapes to be created, for example with a gradual decay in the sustain part of the curve.

It can be triggered in various ways such as by modulation of the gate input, by other modules (such as an Action Sequencer) or by action buttons on an interface screen.

The Envelope Type can be set to Sustain Until Released or Fixed Length.
A Sustain Until Released envelope will pause at sustain level 2 until the end of the trigger or gate.
A Fixed Length envelope will complete every stage of the envelope.


Controls:

Env presets - presets for various standard envelope shapes.

Envelope display box - this shows each stage of the envelope by level (up-down) and time (left-right).

Pre-Delay - the delay time before the envelope begins.

Attack - the attack time of the envelope

Decay - the decay time of the envelope

Sustain level - the level of the start of the sustain part of the envelope.

Sustain level 2 - the level of the end of the sustain part of the envelope.

Sustain time - the length of the sustain part of the envelope.

Release - the release time of the envelope

Curve - this sets the shape of the curve for all parts of the envelope.

Env. Type - this can be set to Sustain Until Released or Fixed Length.
A Sustain Until Released envelope will pause at sustain level 2 until the end of the trigger or gate.
A Fixed Length envelope will complete every stage of the envelope.

Level scale - scales the level of the curve.

Gate - sets the gate to trigger or stop the envelope.


Control Modulation Input Busses:

Gate

Delay

Attack

Sustain level

Sustain level 2

Sustain time

Output Busses:

Env Out - the envelope signal

Env End Trig - outputs a trigger when the envelope ends
