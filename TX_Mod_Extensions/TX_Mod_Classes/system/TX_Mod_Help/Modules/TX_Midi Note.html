Midi Note


Module type: Control Source


Introduction:

This module outputs 3 control signals based on the midi note messages received: Note value, Note On trigger, and Note Off trigger.
The Note value output is scaled to the Output range.
The Note On and Off trigger outputs will be set to 1 momentarily when triggered. Otherwise they will output 0.


Controls:

Midi channels - set the midi channels the module will listen to (outside this range are ignored).

Note range - which notes to listen to (outside this range are ignored).

Vel range - which velocities to listen to (outside this range are ignored).

Output range - of the Note value output, representing the midi note range. The preset output ranges include negative options.
