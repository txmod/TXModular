Sample & Hold


Module type:  Control Insert


Introduction:

This module outputs a control signal based on the input signal as well as a trigger input signal coming in on a control modulation input bus. 

Two types of processing are available:  Sample & Hold and Gate & Hold - see below for details.


Controls:

Type -  there are two options:

  Sample & Hold - outputs input signal value when trigger input signal goes above zero. Holds this value until next trigger.

 Gate & Hold - allows input signal through when trigger input signal is above zero,  otherwise holds last value.

Trigger - this button will trigger the sample and hold module. It is useful for testing. 


Control Modulation Input Busses:

Trigger - the trigger input signal.  A trigger happens when the signal goes above zero.

A trigger signal could come from several other modules such as Trigger Impulse, Env DADSR and LFO.


See the example files SampleAndHold and TriggerImpulse.
