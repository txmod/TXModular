Env DADSSR


Module type: Control Source (triggered)


Introduction:

This module creates polyphonic DADSSR (Delay Attack Decay Sustain1 Sustain2 Release) envelopes as control signals.

This is similar to a DADSR envelope but having start and end sustain levels allows many more envelope shapes to be created, for example with a gradual decay in the sustain part of the curve.

The Envelope Type can be set to Sustain Until Released or Fixed Length.
A Sustain Until Released envelope will pause at sustain level 2 until the end of the trigger or MIDI note.
A Fixed Length envelope will complete every stage of the envelope.


Controls:


Env presets - presets for various envelope shapes.

Envelope display box - this shows each stage of the envelope by level (up-down) and time (left-right).

Pre-Delay - the delay time before the envelope begins.

Attack - the attack time of the envelope

Decay - the decay time of the envelope

Sustain level - the level of the start of the sustain part of the envelope.

Sustain level 2 - the level of the end of the sustain part of the envelope.

Sustain time - the length of the sustain part of the envelope.

Release - the release time of the envelope

Curve - this sets the shape of the curve.
Some options are for one curve that applies to all parts of the envelope.
Other options give an upwards curve for the attack stage, and a downwards curve for the decay, sustain and release stages.

Env Type - Sustain Until Released or Fixed Length.
A Sustain Until Released envelope will pause at sustain level 2 until the end of the trigger/gate/MIDI note.
A Fixed Length envelope will complete every stage of the envelope.


MIDI button:

Midi Listen - this allows the module to be triggered by a midi note.

Midi Chan. - set the midi channels the module will listen to.

Note range - which notes will trigger the module.

Vel range - which velocities will trigger the module.

Polyphony - maximum number of simultaneous envelopes.


Trigger Envelope button - this will trigger an envelope for 1 second whenever it is pressed.
