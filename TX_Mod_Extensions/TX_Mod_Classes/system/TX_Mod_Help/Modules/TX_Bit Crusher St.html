Bit Crusher St


Module type:  Audio Insert (stereo input, stereo output)


Introduction:

This module distorts the input signal in by reducing the sample rate and bit size.


Controls:

Process - selects the type of process used. Options are:  Bit Crusher/ Decimator/ Smooth Decimater. 

In Gain - cuts/boosts the signal before distorting it. This can affect how much distortion occurs. 

Sample rate - set the sample rate - low values are more lo-fi.

Bit size - set the bit size - low values are more lo-fi. 

Smoothing - set the bit size - low values are more lo-fi. 

(Note - not all processes use Bit size and Smoothing. The Process name will say which controls are used.)

Dry-Wet Mix - this sets the mix of the original signal with the affected signal.  For send effects this will usually be set to 1 - i.e. 100% affected signal.


Control Modulation Input Busses:

Sample rate

Bit size

Dry-Wet Mix
