Chorus


Module type: Audio Insert (mono input, mono output)


Introduction:

This module creates a chorus effect on a mono audio signal using several modulated delay lines, with the delay time being modulated by low frequency oscillators (LFOs), with slightly different frequencies for each delay.


Controls:

No. Layers - the number of delay lines used, from 2 to 7.

LFO Wave - selects different kinds of waveforms for the LFO.

LFO Rate - the frequency of the LFO.

LFO Depth - the amount of modulation. If set to 0, the flange time will stay constant.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Control Modulation Input Busses:

LFO rate

LFO depth

Dry-Wet Mix
