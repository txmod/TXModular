Disintegrator


Module type: Audio Insert (mono input, mono output)


Introduction:

This module distorts the input by amplifying random half-cycles of the signal.


Settings to try:

Set Multiply to 0 and increase Probability from 0 to 1for disintegrating fade effect.

Set Probability to 0.5 and decrease Multiply from -1 to 1 for a gradual distortion effect.


Controls:

Probability - probability of amplifying each half-cycle of the signal.

Multiply - the value used to amplify selected half-cycles.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Control Modulation Input Busses:

Probability

Multiply

Dry-Wet Mix
