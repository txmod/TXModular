Matrix A 8x8


Module type: Audio Source (8 outputs)


Introduction:

This module mixes 8 audio input signals into 8 output channels using a 64-knob matrix.

Once the module has been added, it's inputs will become valid destinations for any audio channel.

All the inputs and outputs can be named.


Controls:

Input 1 ... 8 - each row of 8 knobs sets the level sent to outputs 1... 8.

Auto scale (output reduced if mixed level headroom exceeds 1) - automatically scales the mixed level so it is within system limits.

Output Level - sets the output level of the module.


Audio Input Busses:

Input 1 ... 8


Control Modulation Input Busses:

In 1 -> Out 1

In 1 -> Out 2

In 1 -> Out 3
