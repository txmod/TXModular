Slope


Module type: Control Insert


Introduction:

This module measures the velocity or acceleration of a control signal.


Controls:

Slope type - set whether velocity or acceleration is measured. Velocity is the slope of a signal. Acceleration is is the slope of the slope of a signal.

Sensitivity - set how sensitive the measurements are. The default value is 1. It can be useful to have check the module output running with

Output - sets the range of values for the output. This can be either -1 to +1, or 0 to +1. The second option is for those situations where you only want to measure the amount a signal has changed, not whether the change was positive or negative.


Control Modulation Input Busses:

Sensitivity
