DistortionMB (Multi-Band)


Module type: Audio Insert (mono input, mono output)


Introduction:

This module splits the input signal into 4 frequency bands and adds distortion, giving a roughness and/or warmth to the signal. The effect produced depends on what signal is being distorted and the type of distortion.


Note: Split frequencies cannot be modulated or automated.


Controls:


Bands Button

Distortion 1-4 - selects the type of distortion for each band

In Gain 1-4 - cuts/boosts the signal before distorting it. This can affect how much distortion occurs.

Depth 1-4 - this is only used in the "Waveshape with Depth" types of distortion.

Distort Gain 1-4 - useful for reducing the volume of the distorted signal. This allows the dry and wet signals to be balanced in volume.

Dry-Wet Mix 1-4 - this sets the mix of the original signal with the distorted signal.

Band Level 1-4 - sets the output level for each band.


Band-Split Frequencies Button

Split Freq 1 - sets the frequency to split bands 1 & 2.

Split Freq 2 - sets the frequency to split bands 2 & 3.

Split Freq 3 - sets the frequency to split bands 3 & 4.


Control Modulation Input Busses:

In Gain 1-4

Depth 1-4

Dry-Wet Mix 1-4

Band Level 1-4
