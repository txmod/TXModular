EQ6M


Module type: Audio Insert (mono input, mono output)


Introduction:

This module has 6 switchable bands of EQ: Lo-shelf, Parametric 1 - 4, & Hi-shelf. This "Mastering" version has +/- 6db gain per band.

These can be used to sculpt a very precise EQ curve. Each band can be switched on independently.

The first and last bands are high and low shelving fliters, the middle four bands are parametric filters with variable resonance. All cut or boost frequencies in an audio signal by a specified amount, expressed in decibels (db).

This module filters mono audio signals.


Controls:

Band on/off - switches for each filter.

Frequency - the centre or cut-off frequency of each filter.

Resonance - only parametric filters have resonance, which is how focussed the filter is around the centre frequency.

Gain - the amount of cut or boost (in db).

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.
