Mod Sequencer


Module type: Control Source


Introduction:

This triggered module is a 16-step sequencer that is focussed on modulation, rather than note triggering, with all controls including the individual steps modulatable.

It outputs control signals for note and amplitude (volume) values, 3 controls - called CV1/ CV2/ CV3, a clock output, an end of sequence trigger (when the next trigger is received after the last step has played), and a gate trigger (for all played steps).

Up to 16 Steps can be used with various options for the step order, including random.

An internal clock can be used to trigger to the next step, or other modules can trigger it using Actions or modulation inputs - for example Clock Pulse and Pulse Divider modules.

If the internal clock is used, the length of each step (in beats) is set independently.

Other help that might be useful: Modulation Vs Automation


Controls:

Play Clock - start or stop the clock

Restart Sequence - once this is triggered, the sequence will return to the first step at the next clock pulse.

Trigger Pulse - triggers the next step.


Parameters button

Step order - can be: 1-Forwards, 2-Backwards 3-Random or 4-Use Select Index.

Select Index - if Step order is set to 4, the current step is set to the value of Select Index at the next clock pulse.

Start step - which step to start playing from.

Seq length - the number of steps to use in the sequence. This will wrap around to the first step if high enough. For example, if the start step is 16 and sequence length is 4, the steps played will be 16, 1, 2, 3.

Loop - if not looped, the sequence will stop responding to triggers once all the selected steps have played once - unless restart is triggered. If looped it will continue responding to triggers.

Note: Loop changes will only take effect when the sequence is restarted.

Freq range - the range in frequencies for each step shown as midi notes. Musical instrument and octave range presets can be selected.

Freq Quantise - shifts all step frequencies to the nearest midi note.

Beats range - the range in beats for each stepâ€™s length

Beats Quantise - if not set to 0 (off), this quantises all steps beat values.


Steps button

Step 1-16 Freq: MIDI note - both frequency and MIDI note are shown

Step 1-16 Amp - amplitude for each step

Step 1-16 CV1 - for each step

Step 1-16 CV2 - for each step

Step 1-16 Beats /CV3 - for each step, with beats shown below each knob. The beats value is only relevent if the internal clock is being used. The value of the knob expressed in the range 0 to 1 is used as the output value for CV3.

MIDI learn notes & amps (velocities) - when this is switched on, any MIDI notes and velocities received on any MIDI channels will be recorded into the steps of the sequencer starting at the Next Step, which is shown next to it. The new notes will be displayed on the screen.


Clock button

BPM & Multiply BPM - set the clock speed

Jitter - adds random variation to the BPM
