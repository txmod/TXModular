Delay St


Module type:  Audio Insert (stereo input, stereo output)


Introduction:

This module delays the left and right channels of a stereo audio signal. The left and right channels can be set differently. If the feedback in not zero, the delays will be repeated and gradually die away. The more feedback the longer they take to die away.


Controls:

Maximum time - this allocates more computer memory if you need longer delay times. Up to 5 minutes delay time can be set. 

Tap tempo button - this displays measures the tempo when it is clicked with a mouse on each beat. Use the Command (Apple) key  with a mouse drag to copy the BPM to the Delay time BPM.

Delay time L & R - this is shown in milliseconds as well as beats per minute (bpm). The left and right channels can be set differently.

time x 2 , time x 3 , time / 2 , time / 3  - buttons to change the delay time. 

Smooth time - smoothes changes to the delay time so speed changes are more gradual and less jerky.  

Feedback  L & R  - sets how long delays will be repeated for.  

Feedback Type - can be positive or negative. If negative, feedback will emphasize odd harmonics at an octave lower

Dry-Wet Mix - this sets the mix of the original signal with the delayed signal.  

Freeze - this ignores any signal going into the delay, and maximises the feedback so the sound will keep repeating without dying out.

Clear - This button clears the delay, so any sounds in the delay buffer are silenced.
