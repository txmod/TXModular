RingMod


Module type:  Audio Insert (mono input, mono output)


Introduction:

This module ring modulates one mono audio signal (called the Modulator) onto another mono audio signal (called the Carrier). Ring modulation creates a complex signal which constists of the frequencies of the Modulator and Carrier being added together as well as subtracted from each other.

This module is an audio insert module which should be added to the channel of the audio signal that is the Carrier.

Then on another audio channel, the destination can be set to the RingMod module's Modulator bus (see Audio Input Busses below).


See the example file RingMod.


Controls:

Dry-Wet Mix - this sets the mix of the original signal with the affected signal.  For send effects this will usually be set to 1 - i.e. 100% affected signal.


Audio Input Busses:

Modulator - mono audio signal which is ring modulated with the Carrier audio signal. 

  

Control Modulation Input Busses:

Dry-Wet Mix
