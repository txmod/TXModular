Dual Filter St


Module type: Audio Insert (stereo input, stereo output)


Introduction:

This module filters stereo audio signals.

A filter cuts or boosts certain frequencies in an audio signal.

In this module, 1 or 2 filters (named Filter A & Filter B) are used, depending on the Filter Mode. There are 4 modes:

Filter Mix: Mix between A and B - crossfades between 2 filters

Filter Morph: Morph between A and B [Filter type B ignored] - only 1 filter is used, based on Filter type A, morphing between the settings of A & B

Filters in Series: A is filtered by B [Morph/Mix ignored] - the signal is filtered by A and then by B

Subtract Filters: Subtract B from A [Morph/Mix ignored] - this can be used creatively, for example subtracting 2 resonant low-pass filters creates a band-pass with 2 resonant peaks.


Controls:


Global button

Filter Mode - see above for details.

Morph/Mix - sets the morph or mix between Filter A and Filter B (only used in first 2 modes).

Freq Smooth - for smoothing out changes to the filter frequecy, this sets what type of smoothing to use - linear or 3 types of exponential smoothing which get progressively smoother.

Smooth Time - sets how long it takes (in seconds) for the frequency to reach a new value.

Level control - options:
None - level unchanged
Balance - match level to dry signal level
Limiter - keep the signal level below 0db
Normalize - force the signal level to just below 0db
Tanh Clipping - uses hyperbolic tangent of signal to clip it (this adds some distortion)

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Filter A button

Filter Mode - see above for details.

Morph/Mix - sets the morph or mix between Filter A and Filter B (only used in first 2 modes).

Filter type A - type of filter. The name will indicate whether the Resonance and Saturation controls are used - they are only used by some filters.

Frequency A - the centre or cut-off frequency of the filter.

Resonance A - only certain filter types have resonance controls (see filter names). Resonance is how focussed the filter is around the centre frequency.

Saturation A - only certain filter types have saturation controls (see filter names). Saturation boosts the filter signal and can introduce mild distortion.

Gain A - the output level of the filter.


Filter B button

Filter Mode - see above for details.

Morph/Mix - sets the morph or mix between Filter A and Filter B (only used in first 2 modes).

Filter type B - type of filter. The name will indicate whether the Resonance and Saturation controls are used - they are only used by some filters.

Frequency B - the centre or cut-off frequency of the filter.

Ratio B - if the button Use Ratio B x Freq A for Freq B is switched on, Ratio B is used to set the Frequency of Filter B by multiplying the ratio by the Frequency A, so it will change proportionately whenever Frequency A changes.

Resonance B - only certain filter types have resonance (see filter names). Resonance is how focussed the filter is around the centre frequency.

Saturation B - only certain filter types have saturation (see filter names). Saturation boosts the filter signal and can introduce mild distortion.

Gain B - the output level of the filter.

Use Ratio B x Freq A for Freq B - this forces Freq B to move with Freq A in a set ratio.

Use Resonance A for Resonance B - this keeps the Resonance of A & B the same.

Use Saturation A for Saturation B - this keeps the Saturation of A & B the same.

Use Gain A for Gain B - this keeps the Gain of A & B the same.


Filter A & B button

This button shows the controls for both filters for easy editing.

Note: Min & Max values for the controls aren't shown - use Filter A and Filter B buttons to edit Min & Max values.
