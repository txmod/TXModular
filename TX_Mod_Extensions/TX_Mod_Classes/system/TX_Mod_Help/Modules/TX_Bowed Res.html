Bowed Res


Module type: Audio Insert (mono input, mono output)


Introduction:

This module is a digital wave guide physical model of a soundboard resonator for a bowed instrument.


Controls:


Gain button

Controls for gain, levelling & dry-wet mix.

Model button

Controls to change the model. These are currently undocumented.

Band Parameters button

Controls to set ranges for the EQ bands.

Bands & Hi-cut button

Controls for 4 EQ bands & a Hi-cut filter.
