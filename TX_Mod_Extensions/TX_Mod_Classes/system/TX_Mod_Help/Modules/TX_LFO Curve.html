LFO Curve



Module type: Control Source


Introduction:

Low Frequency Oscillator with an editable waveform curve - using a mouse. This module can be used for modulating parameters of other modules.


Controls:

Restart - triggers the LFO to jump to start of the curve.

Frequency - the rate of change of the waveform shown as frequency and bpm.

LFO Curve - the curve used for the waveform. You can edit the curve with a mouse.

See help file Curve Editing for details.

Grid rows - no of rows (1-128) separated by white lines in the grid behind the curve.

Grid cols - no of columns (1-128) separated by white lines in the grid behind the curve.

Output range - preset output ranges including negative values



Control Modulation Input Busses:

Frequency
