Toggle Switch


Module type: Control Source


Introduction:

This module outputs either 0 (Off) or 1(On). It has a single trigger which causes a change in the output value, from 0 to 1, or 1 to 0 - unlike the On Off Switch module which has separate on and off triggers.

It can be switched in various ways such as by modulation of the Trigger input, by other modules (such as an Action Sequencer) or by action buttons on an interface screen.


Controls:

Trigger - triggers a change in the output value, from 0 to 1, or 1 to 0.


Control Modulation Input Busses:

Trigger
