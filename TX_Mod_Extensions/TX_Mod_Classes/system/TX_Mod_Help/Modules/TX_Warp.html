Warp

Module type: Control Insert

Introduction:

This module is for "mapping" or changing the behaviour of a control signal.

The incoming signal is mapped using an editable transfer curve.

The horizontal (left-right) axis of the curve represents the range of values of the input signal (based on the Input range control).

The vertical (up-down) axis of the curve represents the range of values of the output signal (based on the Output range control).

The default curve is a linear curve which will have no effect on the incoming signal.

There are 5 slots for storing curves - useful if you are trying out different curves.

Controls:

In/Out & Quantise button

Input range - limits the range of value coming in. Values outside the range will be forced to be within range.

Output range - sets the range of values that the output signal will be mapped to.

Quantise - activates the quantise.

Quant Type - sets whether the signal is quantised before or after warping.

Quant Steps - the number of steps in the quantise.

Curve button

Warp curve - sets a transfer curve which maps the input signal of the warp module to new output values.

See help file Curve Editing for details.

Grid rows - no of rows (1-128) separated by white lines in the grid behind the curve.

Grid cols - no of columns (1-128) separated by white lines in the grid behind the curve.
