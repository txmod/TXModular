EQ Graphic


Module type: Audio Insert (mono input, mono output)


Introduction:

31-band EQ that cuts or boosts frequencies in an audio signal.

This module EQs mono audio signals.


Controls:

EQ curve - sets the EQ curve for 31 bands. The bands range from 20hz to 20khz, with 3 bands per octave.

Display Freq - moving the slider displays each frequency band being cut or boosted.

Reset - resets the curve to a linear curve.

Load 1/2/3/4/5 - to load curves stored in 5 slots.

Store 1/2/3/4/5 - to store curves in 5 slots.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Control Modulation Input Busses:

Dry-Wet Mix


Technical information:

EQ curve - The band frequencies are (in hz):

20, 25, 31.5, 40, 50, 63, 80, 100, 125, 160, 200, 250, 315, 400, 500, 630, 800, 1000, 1250, 1600, 2000, 2500, 3150, 4000, 5000, 6300, 8000, 10000, 12500, 16000, 20000.
