EQ Shelf


Module type: Audio Insert (mono input, mono output)


Introduction:

A shelving filter which cuts or boosts frequencies in an audio signal above or below a cutoff frequency.

This module filters mono audio signals.


Controls:

Filter - type of filter, low or high shelf.

Frequency - the centre or cut-off frequency of the filter.

Smoothing - for smoothing out changes to the filter frequecy, this sets what type of smoothing to use - linear or 3 types of exponential smoothing which get progressively smoother.

Smooth Time - sets how long it takes (in seconds) for the frequency to reach a new value.

Cut-Boost - the amount of cut or boost (in db).

Slope - the slope of the EQ curve.

Dry-Wet Mix - this sets the mix of the original signal with the affected signal. For send effects this will usually be set to 1 - i.e. 100% affected signal.


Control Modulation Input Busses:

Frequency

Smooth Time

Cut-Boost
