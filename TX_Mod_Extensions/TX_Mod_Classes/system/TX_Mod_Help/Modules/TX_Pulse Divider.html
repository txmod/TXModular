Pulse Divider


Module type: Control Source


Introduction:

This module outputs clock pulses by dividing the number of triggers it receives by various amounts. For example, if the division is 3, only the first of every 3 pulses will be output.

Each output, labelled Div 2 to Div 8, divides the input by a different fixed number.

This module can be triggered in various ways such as by modulation of the Trigger input, by other modules (such as an Action Sequencer) or by action buttons on an interface screen.

See also Pulse DividerV which is similar but different.


Controls:

Trigger Pulse - this is for manually triggering a pulse into the dividers.

Reset Dividers - this resets all dividers so they will all trigger on the next trigger received.

Probability - the chance that an input pulse will be sent into the dividers. 1 means always, 0 means never.

Output range - preset output ranges including negative values
