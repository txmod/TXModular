Midi Velocity


Module type: Control Source


Introduction:

A midi note has a velocity between 0 and 127.

This module outputs a control signal based on midi velocity messages received.


Controls:

Midi channels - set the midi channels the module will listen to (outside this range are ignored).

Note range - which notes to listen to (outside this range are ignored).

Vel range - which velocities to listen to (outside this range are ignored).

Output range - representing the values in the velocity range. The preset output ranges include negative options.
