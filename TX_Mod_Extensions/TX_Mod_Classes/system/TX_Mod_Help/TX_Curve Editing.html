Curve Editing

Several modules use curves, including: Env Curve, LFO Curve, Loop Player, Warp, Warp Morph, Waveshaper, WaveshaperX.

Using a mouse, click and drag to draw either freehand points or pre-selected line shapes (see controls below).

Zoom sliders allow very detailed editing of the curve.

History buttons keep the last 20 curve edits for undo/redo.

A separate Curve Builder can also be opened for more options for manipulating curve shapes. See help file Curve Builder for details.

Note: Most TX modules display their curves horizontally, though an exception is Groove curves in Loop Player modules which are shown vertically (see image on right). However, all curves will be shown horizontally if they are opened in the Curve Builder. This is to allow adequate screen space for editing multiple curves.

Controls:

< > - to move backwards/forwards through the curve edit history

drag & drop - use this box to drag or drop a curve onto the clipboard or the Curve Builder.

Curve Builder - this button opens the current curve inside the Curve Builder, which is in a separate window.

Draw Points/Lines - this selects between 2 different draw modes:

- Draw Points is for drawing a curve freehand.

- Draw Lines uses preset curve shapes, or straight lines, to draw lines between mouse-down and mouse-up points. If the Shift button is down when the mouse button is pressed, a line is drawn from the last drawn point to the current mouse point - this allows a chain of points to be linked together.

Quantise [only in Draw Lines mode] - this gives the option of quantising the start and end points of the line using the grid behind the curve. Options are:
quant off - no quantising is used
quant x+y - quantise both to columns and rows
quant x - quantise to columns only
quant y - quantise to rows only

Shape - [only in Draw Lines mode] - selects the line shape to draw - the default is linear for straight lines.

Reverse - reverses the curve from right to left.

Invert - turns the curve upside down.

Smooth - evens out spikes or sudden changes in the curve. Useful when curve is drawn by the mouse.

Mirror - Rebuild curve by mirroring - this button can be useful for creating symmetrical curves. The curve is mirrored at the right hand edge.

Mirror+Inv - Rebuild curve by mirroring & inverting - for this button, the curve is mirrored at the right hand edge, inverted and finally the whole curve is normalised.

Smooth - smooths the curve by mixing values of points on the curve with their immediate neighbours on either side.

Smooth 5 - runs a smooth 5 times

Reset - resets the curve to a linear curve.

Zoom out - zooms out to show the whole curve.

Load 1/2/3/4/5 - to load curves stored in 5 slots.

Store 1/2/3/4/5 - to store curves in 5 slots.

Preset curves - select a presets to load it.

Modify curve - select from the following options:

- normalise - scales and adjusts values in the curve so the minimum value is 0 and the maximum is 1.

- double copy - replaces the curve with 2 copies (at half resolution)

- triple copy - replaces the curve with 3 copies (at one third resolution)

- phase shift 25-75% - shift/rotate the curve along the X-axis.
