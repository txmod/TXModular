Module Presets

All modules can have their settings stored as presets which can be reloaded at a later time. When you store a preset, the text in the module's Comments box will be used as the Preset Name.

Module settings can be copied or pasted to a clipboard, or saved to disk as preset files. This makes it easy to copy module settings between different modules in the same system, or on different systems.

Presets are saved or loaded using the Presets popup menu button which is on the top row of each module's box.


Note: Meaning of "excluding presets".

When a module's settings are copied to the clipboard or saved to a file, the settings include any presets that you have saved.

Then when you paste settings from the clipboard, or load settings from a file, on the Presets popup menu you are given an extra option to save or load settings "excluding presets".

The reason this option is given is that you may want to keep the presets you currently have, but only load the settings from the other module.

Otherwise, the presets from the other module will also be copied into the current one so you will lose the any current presets.
