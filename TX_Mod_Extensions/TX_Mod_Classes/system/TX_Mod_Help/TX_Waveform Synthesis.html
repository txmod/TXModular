Waveform Synthesis

Various TX modules, including Waveform (mono and stereo), Wave Synth, Wave Synth + and Harmoniser, can use waveforms which offer many kinds of software synthesis, from the more usual sine or square waves, to various types of noise, to more exotic vowel shapes as well as FM synthesis (see full list below).

These different types of synthesis are selected using the Waveform control.

Some waveforms have one or two parameters that can modify their sound in various ways. These are indicated by the text M1 and M2 following the name of the waveform.

Underneath the waveform name are two sliders labelled "Modify 1" and "Modify 2" which can be moved to affect the sound of the waveform. These only have an effect if M1 or M2 is shown in the name of the waveform

For example, there is a waveform called "Sine -> Sawtooth wave - M1". In this case, the Modify 1 slider will change the waveform from a sine wave to a sawtooth wave.


Note - the frequency of the waveform is set in different ways in these three modules:

In the Waveform module, the frequency is set on the screen and can also be modulated.

In the Wave Synth / Wave Synth + modules, the frequency is set by the sequencer or midi keyboard that is triggering the synth.

In the Harmoniser module, the frequency is set by detecting the frequency of the audio signal coming into the module (this module is an audio insert module, so audio will be coming into it from the channel it is in).


List of Waveforms:

Sine wave - Modify 1 is not used. Modify 2 is not used.

Chorusing sine wave - M1 - Mix of 2 detuned sine waves. Modify 1 sets the depth of detune, range: 0 - 2% Modify 2 is not used.

Sine -> Sawtooth wave - M1 - Modify 1 crossfades between the waveforms. Modify 2 is not used.

Sine -> Square wave - M1 - Modify 1 crossfades between the waveforms. Modify 2 is not used.

Sawtooth wave - Modify 1 is not used. Modify 2 is not used.

SyncSaw wave - M1 - Sawtooth with a hard-synced second waveform. Modify 1 is the frequency ratio of the second waveform, range: 1 - 2.5 Modify 2 is not used.

Sawtooth -> Square wave - M1 - Modify 1 crossfades between the waveforms. Modify 2 is not used.

Sawtooth wave with flattened harmonics - M1 - Modify 1 sets the amount of flattening. Modify 2 is not used.

Sawtooth wave with sharpened harmonics - M1 - Modify 1 sets the amount of sharpening. Modify 2 is not used.

Square wave - Modify 1 is not used. Modify 2 is not used.

Pulse wave - M1 - Modify 1 sets the width of the pulse, range: 50% - 97% Modify 2 is not used.

Square wave with flattened harmonics - M1 - Modify 1 sets the amount of flattening. Modify 2 is not used.

Square wave with sharpened harmonics - M1 - Modify 1 sets the amount of sharpening. Modify 2 is not used.

Formant osc 1 - M1 - Generates a set of harmonics around a formant frequency at a given fundamental frequency, with pulse width frequency affecting the bandwidth of the formant. Pulse width frequency is same as fundamental frequency. Modify 1 sets the formant frequency ratio, range: 1 - 10. Modify 2 is not used.

Formant osc 2 - M1 - Generates a set of harmonics around a formant frequency at a given fundamental frequency, with pulse width frequency affecting the bandwidth of the formant. Pulse width frequency is twice the fundamental frequency. Modify 1 sets the formant frequency ratio, range: 1 - 10. Modify 2 is not used.

Formant osc 3 - M1 - Generates a set of harmonics around a formant frequency at a given fundamental frequency, with pulse width frequency affecting the bandwidth of the formant. Pulse width frequency is 4 times the fundamental frequency. Modify 1 sets the formant frequency ratio, range: 1 - 10. Modify 2 is not used.

Formant osc 4 - M1 - Generates a set of harmonics around a formant frequency at a given fundamental frequency, with pulse width frequency affecting the bandwidth of the formant. Pulse width frequency is 8 times the fundamental frequency. Modify 1 sets the formant frequency ratio, range: 1 - 10. Modify 2 is not used.

Formant osc 5 - M1 - Generates a set of harmonics around a formant frequency at a given fundamental frequency, with pulse width frequency affecting the bandwidth of the formant. Formant frequency is twice the fundamental frequency. Modify 1 sets the pulse width frequency, range: 1 - 9 Modify 2 is not used.

Formant osc 6 - M1 - Generates a set of harmonics around a formant frequency at a given fundamental frequency, with pulse width frequency affecting the bandwidth of the formant. Formant frequency is 4 times the fundamental frequency. Modify 1 sets the pulse width frequency, range: 1 - 9 Modify 2 is not used.

Formant osc 7 - M1 - Generates a set of harmonics around a formant frequency at a given fundamental frequency, with pulse width frequency affecting the bandwidth of the formant. Formant frequency is 8 times the fundamental frequency. Modify 1 sets the pulse width frequency, range: 1 - 17 Modify 2 is not used.

Formant osc 8 - M1 - Generates a set of harmonics around a formant frequency at a given fundamental frequency, with pulse width frequency affecting the bandwidth of the formant. Formant frequency is 16 times the fundamental frequency. Modify 1 sets the pulse width frequency, range: 1 - 17 Modify 2 is not used.

Formant osc 9 - M1 & M2 - Generates a set of harmonics around a formant frequency at a given fundamental frequency, with pulse width frequency affecting the bandwidth of the formant. Modify 1 sets the formant frequency, range: 1 - 10 Modify 2 sets the pulse width frequency, range: 1 - 16

Formant osc 10 - M1 & M2 - Generates a set of harmonics around a formant frequency at a given fundamental frequency, with pulse width frequency affecting the bandwidth of the formant. Modify 1 sets the formant frequency, with the direction reversed, range: 10 - 1 Modify 2 sets the pulse width frequency, range: 1 - 16

White Noise - Noise where frequency spectrum has equal power at all frequencies. Modify 1 is not used. Modify 2 is not used.

Pink Noise - Noise where frequency spectrum falls off in power by 3 dB per octave. This gives equal power over the span of each octave. It has more lower frequencies than white noise. Modify 1 is not used. Modify 2 is not used.

Brown Noise - Noise where frequency spectrum falls off in power by 6 dB per octave. It has more lower frequencies than white and pink noise. Modify 1 is not used. Modify 2 is not used.

Clip Noise - Noise with maximum energy. Modify 1 is not used. Modify 2 is not used.

Crackle - M1 - Noise generator based on chaos algorithm. Modify 1 sets the chaos parameter. Higher values crackle. Modify 2 is not used.

Dust - M1 - Noise based on random impulses. Modify 1 sets the density (average frequency) of the dust, range: 1 - 1001 Modify 2 is not used.

Gray Noise - Varying noise where frequency spectrum is emphasized towards lower frequencies. Modify 1 is not used. Modify 2 is not used.

Vowel - Man 1 - M1 & M2 - Vowel sounds from formants based on a man's voice and the vowel sounds in BEAT, BIT, PART and BOOK. Modify 1 & Modify 2 together control the mix between the vowel sounds.

Vowel - Man 2 - M1 & M2 - Vowel sounds from formants based on a man's voice and the vowel sounds in BUT, BOOT, BAT and BET. Modify 1 & Modify 2 together control the mix between the vowel sounds.

Vowel - Woman 1 - M1 & M2 - Vowel sounds from formants based on a woman's voice and the vowel sounds in POT, BIT, BET and PART. Modify 1 & Modify 2 together control the mix between the vowel sounds.

Vowel - Woman 2 - M1 & M2 - Vowel sounds from formants based on a woman's voice and the vowel sounds in BOOK, PERT, BAT and BOOT. Modify 1 & Modify 2 together control the mix between the vowel sounds.

Vowel - Child 1 - M1 & M2 - Vowel sounds from formants based on a child's voice and the vowel sounds in PART, BET, POT and BIT. Modify 1 & Modify 2 together control the mix between the vowel sounds.

Vowel - Child 2 - M1 & M2 - Vowel sounds from formants based on a child's voice and the vowel sounds in BEAT, BAT, BUT and PERT. Modify 1 & Modify 2 together control the mix between the vowel sounds.

Vowel - SizeMorph 1 - M1 & M2 - Vowel sounds from formants. Modify 1 sets the mix between the vowel sound in BUT and BOOK. Modify 2 shifts the formant frequencies.

Vowel - SizeMorph 2 - M1 & M2 - Vowel sounds from formants. Modify 1 sets the mix between the vowel sound in BUT and BOOK. Modify 2 shifts the formant frequencies.

Vowel - SizeMorph 3 - M1 & M2 - Vowel sounds from formants. Modify 1 sets the mix between the vowel sound in BOOT and PART. Modify 2 shifts the formant frequencies.

Vowel - SizeMorph 4 - M1 & M2 - Vowel sounds from formants. Modify 1 sets the mix between the vowel sound in BEAT and PERT. Modify 2 shifts the formant frequencies.

FM 0 - 2C.2M, freq ratios 2 & 1.5 - M1 & M2 - Frequency modulation using sine waves. Has a mix of 2 carriers each with a modulator. Modify 1 sets the depth of modulation using 2:1 frequency ratio. Modify 2 sets the depth of modulation using 1.5:1 frequency ratio.

FM 1 - 2C.2M, freq ratios 1.5 & 3.5 - M1 & M2 - Frequency modulation using sine waves. Has a mix of 2 carriers each with a modulator. Modify 1 sets the depth of modulation using 2:1 frequency ratio. Modify 2 sets the depth of modulation using 1.5:1 frequency ratio.

FM 2 - 2C.2M, freq ratios 1.333 & 1.666 - M1 & M2 - Frequency modulation using sine waves. Has a mix of 2 carriers each with a modulator. Modify 1 sets the depth of modulation using 1.333:1 frequency ratio. Modify 2 sets the depth of modulation using 1.666:1 frequency ratio.

FM 3 - 2C.2M, freq ratios 1.25 & 1.75 - M1 & M2 - Frequency modulation using sine waves. Has a mix of 2 carriers each with a modulator. Modify 1 sets the depth of modulation using 1.25:1 frequency ratio. Modify 2 sets the depth of modulation using 1.75:1 frequency ratio.

FM 4 - 1C.2M, freq ratios 2 & 1.5 - M1 & M2 - Frequency modulation using sine waves. Has 1 carrier being modulated by a modulator with a 2:1 frequency ratio, which is being modulated by another modulator with a 1.5:1 frequency ratio. Modify 1 sets the depth of modulation using 2:1 frequency ratio. Modify 2 sets the depth of modulation using 1.5:1 frequency ratio.

FM 5 - 1C.2M, freq ratios 1.5 & 3.5 - M1 & M2 - Frequency modulation using sine waves. Has 1 carrier being modulated by a modulator with a 1.5:1 frequency ratio, which is being modulated by another modulator with a 3.5:1 frequency ratio. Modify 1 sets the depth of modulation using 1.5:1 ratio. Modify 2 sets the depth of modulation using 3.5:1 ratio.

FM 6 - 1C.2M, freq ratios 1.25 & 2.25 - M1 & M2 - Frequency modulation using sine waves. Has 1 carrier being modulated by a modulator with a 1.25:1 frequency ratio, which is being modulated by another modulator with a 2.25:1 frequency ratio. Modify 1 sets the depth of modulation using 1.25:1 ratio. Modify 2 sets the depth of modulation using 2.25:1 ratio.

FM 7 - 1C.1M, variable freq - M1 & M2 - Frequency modulation using sine waves. Has 1 carrier being modulated by a modulator. Modify 1 sets the frequency ratio of the modulator, range: 1 - 2.5 Modify 2 sets the depth of modulation.

FM 8 - 1C.2M, variable freq - M1 & M2 - Frequency modulation using sine waves. Has 1 carrier being modulated by a modulator with a variable frequency ratio, which is being modulated by another modulator with a variable frequency ratio. This has a shallower modulation than FM 9. Modify 1 sets the frequency ratio of the first modulator, range: 0.5 - 2.5 Modify 2 sets the frequency ratio of the second modulator, range: 0.25 - 3.25

FM 9 - 1C.2M, variable freq - M1 & M2 - Frequency modulation using sine waves. Has 1 carrier being modulated by a modulator with a variable frequency ratio, which is being modulated by another modulator with a variable frequency ratio. This has a deeper modulation than FM 8. Modify 1 sets the frequency ratio of the first modulator, range: 0.5 - 2.5 Modify 2 sets the frequency ratio of the second modulator, range: 0.25 - 3.25

Golden Mean Harmonics - M1 - Generates a set of harmonics based on the frequency ratio of the Golden Mean which is approx. 1.61803 Modify 1 sets the proportion of harmonics added. Modify 2 is not used.

Drum Harmonics - M1 - Generates a set of harmonics based on the vibration of a drum, Modify 1 sets the proportion of harmonics added. Modify 2 is not used.

Cymbal Harmonics - M1 - Generates a set of harmonics based on the vibration of a cymbal, Modify 1 sets the proportion of harmonics added. Modify 2 is not used.

Root 2 Harmonics - M1 - Generates a set of harmonics based on the frequency ratio of the square root of 2 which is approx. 1.41421 Modify 1 sets the proportion of harmonics added. Modify 2 is not used.

Root 3 Harmonics - M1 - Generates a set of harmonics based on the frequency ratio of the square root of 3 which is approx. 1.73205 Modify 1 sets the proportion of harmonics added. Modify 2 is not used.

Blip - M1 - Blip - Band limited impulse oscillator. Modify 1 sets the number of harmonics, range: 1 - 60 Modify 2 is not used.

Chaotic noise, Logistic map, stepped - M1 - Standard map chaotic generator with no smoothing. Modify 1 changes the chaotic behaviour. Modify 2 is not used.

Chaotic noise, Logistic map, linear smoothing - M1 - Standard map chaotic generator with linear smoothing. Modify 1 changes the chaotic behaviour. Modify 2 is not used.

Chaotic noise, Logistic map, cubic smoothing - M1 - Standard map chaotic generator with cubic smoothing. Modify 1 changes the chaotic behaviour. Modify 2 is not used.

Chaotic noise, Standard map, stepped - M1 - Standard map chaotic generator with no smoothing. Modify 1 changes the chaotic behaviour. Modify 2 is not used.

Chaotic noise, Standard map, linear smoothing - M1 - Standard map chaotic generator with linear smoothing. Modify 1 changes the chaotic behaviour. Modify 2 is not used.

Chaotic noise, Quadratic map, stepped - M1 - Quadratic map chaotic generator with no smoothing. Modify 1 changes the chaotic behaviour. Modify 2 is not used.

Chaotic noise, Quadratic map, linear smoothing - M1 - Quadratic map chaotic generator with linear smoothing. Modify 1 changes the chaotic behaviour. Modify 2 is not used.

Chaotic noise, Quadratic map, cubic smoothing - M1 - Quadratic map chaotic generator with cubic smoothing. Modify 1 changes the chaotic behaviour. Modify 2 is not used.

Triangle wave with negative shape distortion - M1 - Triangle wave with negative shape distortion. It sounds more buzzy than positive shape distortion. Modify 1 distorts the waveform shape. Modify 2 is not used.

Triangle wave with positive shape distortion - M1 - Triangle wave with positive shape distortion. It sounds more bassy than negative shape distortion. Modify 1 distorts the waveform shape. Modify 2 is not used.

Phase distortion sync sine wave - M1 & M2 - Phase distortion synthesis using a sine wave. Modify 1 sets the frequency ratio of the sync wave, range: 1 - 32. Modify 2 distorts the waveform shape.

Phase distortion sync sawtooth wave - M1 & M2 - Phase distortion synthesis using a sawtooth wave. Modify 1 sets the frequency ratio of the sync wave, range: 1 - 32. Modify 2 distorts the waveform shape.

Phase distortion sync triangle wave - M1 & M2 - Phase distortion synthesis using a triangle wave. Modify 1 sets the frequency ratio of the sync wave, range: 1 - 32. Modify 2 distorts the waveform shape.
