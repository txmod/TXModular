The TX Modular System - Audio Setup


The TX Modular System uses the default audio device on your computer for both input and output of audio.


If you are using an external audio interface, make sure that the interface has been set to be the default audio device for system input and output on your computer.


To check or change this,

1. run "Audio MIDI Setup" which can be found in the folder:

Applications/Utilities/

2. select the "Audio Devices" screen

3. make sure your soundcard is selected in "Default Input" and "Default Output"

4. Close Audio MIDI Setup.
