Using Samples and Loops

In order to use samples or loops, you need to add them into a Sample Bank or Loop Bank.

To open the relevent window use Sample Banks and Loop Banks buttons near the top of the window.


Sample Banks

There are 99 banks available. You can give a name to the banks you use.

Each bank holds stereo and mono samples for use by modules such as Sample Player, Sample Player+, Granulator, and Convolution.

Use the Drag & drop to add box or Add Samples button to add one or more aiff or wave files to the current Sample Bank.

With other buttons, you can search a chosen folder for missing samples, save the current sample bank to disk, for use by other systems, load banks from disk into your current banks, play samples, remove individual samples from the bank, replace samples, or empty the entire bank.

You can select samples within the bank and see their length (in milliseconds) and the number of channels.

You can also tell the system the midi note or frequency (i.e. pitch in cycles per second) of specific samples in the bank if you know them. This enables the samples to automatically play in tune when you use them in Sample Players.


Loop Banks

There are 99 banks available. You can give a name to the banks you use.

Each bank holds stereo and mono samples for use by Loop Player modules.

It is very similar to the Sample Bank, except that for loops you set the total number of beats in the loop. Once this is set, the BPM of the loop (based on the loop time and total no. of beats) will be shown so you can check that it is correct.

If you change the BPM the Total beats will be calculated automatically.

Note: Take care to set the correct number of beats for each loop used so that Loop Players will play back the loops at the correct BPM.


Using mono and stereo samples or loops

When you use a mono Sample Player or Loop Player, it will only allow you to select mono samples or loops from the bank.

However, the stereo Sample and Loop Players will allow you to select mono and stereo samples or loops.

You can tell whether a sample or loop is mono or stereo by selecting it in the sample or loop bank and checking the number of channels displayed.
