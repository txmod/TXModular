Introduction to TXV app

TXV, which is a separate app, opens in itâ€™s own window and is a modular video system for generating real-time 2D & 3D visuals. It was created to enable a TX Modular user to control both video and audio processes, and link them together.

In the TX Modular, TXV is controlled by adding a TXV System module (see the example below).


TXV Modules

TXV includes various types of modules:

Draw Sources such as image file, movie file, live video camera, 3D Model file, 2D/3D shapes.

Image FX such as Blend, Blur, Pixelate and Colorize.

Draw Destinations include the TXV System, Transform (shift/ scale/ rotate) and Transf3DMulti (transform & make multiple copies of drawn objects), Scene (3D Scene with cameras & lights).

Modulation sources include LFOs, Perlin noise, and a Step Sequencer.


Module Connections

There are two TXV modules which connect other modules together:

TXV Draw Connections are modules that connect Draw Source modules to Draw Destination modules, including the TX System module which draws to the main window.

TXV Nodes are modules that connect modulation sources (such as LFOs) to parameters in other modules to enable modulation to happen.


Image Connections

The various modules that use images have the option either to use image files stored on disk or to directly import an image produced by another image module. This second option allows image FX modules to be chained together for more complex FX.


Example - Building a TXV System

The following are instructions for building a simple TXV System which will display a 2D Shape being moved around using an LFO.

1. First you need to add a new TXV System module. Press the Modules & Channels button near the top left of the window.
In the left panel, press the +Source button and scroll to the bottom of the list to a category labelled Control: TXV System. If you open this category there's an add TXV System button. Press this to add a new TXV System module.

2. Now you need to start the TXV software. If you are using the standalone TX Modular (for MacOS) which includes TXV inside it, you can use the green Start TXV button. If you are using another platform, you will need to have a TXV build for your platform and open it separately. TXV will open in a separate window.

3. Once TXV is running, press the orange Connect to TXV button. To the right of the connect button is a yellow "Status" box which should soon say "TXV System connected".

4. Once TXV is connected, you can start adding modules. The row of blue buttons near the top of the TXV System box are menu buttons for different groups of parameters. Press the TXV Modules button and below that next to "Display Group" press the All Modules button so you can see all available modules.

5. On the left is a list of all possible modules that can be added. Select the module TXV LFO and then press the Add Selected Module button. Then add a TXV Shape2D module in the same way.

6. In order for the shape to be seen on the screen, we need to add a Draw Connection. In the row of blue buttons near the top of the TXV System box, press the Draw Connections button.

7. In the "source" box, click on TXV Shape 2D [1]. In the "destination" box, click on TXV System [1]. Then press the Add Draw Connection button on the right and you should see a small green circle in the middle of the black TXV window.

8. To edit the shape, press the Edit Module button in the left hand panel of the TX Modular window. Below that, press the TXV Shape 2D button so you can see all shape parameters.

9. There are blue buttons near the top of the module window. Make sure Shape is selected, and then try changing the shapeType. Each shape type has certain parameters listed after it's name - for example, "circle (width, circleResolution)". This shows which parameters are relevant for that particular shape. Try changing some of the values of those parameters for different shapes.

10. The other blue buttons enable you to set the Fill color, Outline color and Draw parameters such as position on the screen, scaling and rotation of the shape. Try changing the Fill and Outline colors.

11. In order to allow the shape to be moved around, you need to set up the modulation: Press the Draw button and below that, press the Position, Anchor, Rotate button.

12. You will see a slider labelled positionX and on below it to the right is slider labelled modulation mix. Set modulation mix to a value of 0.5 and you will see the shape has moved left.

13. To set up the modulation, you need to open the TXV System [1] module using the button in the left hand panel. An alternative way to do that is to use the Show parent TXV System button which is in the top row of the Shape 2D panel.

14. Once the TXV System [1] module is open, press the Modulation Nodes menu button.

15. In the "source module" box, click on TXV LFO [1].
In the "source name" box, click on out.
In the "destination module" box, click on TXV Shape 2D [1].
In the "destination name" box, scroll down to positionX.
Press the Add Node button. Now you should see the shape moving quickly from left to right and back again.

16. To edit the LFO speed, in the left hand panel press the TXV LFO [1] button.

17. In the TXV LFO [1] panel, press the BPM, Cycle, Phase button. Then change the cycleTimeBeats to a value around 10 so the shape moves at a slower speed.
