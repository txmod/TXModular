Curve Builder


Several modules use curves, including: Env Curve, LFO Curve, Warp, Warp Morph, Waveshaper, WaveshaperX.

These modules include a basic curve editor - see help file Curve Editing for details.

The Curve Builder adds more ways to manipulate curves, and has clipboards to store temporary copies of the curves while you’re working with them.

The Curve Builder can be opened from the Tool Windows popup on the main window, or from inside a module which uses curves - there's a Curve Builder button on the curve editor . The curve from the module will automatically be copied into Curve 1 of the Curve Builder window, and the Curve Size (see box at the top of the window) and Curves 1/2/3/4 in the builder will be set to the correct size of the curve being edited.

After editing in the Curve Builder, curves can be dragged - using the yellow “drag & drop” boxes or the white/blue drag & drop clipboards at the top and bottom of the window.

There are various options to help with building or modifying curves which are shown on the screen as checkboxes or buttons (see below).

Note: All curves are shown horizontally, from left to right, in the Curve Builder window. Some TX modules use vertical curves (such as the Groove Curves in Loop Player modules), but they will be shown horizontally if they are dragged into the Curve Builder. This is to allow adequate screen space for editing multiple curves.


Controls:

Curve Size - the number of points in the curves.

Grid rows - no of rows (1-128) separated by white lines in the grid behind the curves.

Grid cols - no of columns (1-128) separated by white lines in the grid behind the curves.

Curve Height - adjusts the height of the curves on the screen.

Show Envelope - this checkbox opens an envelope editor with draggable curves. You can add or delete new nodes in the envelope by pressing Ctrl while clicking on the curve. Once you have created the shape you want, use the buttons below the envelope to copy it to Curves 1/2/3/4.

Curve 1 - the initial curve before modifying.

Curve 1 Parameter FX - this button displays options and parameters for modifying Curve 1 using either a single parameter labelled Parameter 1(a slider),or a pair of parameters labelled Parameter 2 (a range slider).

Options include randomising, clamping, scaling, phase shifting, multiplying, dividing, adding, & subtracting values.

The results are put into Curve 4

Curve 1 & 2 FX - this button displays options for modifying Curve 1 using Curve 2.
Options include mixing, multiplying, dividing, adding, & subtracting the 2 curves, or using Curve 2 as an index into Curve 1 or Curve 1 as an index into Curve 2.
The results are put into Curve 4.

Use Curve 3 for FX Mix - this checkbox will make Curve 3 visible instead of the FX Mix slider and uses the curve as a mix control, giving much more control to the user of the final mix along each part of the curve.

FX Active Range - selects the area of the curve to be changed.

Curve 2 - this is only visible if Curve 1 & 2 FX is selected.

FX Mix Slider - this is only visible if Use Curve 3 for FX Mix is NOT selected. It controls the overall mix between the original and modified versions of the curve.

Curve 3 - FX Mix - this is only visible if Use Curve 3 for FX Mix is selected. Curve 3 is used to control the mix between the original and modified versions of the curve at each point along the curve.

Curve 4 - FX Result - the final curve with FX mixed in.
