MIDI/ Note Button


Most polyphonic audio modules have a MIDI/ Note Button, including these:

Filter Synth, FM Synth , Granulator, Pluck, Sample Players, Table Synth, Wave Synth, Wave Synth +.

The following controls are shown:


Controls:

Midi Listen - this allows the module to be triggered by midi Notes, and to respond to midi Pitchbend and midi Sustain Pedal (controller no. 64) messages.

Midi Channels - set the midi channels the module will listen to.

Note range - which notes will trigger the module.

Vel range - which velocities will trigger the module.

Keyboard tracking - sets whether pitch will be fixed or follow the keyboard note that is sent when the module is triggered.

Transpose - transposes the note played up or down.

Pitch bend - sets the value and range of the pitch bend.

Polyphony - maximum number of simultaneous notes that can be played at one time. If more are played, note stealing will happen - i.e. existing notes will be cut short.

Intonation - option for different sets of note ratios. The default is equal temperament, which is how most modern western instruments are tuned, with semitones equally spaced in each octave. Historically, many other tunings have been used and are still used around the world.The other options give a few different sets of just intonation note ratios which are much more natural sounding than equal temperament. Simple intervals and chords often sound very beatiful with these tunings. Sometimes, just intonation tunings can sound a bit strange when you play in certain keys harmonically far away from the root note. This is because the semitones are not equally spaced.

Note ratios - display of the note ratios for all 12 semitones

Key/ root - the root note from which all other notes are calculated using the note ratios. This only makes a difference to just intonation tunings, not equal temperament.

Keyboard - this can be used to trigger the module, by clicking with a mouse.The keyboard range is from midi note C1 to B6 (midi note numbers 36 to 107). When using this keyboard, the Velocity will be taken from the velocity slider below it.
