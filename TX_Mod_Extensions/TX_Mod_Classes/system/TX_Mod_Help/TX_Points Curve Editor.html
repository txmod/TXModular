Points Curve Editor


Introduction

Points curves are curves that are created by connecting a series of points together. Different curve types are available (see below).

Modules that use points curves include: TXV Curves and TXV Delay Curves.

The Points Curve Editor allows you to edit point curves in various ways: by moving individual points, by drawing each point in turn, or by shifting, scaling or rotating the whole curve.

The editor can be opened from the Tool Windows popup on the main window, or from inside a module which uses points curves. The curve from the module can be automatically copied into the Points Curve Editor window.

After editing in the Points Curve Editor, curves can be dragged back to the TX Module using the yellow “drag & drop curve” box.

Keyboard shortcuts - editing mode options can be set using single key shortcuts as follows:
e - edit points
d - draw points in seq
m - move all points
x - x-zoom all points
y - y-zoom all points
z - zoom all points
r - rotate all points
g - show / hide grid
p - show / hide points
n - show / hide point names


Controls:

Preset controls - to name, save, load and delete presets. There are no presets to begin with, but if you save any, they are stored on disk and will be available the next time you open TX Modular. (Presets can be useful for keeping temporary versions of curves - so you can go back to a previous version if you don't want to keep any changes you've made.)


Curve Type - Different types of curves are available.
- To see examples of all the curve types, select the image "Points Editor 2.png" [in the box on the right].
- All types of curve use from 2 to 8 draw points. The curve is created by connecting all the draw points, which are shown as blue boxes.
- All curves - apart from Straight Line - use extra points called control points which are shown as green boxes. The curve isn't drawn through the control points, but they are used to change the curvature of the curve as it goes through each draw point. (An easy way to understand what control points do is to edit a Bezier curve and drag around the green control points.).
- The different curve types are:
Auto Curve - This connects the draw points together using with a smooth curve. There are 2 optional control points that can be used to shape the start and end of the curve.
Straight Line - This simply connects all the draw points together with straight lines. No control points are used.
Bezier - Control P1 - This draws a Bezier curve through all the points using a single control point between each pair of draw points. The control point affects the curvature of the pair of draw points in the same way.
Bezier - Control P1 smoothed - This draws a Bezier curve through all the points using a single control point between each pair of draw points. The curve is smoothed at each draw point - by adding an extra hidden symmetrical control point - so that there are no kinks in the curve.
Bezier - Control P1 + P2 - This draws a Bezier curve through all the points using two control points between each pair of draw points.

No. Points - the number of draw points in the curve.

Auto Close - If this is switched on, a straight line is drawn between the first and last draw point.

Use first draw point for AC Start - This control will only appear if the the curve type is Auto Curve.
If it is switched off, a control point called "acstart" will be shown, which changes the curvature around the first draw point.

Use last draw point for AC End - This control will only appear if the the curve type is Auto Curve.
If it is switched off, a control point called "acend" will be shown, which changes the curvature around the last draw point.


Grid rows, columns & presets - settings for the grid shown behind the curve if the "show grid" switch is on.

drop background PNG image file - to add an image to be shown behind the curve. This can be used, for example, for copying the outline of a shape in the image with a points curve.

Remove background image - button to remove the image.

Print list of points - this prints a list of points and their X & Y coordinates to the SuperCollider post window.


Curve controls - the lower section of the window is for editing the curve. Most of the controls are self-explanatory - a few need clarifying:

Curve Edit Mode - options are: e - edit points, d - draw points in seq, m - move all points, x - x-zoom all points, y - y-zoom all points, z - zoom all points, r - rotate all points.

undo / redo - A history of recent edits is kept.

lock end to start - This switch forces the last draw point to be at the same position as the first draw point. Then if you drag one point, both will move together.

quantise controls adjust the x/y position of all draw and control points:
auto quantise X, Y & quantise X, Y move them to the nearest grid crossing point.
quantise X moves them to the nearest grid column.
quantise Y moves them to the nearest grid row.

realign control points - this button will move all control points so they lie on a straight line between each pair of draw points. The effect of this is to straighten the curve between all draw points.
