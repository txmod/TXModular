1.0 Basic concepts in the TX Modular System


1.1 Signals

There are 2 type of signals flowing around a TX system: audio signals can be heard through loudspeakers, whereas control signals can't be heard but are used to control things within the system - such as the loudness of a synth.


1.2 Modules

A module is a basic unit in the system which does something, such as generate a waveform, play a sample or filter a sound. A source module (such as a waveform or sample player) generates a signal from scratch, whereas an insert module (such as filter or delay) takes a signal and processes it in some way before outputting a new signal.

Different modules output different kinds of signals, so there are audio and control source modules, as well as audio and control insert modules.

There are also action modules, such as Action Sequencer, which don't use signals at all. They usually just give commands or instructions to other modules.


1.3 Channels

Think of a channel like the channels on a mixing desk. It is where a signal coming from a source gets connected to a destination. For example, you may want to take a sample player output put it through a flanger and connect it to your soundcard output. You would use a channel to make the connections.

When a new source module is created, a channel is created automatically.

Inside the channel, you can add up to 5 insert modules to process the signal, as well as setting a destination, ie. where you want the signal to be sent.

There are audio and control channels.


1.4 Busses

Within the system, there are also audio and control busses carrying various signals.

Busses are used by channels to connect modules to each other or to the audio outputs.

Insert modules have one or more input busses going into them.

Source and insert modules have one or more output busses coming out of them.

There are 16 main out busses which send the audio signals to your audio outputs on your computer or audio interface.

There are 4 FX send busses which, just like in mixing desk, are used for sending signals from audio channels to a shared bus to then be processed in some way - e.g. for adding reverb.

Finally, there are 10 audio aux busses and 10 control aux busses. These are for mixing and routing audio and control signals within the system.

2.0 Using the TX Modular System


2.1 The TX Modular Window

The top 2 rows of buttons are visible on all windows in the system.

The top row of buttons are system actions:

The Help button opens the main help file with links to all other help files.

Next are buttons to open a file of previously saved system, and save the current system to a file. If you save or open a file, its name will appear on the line below these buttons.

Next are buttons to rebuild system, clear system or quit. You occasionally need to rebuild a system if one of the modules crashes, but it shouldn't be needed often. Clearing a system means removing all the modules and channels. Use quit to close down the TX Modular system. Another way to quit is to close the TX Modular System window.

The Sync Start button is for starting any modules in the system that have their sync start option selected. Certain modules - Action Sequencers, Step Sequencers, File Players and File Recorders have a Sync Start option so that multi layered sequences can be started together.

The Sync Stop button is for stopping any modules in the system that have their sync stop option selected.

The Stop All button is for stopping all modules in the system regardless of any sync options.

Sync Start, Sync Stop & Stop All - these 3 buttons allow Action Sequencers, Step Sequencers, Fle Players and File Recorders to be started and stopped together. Each one can be set to respond to Sync Start and Sync Stop instructions. Stop All will cause all modules to stop regardless of their sync start and stop settings.

The Panic! All Notes Off button stops any notes still playing in the system. This is useful if you ever get a stuck note from a midi keyboard. Please note that any notes with a slow decay envelope setting will take their own time to end after the panic button is pressed.

The Play Test Note button will play a short note on all audio outputs. This is useful for testing that all audio is working and connected properly.

N.B. If you don't hear a note when you press the Play Test Note button, then please see previous help file: 2 Audio Setup

The Volume slider sets the output level to your soundcard.


2.2 The System and the Interface

The second row of buttons set which window. Each window below shows different parts of the system and the interface:

The system and the interface are like the engine and cockpit, the back and front of the TX Modular System.

The system is where you create modules and connect them together using channels.

The interface is where you can design up to 20 screen layouts, with"widgets" such as knobs, sliders, buttons, and text boxes which are used to control the modules in the system.

MIDI control is also available. For example, you could create a screen with 8 sliders and 8 buttons. These "widgets" can then be connected by MIDI to an external midi controller with 8 sliders and buttons.


2.3 System buttons

The Modules & Channels window is opened when the system starts. It is where you can add new modules to the system and connect them using channels to other modules or outputs or aux busses. This window is explained more fully below.

The Signal Flow window is designed to make it easier to see how a whole system works. It shows how audio and control signals flow around the system between modules, channels and busses. As you add modules and connect them together in the Modules & Channels window, they will automatically be added to the Signal Flow window. Blue lines show audio signals and green lines show control signals. The layout of the Signal Flow panel can be easily changed (by clicking and dragging).

The Sample Banks and Loop Banks windows are for adding individual samples or loops into the system. They can then be used by Sample Player, Loop Player and other modules.

You can set the frequency of samples and the total number of beats in loops.

Groups of samples or loops can also be saved separately as a bank. You can also load one or more whole banks that have been previously created into your current system.

The Notes & Options window is where you can set a background image for the main window (with various image modes for stretching and tiling), change the window colour and transparency, and set "Ask for confirmation before deleting Modules and Channels", which is YES by default. There is also space for adding notes about a built system.


2.4 Interface buttons

The Run Interface window is for running the interface - once it's been built.

The Design Layout window is for adding or deleting widgets - such as sliders or buttons - and setting what each widget looks like and what it controls.

For further details on the Interface windows, see help file: 5 Building an Interface


2.5 History buttons < & >

These move backwards and forwards (< & > ) through the history of previously viewed windows. The arrows turn from grey to white when active.

3.0 Building a system in the Modules & Channels Window with [Example]

[Note: To make the explanation easier to understand, I've added some notes for building an example system in blue italics.

Example: First press the Modules & Channels button.]

Since the Modules & Channels Window is the window you will initially be working with some detailed notes follow.

The window is divided into 4 panels, and each one has a white bar on the right which can be dragged to make the panel wider or narrower. Small coloured square buttons on the top and bottom of the white bar change the width of the panel, from narrowest (black) to widest (orange).

The first and third panels are narrow and contain menu buttons which set what will be displayed in the panel below.

The second panel, which is initially empty, shows the currently selected module with all of its controls. When you add a module, it will immediately be displayed in the second panel, replacing what was previously shown.

The fourth panel, which is also initially empty, shows the all the channels in the system.

The menu buttons in the first and third panels are:

+Source - add source module - this shows all possible source modules that can be added to the system. They are shown alphabetically and in categories.The category buttons can be pressed to close or open the list of modules for that category. The first category is >All Source Modules which is alphabetical.

The “v” and “>” buttons near the top of the panel can be used to open and close all categories in that panel.

At the start of each category name, the “v” and “>” shows whether pressing the button will open or close the category.

+Meter - add meter from source - this shows buttons for adding a meter showing live output levels from any modules or busses in the system. All meters will be shown on a separate window, named Meter Bridge, which opens automatically when you add a meter. The window can be positioned where you like and made to be transparent, or semi-transparent. Audio signal meters are shown in blue, range -60db to 0db. Control signal meters are bipolar - range -1 to +1, and shown in yellow, with a red bar in the middle at 0. A Frequency Scope and Oscilloscope can also be opened using the output of any module or bus.

Edit Modules - edit or delete module - this shows all modules in the system, and will be initially empty. Modules are shown as buttons which can be pressed to display all the module controls in the second panel, Delete buttons are shown next to the module buttons, but can be hidden by making the panel narrower - by dragging the white bar on the right of the panel.

Titles - this shows how many channels are in the system as a total and separately - as Audio and Control channels - there will be no channels initially. Below that are titles for each row of the channels displayed in the next panel followed by controls to select which channels are shown

+Channel - add channel from source - this shows buttons for adding a channel using output signals from any modules or busses in the system.

+Insert - drag to add insert module - this shows all possible insert modules that can be added to the system. They are shown alphabetically and in categories.


3.1 Adding a new module and channel

Press the +Source button to see all the available modules. Use the category buttons to see groups of similar modules.

When you press the Add button next to a module name, the module gets created. Also an audio or control channel is automatically created with it, depending on the type of module you selected. (If you add an action module, then a channel won't be needed.)

[Example: Press the +Source button. In the first category >All Source Modules scroll down to the module called Waveform . Waveform is a mono synthesiser. Press the Add button. N.B. You won't hear any sound until later on in the example.]

The new module will be shown in the second panel and the new channel in the fourth panel - the new module is shown in the channel's source.

Note that once you have added a new module, a number appears in square brackets next to the module's name. This makes it easier to know which module you are dealing with if you create lots of modules of the same type in your system.

[Example: See the button labelled Waveform [1] in the new channel, Channel 1].


3.2 Naming a channel

A white space is provided to the right of the word Name for you to enter a name for the channel.

Note - When you are adding text or numbers onto the screen, once you have finished typing, always make sure to press the Enter or Caraige Return button on your computer keyboard to tell the system to update itself. Otherwise, the system will ignore what you have typed in.

[Example: Type "Synth" into the channel name, and press the Enter button on your computer keyboard.]


3.3 Displaying the controls for a module

This can be done in 2 ways:

1- By pressing the Edit Module button and selecting the module you want.

2- By pressing on the module button in a channel that it's part of, as a source, insert or destination.


3.4 Editing the controls for a module

Once the controls for the new module are shown in the second panel, you can edit them.

Any numbers in the controls of a module can be changed quite quickly by clicking on the number and dragging your mouse.

[Example: Click on Waveform [1] in Channel 1 to display the controls.

Change the waveform from a Sine wave to a Sawtooth wave using the Waveform box.

Then click on the Frequency button, and change the Frequency to 200 using the Freq slider or by clicking and dragging over the number itself.]

Note: Help, Pause, Delete, Rebuild and Presets buttons:

All modules have a Help button which opens a help window specifically for that module.

Most modules also have a Pause button (which stops it running temporarily until pressed again). All output from the module will stop if it is paused.

They also have a Delete button. If you delete a source module, the channels that it is in will also get deleted, along with any insert modules in the channel.

Most modules also have a Rebuild button. This rebuilds a module in case for any reason it has stopped running or crashed (this is rarely needed!).

There is also a Presets popup button with options for:

- storing and loading module settings as Presets.

- copying and pasting all the settings from one module to another using a clipboard - both modules will need to be the same module type for settings to be copied.

- saving and loading all the settings to or from a file (useful for using the same settings in a different TX Modular system).

A Comments box is provided at the top of each module for you to add any notes. When a module Preset is created, the comments text will be displayed next to the Preset no, so it can be used to give each Preset a name.


3.5 Selecting a source bus in a channel

Some modules, such as the Audio In module, will have more than one output bus that you can select in the popup box next to the words Source Bus .

[Example: The Waveform [1] module has only one option for source bus: Out. There is a also a similar module - Waveform St - which is a stereo version and has output options Out L + R, Out L Only, Out R Only]


3.6 Adding insert modules to a channel

You have the option of adding insert modules into the 5 slots provided in each channel. Audio and control channels have different options for insert modules.

Press the +Insert button to see all the available modules. The first category shows All Insert Modules alphabetically. Other categories are separated into Audio and Control types. Audio insert modules have blue drag buttons & Control insert modules have green buttons.

To add an insert module, click on the drag button to the left of the module name, and drag it to one of the insert slots in a channel. The module will be added and it's controls will appear in the second panel.

Note - If you add an incorrect insert module, you can use the d (delete) button which appears next to it to remove it from the channel.

[Example: Press the +Insert button, find the Flanger module and drag from the drag button to one of the insert slots of Channel 1. The controls for the flanger will appear in the second panel.

Change the Dry-Wet Mix of the Flanger to 0.2 for a more subtle effect.]


3.7 Setting a destination for a channel

The orange box marked add destination selects where the signal in a channel will be sent.

[Example: Select Main Outs as the destination]

Note - If you add an incorrect destination module, you can use the d (delete) button which appears next to it to remove it from the channel.

If the destination has more than one destination bus, a different one can be selected in the popup button to the right of the word Dest. Bus.

[Example: Main Outs has 8 options for Dest. Bus: Out 1 + 2, Out 3 + 4, ... Out 15 + 16.

Set it to Out 1 + 2. Adjust the volume using the Level slider for Channel 1. You should now hear the synth playing and see its output level in the channel meters to the right of the level slider.]


3.8 Modulating the controls of a module

Once you've added a module, if you want to modulate any of its controls, you can add a new control source module (such as an LFO) to the system, and set the channel destination and destination bus to the module control you want to modulate. This is best understood by seeing the example below.

[Example: Press the the +Source button, if not already pressed. Find and add a new LFO module by pressing the relevant Add button.

Set the Frequency of LFO [1] to 0.25.

You will see a new control channel , Channel 2, with LFO [1] as the source.

Set the destination of this channel to Waveform [1].

Set the dest. bus of this channel to Frequency and set the level slider of Channel 2 to 50.

You will hear the LFO affecting the frequency of the Waveform [1] module and see its output in the channel meter to the right of the level slider.]

You can use the level slider to control how much modulation is applied. N.B. However much modulation you add, the control that you are modulating will stay within the limits of its minimum and maximum values.

[Example: Use the level slider on Channel 2 to change the amount of modulation. Notice that on the maximum level, the frequency reaches a ceiling level which it cannot go above. This ceiling is set by the Min-Max controls below the Frequency slider in the Waveform [1] module. By changing these, you can change how high or low the Frequency is allowed to go.]

The invert checkbox will invert the modulation meaning that the control signal will be subtracted instead of added.

The off checkbox in the channel will stop the modulation.

[Example: Press the Invert checkbox and you will hear the LFO reducing the frequency of the Waveform [1] module, instead of increasing it. Press it again to go back to the increasing frequency.

Press the Off checkbox and the modulation will stop. Press again and the modulation will start again.]

4.0 Using FX send busses in audio channels

Once you've activated an audio channel, you can send the signal to an FX Send bus by pressing the white FX send buttons to the right of the words FX Send 1-4 (press the Titles button to see the channel row titles).

Once they have been pressed the FX send buttons will show the number of the FX Send bus selected (1/2/3/4).

To the right of each FX send button is a slider to set the signal level sent to the FX send bus.

[Example: Turn on the FX send 1 button in Channel 1 for the Waveform [1] module.

Then set the FX send 1 slider of Channel 1 to 100 so the full signal is sent.

You won't hear any changes yet.]

Note - these are post-fader FX sends, meaning that the channel level slider affects the level of the signal sent to the FX send bus as well as the FX send slider. If the channel level is 0, or if the channel is muted, then no signal will be sent to the FX send bus even if the FX send slider is turned up high.


4.1 Creating a channel for an FX send bus

Press the +Channel button to see all the available channel sources.

Press the Add button for the required FX send bus to add the channel.

Once the new channel has been created, insert modules and a destination can then be set.

[Example: Press the +Channel button, find FX Send 1 and press the Add button to add a new channel - Channel 3 - with FX send 1 as the source.

Press the +Insert button, find the ReverbG module and drag from the drag button to one of the insert slots of the new Channel 3.

Select Main Outs as the destination and set the level slider to 50.

You should now hear the ReverbG added to the audio signal.

On the ReverbG [1] settings, try changing the Room size or Reverb time.

Select one of the reverb Presets that is marked "as send effect".]

5.0 Connecting a module to more than one destination

Press the +Channel button to see all the available channel sources.

You can use the Add buttons to make a new channel for any module that you have already created or any busses which are built into the system, such as Aux and FX Send busses.

This can be very useful, for example to enable an LFO to modulate more than one destination.

[Example: Press the +Channel button, find LFO [1] module and press the Add button to add a new channel .You will see a new Channel 4 created with the LFO [1] module as the source.

Select Flanger [1] as the destination and LFO rate as the dest. bus.

Gradually bring up the Level of the new channel 4 and you will hear the LFO affecting the speed of Flanger [1] .

Click on the LFO [1] button to display the LFO settings. Try changing the Frequency or Waveform of the LFO to hear how they affect the sound.]

Note - you can also use the Dup button at the top of each channel. This will duplicate the channel, creating a new channel next the current channel and with the same source module.
