The TX Modular System - Links


The TX Modular System is free audio-visual software available from: http://www.palemoonrising.co.uk

The TX Modular System audio engine is written using SuperCollider (https://supercollider.github.io), an environment and programming language for real time audio synthesis and algorithmic composition.

The TXV visual app is built using openFrameworks (https://openFrameworks.cc).
