Frequency Modulation Synthesis


Introduction:

Frequency Modulation (FM) synthesis is used in various TX modules including Granulator FM, Waveform & Wave Synth.

This help file focusses on the modules FM Drone & FM Synth which both use 6-Operator Frequency Modulation. Operators are the equivalent to oscillators in FM terminology, and they can each modulate the frequency of any of the 6 operators, including themselves (self-modulation).

An operator which modulates another operator is called a Modulator, and an operator which can be heard is called a Carrier.

The term Algorithm (see controls below) means a particular configuration of modulation and output level settings. It's also possible to randomise the modulation connections.

The image "Frequency Modulation Synthesis 1.png" shows 32 Algorithms that were used in the Yamaha DX7 synth (and other models) These can be loaded as preset patterns of modulation.

[Note - technically speaking, although it's called Frequency Modulation, it is actually Phase Modulation where the phase of one oscillator is modulated by another oscillator.]


Operator Controls in modules FM Drone & FM Synth:

(See image: "Frequency Modulation Synthesis 2.png")


All 1/All 2/Op 1/Op 2/Op 3/Op 4/Op 5/Op6/Ratios/Freqs buttons - these select which operator(s) are being edited.

All 1/All 2/Ratios/Freqs buttons show all 6 operators in various configurations for easy editing. Various randomisation processes are shown for generating new settings.

Use fixed freq - if turned on, the fixed frequency will be used instead of the frequency ratio to set the operators frequency.

Freq ratio - this sets the frequency of the current operator in relation to the base frequency (in FM Synth the base frequency is the MIDI note played).
For example, if the frequency ratio is 1, the operator's frequency will be the same as the base frequency. Whereas if the frequency ratio is 0.5, the operator's frequency will be an octave below the base frequency; or 2 for an octave above.

Fixed freq - sets the fixed frequency of the current operator

Detune - detunes the frequency between around -1 and +1 semitones

Phase - this sets the starting phase of the current operator.

Amplitude - the amplitude of the operator affects how much it can modulate the frequency of other operators.

Mod Op 1/Mod Op 2/Mod Op 3/Mod Op 4/Mod Op 5/Mod Op 6 - these set the amount of frequency modulation coming from operators 1- 6 to affect the frequency of the current operator. This allows for self-modulation and feedback.

Level - this sets the output level of the current operator. If the level is above zero, the operator is heard in the audio output (i.e. it is a Carrier)

Algorithm - select an algorithm to be loaded. An algorithm will set controls Mod Op 1,Mod Op 2,Mod Op 3,Mod Op 4,Mod Op 5,Mod Op 6, Level for each operator.
The first 32 are the DX7 algorithms (see above). The others use randomised settings to create new modulation connections.

Algo feedback - this sets the modulation level where there is self-modulation or feedback in the modulation path. It affects all algorithms including the random ones & can be used to help limit the amount of extreme modulation (which can sound harsh).

Load Algorithm - this loads the chosen algorithm taking into account the Algo feedback setting.
