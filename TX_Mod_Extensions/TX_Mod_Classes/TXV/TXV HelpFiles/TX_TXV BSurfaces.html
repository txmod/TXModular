TXV BSurfaces

Properties: DrawSource ImageAsset

See also: TXV System


Introduction:

This module creates multiple Bezier surfaces, which are curved surfaces based on Bezier curves with 16 control points in a 4 X 4 grid.
The curved surface is based on the shape of the grid. Each point can be controlled independently and modulated.

There are also preset shapes, 20 user-defined and & 30 pre-defined, which can be morphed between to define the shape of the surface.
User-defined preset shapes are created by adding up to 20 x/y/z point offsets to particular groups of points over a range of presets.
Morphing between up to 5 presets can be defined in 1 or 2 dimensions.

Variations between multiple surfaces can be created: points and surface material colours have optional end values with point and color spread curves. The spread curves define where the parameter values fall between their start and end values for each surface.

An optional switch "useMirroring" creates a second surface as a mirror image of the first, with a slider "mirrorEdgeMerge" which joins the edges of both surfaces together to make a single continuous surface.
