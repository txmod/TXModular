Introduction

TXV is a modular video app for generating real-time 2D & 3D visuals. It was created to enable a TX Modular user to control both video and audio processes, and link them together.

TXV is a separate app that opens in it’s own window TXV is controlled by adding a TXV System module to the TX Modular and editing the controls or using modulation or automation.

Tech note: TXV was written using C++ and GLSL languages running inside OpenFrameworks 0.9.8. Internally, OSC network messages are used to communicate between the TX Modular System and TXV.


Installing TXV on different platforms

There are various ways to install TX Modular:

There’s a MacOS standalone TX Modular app that includes SuperCollider & TXV within it.
The standalone will only work with MacOS Mojave (version 10.14) and earlier versions of MacOS (versions 10.11,10.12 & 10.13), but NOT MacOS Catalina (version 10.15) or later.

The TX audio engine, in the form of SuperCollider code, should work with MacOS, Windows and Linux platforms with SuperCollider installed.

The TX visual engine (the separate TXV app) has been compiled for MacOS and Windows.
TXV compiled for MacOS will work with Mojave (version 10.14) and earlier MacOS versions (10.11,10.12 & 10.13) - but NOT MacOS Catalina (version 10.15) or later.

To use the TXV visual engine on Linux or MacOS Catalina (version 10.15) or later, it should hopefully be possible to compile TXV from source code using openFrameworks 0.9.8.
(Note: compiling on these platforms has not yet been tested)


TXV Modules

TXV includes various types of modules:

Draw Sources such as image file, movie file, live video camera, 3D Model file, 2D/3D shapes.

Image FX such as Blend, Blur, Pixelate and Colorize.

Draw Destinations include the TXV System, Transform (shift/ scale/ rotate) and Transf3DMulti (transform & make multiple copies of drawn objects), Scene (3D Scene with cameras & lights).

Modulation sources include LFOs, Perlin noise, and a Step Sequencer.


Module Connections

There are two modules which connect TXV modules together:

Draw Connections are modules that connect Draw Source modules to Draw Destination modules. For example a TXV Movie module could be connected to the TXV System to display it on the main window.

Nodes are modules that connect modulation sources (such as LFOs) to parameters in other modules to enable modulation to happen.


Image Connections

The various modules that use images have the option either to use image files stored on disk or to directly import an image produced by another image module (including a video camera). This allows image FX modules to be chained together for more complex FX.
