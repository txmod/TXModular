TXV Rendering Images


See also: TXV System


Introduction

Within TXV, some modules are able to render images directly. This is shown in the module properties at the top of each module's help file, marked as "Renders".

The help file TXV List Of All Modules shows the properties of all modules. They are shown in groups:
In the group Draw Sources, a few modules render an image - for example TXV Gradient & TXV Slide Show.
In the group Image FX all the modules render an image.
In the group Draw Destinations, a few modules render an image - for example TXV EnvMap & TXV Feedback.

Any other module that uses images can import the rendered image directly. So for example, by using each rendered image as the source of the next effect, you could create a chain of FX before using the final FX image in the material wrapped around a 3D shape.


TXV Render Image - render & save images & video

The TXV Render Image module can be used to create an image from drawn elements. It is a Draw Destination module, so other modules can be drawn onto it.

For example, you might want to create an image from some drawn 2D shapes to be used as a mask image.

TXV Render Image module can also record single images or continuous video, so this is how you would save any output from the TXV.


TXV Image vs. TXV Tile & Mask

The difference between these 2 modules is also to do with rendering:

The TXV Image module draws an image (or part of it) with optional tiling and masking.

TXV Tile & Mask is exactly the same except that it also renders the image before drawing it. This is useful if you need to import the tiled or masked image directly into another module, rather than just drawing it.
