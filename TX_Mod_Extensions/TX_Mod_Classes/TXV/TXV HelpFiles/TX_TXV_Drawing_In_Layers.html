TXV Drawing In Layers


See also: TXV System


Within TXV, some modules are Draw Sources and some modules are Draw Destinations. (This is shown in the module properties at the top of each module's help file, marked as DrawSource & DrawDestination).

See TXV List Of All Modules for the properties of all modules, grouped as Draw Sources, Image FX (which are also Draw Sources) and Draw Destinations.

In order for drawing to happen a Draw Source module needs to be connected to a Draw Destination using a TXV DrawConnection module.

Each Draw Destination module has 20 layers which are drawn in numerical order, so layer 1 is drawn before layer 2. There's also a parameter "drawLayersRule" to choose which single layer or group of layers are drawn.

Multiple draw sources can be connected to each layer of a Draw Destination module, but if draw order is important, each source should have its own layer.

Chains of draw connections can be made, for example connecting an Image module to a Transform module to scale & rotate it & connecting the Transform module to the TXV System to display it.

To draw to the main window, set the Draw Destination of a DrawConnection to the TXV System module.
