TXV Node


See also: TXV System & TXV Using Modulation


Introduction:

This module connects a modulation source to a parameter in another module so it can be modulated.
Note: the destination parameter needs to have it's Modulation Mix set above zero for the modulation to take effect.

Negative modulation, which is optional, will invert the modulation signal.


Controls:

active - switch (def: 1).

modAmount - float with modulation (def: 1, range: 0 : 1).

negativeModulation - switch with modulation (def: 0). This inverts the modulation signal.


Modulation Inputs:

modAmount

negativeModulation
