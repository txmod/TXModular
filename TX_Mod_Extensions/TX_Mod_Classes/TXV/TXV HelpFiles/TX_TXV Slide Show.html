TXV Slide Show

Properties: DrawSource Renders

See also: TXV System


Introduction:

This module creates a slide show, crossfading between images from a folder, with automatic or manual triggering.


Controls:


Source & X-fade Button

sourceImageScaleMode - index with modulation (def: 1, range: 0 : 2) options: [0]_scale to fit, [1]_scale & crop to fill, [2]_stretch to fill.

folderName - Text.

randomiseOrder - switch with modulation (def: 0).

xFadeType - index with modulation (def: 23, range: 0 : 23) options: [0]_Cubic In, [1]_Cubic Out, [2]_Cubic In & Out, [3]_Elastic In, [4]_Elastic Out, [5]_Elastic In & Out, [6]_Exponential In, [7]_Exponential Out, [8]_Exponential In & Out, [9]_Linear In (default), [10]_Linear Out, [11]_Linear In & Out, [12]_Quadratic In, [13]_Quadratic Out, [14]_Quadratic In & Out, [15]_Quartic In, [16]_Quartic Out, [17]_Quartic In & Out, [18]_Quintic In, [19]_Quintic Out, [20]_Quintic In & Out, [21]_Sine In, [22]_Sine Out, [23]_Sine In & Out.

runSlideShow - switch with modulation (def: 0).

pauseSlideShow - switch with modulation (def: 0).

triggerXFadeForwards - switch with modulation (def: 0).

triggerXFadeBackwards - switch with modulation (def: 0).

triggerXFadeToRandom - switch with modulation (def: 0).

jumpToFirstImage - switch with modulation (def: 0).

jumpToNextImage - switch with modulation (def: 0).

jumpToPreviousImage - switch with modulation (def: 0).

jumpToRandomImage - switch with modulation (def: 0).


Times Button

xFadeTimeFixed - float with modulation (def: 3, range: 0.1 : 1000).

xFadeTimeRandom - float with modulation (def: 0, range: 0 : 1000).

holdTimeFixed - float with modulation (def: 3, range: 0.1 : 1000).

holdTimeRandom - float with modulation (def: 0, range: 0 : 1000).


Clear & Render :: Clear Button

clearBeforeRender - switch with modulation (def: 1).

clearNow - switch with modulation (def: 0).

clearColorHue - float with modulation (def: 0.5, range: 0 : 1).

clearColorSaturation - float with modulation (def: 0.6, range: 0 : 1).

clearColorBrightness - float with modulation (def: 0, range: 0 : 1).

clearColorAlpha - float with modulation (def: 0, range: 0 : 1).

clearColorHSBAsRGB - switch (def: 0).


Clear & Render :: Render Button

renderContinuosly - switch with modulation (def: 1).

renderNow - switch with modulation (def: 0).

renderWidthHeightRule - index (def: 0, range: 0 : 6) options: [0]_Use Screen Width & Height, [1]_Square - Use Max of Screen Width & Height, [2]_Square - Use Min of Screen Width & Height, [3]_Square - Use Screen Width, [4]_Square - Use Screen Height, [5]_Use Custom Size, [6]_Use Source Image Size.

renderCustomWidth - integer (def: 1024, range: 1 : 4096).

renderCustomHeight - integer (def: 768, range: 1 : 4096).


Draw :: Active, Size, Alpha Button

drawActive - switch with modulation (def: 1).

maxWidthHeightRule - index (def: 0, range: 0 : 5) options: [0]_Use Screen Width & Height, [1]_Square - Use Max of Screen Width & Height, [2]_Square - Use Min of Screen Width & Height, [3]_Square - Use Screen Width, [4]_Square - Use Screen Height, [5]_Use Custom Size.

customMaxWidth - integer (def: 1024, range: 1 : 4096).

customMaxHeight - integer (def: 768, range: 1 : 4096).

customMaxDepth - integer (def: 1024, range: 1 : 4096).

drawAlpha - float with modulation (def: 1, range: 0 : 1).

drawWidth - float with modulation (def: 1, range: 0 : 1).

drawHeight - float with modulation (def: 1, range: 0 : 1).


Draw :: Scale Button

scaleX - float with modulation (def: 1, range: -10 : 10).

scaleY - float with modulation (def: 1, range: -10 : 10).

useScaleXForScaleY - switch (def: 0).


Draw :: Position, Anchor, Rotate Button

drawPosX - float with modulation (def: 0.5, range: 0 : 1).

drawPosY - float with modulation (def: 0.5, range: 0 : 1).

anchorX - float with modulation (def: 0.5, range: 0 : 1).

anchorY - float with modulation (def: 0.5, range: 0 : 1).

rotate - float with modulation (def: 0, range: -360 : 360).

rotateMultiply - float with modulation (def: 1, range: 0 : 10).
