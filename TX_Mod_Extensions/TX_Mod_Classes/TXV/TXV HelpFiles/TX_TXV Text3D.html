TXV Text3D

Properties: DrawSource FontAsset

See also: TXV System


Introduction:

This module draws 3D text using a chosen font.

Note: Font assets are added directly to the TXV System module.


Controls:


Text Controls Button

fontAsset - index (def: 0) - Font Asset.

textString - Text.

fontsize - integer with modulation (def: 24, range: 1 : 1000).

scaleFontToWidth_1024 - switch (def: 1). When this is on, the font size will only be accurate if the the screen width is 1024. Otherwise the font will be scaled based on the current screen width relative to 1024. This makes the text a consistent size for all screen widths. Turn this off if you want font size to always be accurate.

depth - integer with modulation (def: 24, range: 1 : 1000).

depthResolution - integer with modulation (def: 1, range: 1 : 1000).

fontSimplify - float with modulation (def: 0.3, range: 0 : 1).

spaceSize - float with modulation (def: 5, range: 0.01 : 1000).

letterSpacing - float with modulation (def: 1, range: 0.01 : 100).


Face Options Button

drawLetterFrontSolid - switch with modulation (def: 1).

drawLetterFrontWireframe - switch with modulation (def: 0).

drawLetterBackSolid - switch with modulation (def: 1).

drawLetterBackWireframe - switch with modulation (def: 0).

drawLetterSidesSolid - switch with modulation (def: 1).

drawLetterSidesWireframe - switch with modulation (def: 0).

colorFrontSaturation - float with modulation (def: 0.6, range: 0 : 1).

colorFrontBrightness - float with modulation (def: 0.5, range: 0 : 1).

colorFrontHSBAsRGB - switch with modulation (def: 1).

Draw :: Active & Position Button

drawActive - switch with modulation (def: 1).

maxWidthHeightRule - index (def: 0, range: 0 : 5) options: [0]_Use Screen Width & Height, [1]_Square - Use Max of Screen Width & Height, [2]_Square - Use Min of Screen Width & Height, [3]_Square - Use Screen Width, [4]_Square - Use Screen Height, [5]_Use Custom Size.

customMaxWidth - integer (def: 1024, range: 1 : 4096).

customMaxHeight - integer (def: 768, range: 1 : 4096).

customMaxDepth - integer (def: 1024, range: 1 : 4096).

positionX - float with modulation (def: 0.5, range: -20 : 20).

positionY - float with modulation (def: 0.5, range: -20 : 20).

positionZ - float with modulation (def: 0, range: -20 : 20).

scaleZ - float with modulation (def: 1, range: -10 : 10).

useScaleXForScaleY - switch (def: 0).

useScaleXForScaleZ - switch (def: 0).

rotateY - float with modulation (def: 0, range: -360 : 360).

rotateZ - float with modulation (def: 0, range: -360 : 360).

rotateMultiply - float with modulation (def: 1, range: 0 : 10).

anchorX - float with modulation (def: 0.5, range: 0 : 1).

anchorY - float with modulation (def: 0.5, range: 0 : 1).

anchorZ - float with modulation (def: 0, range: 0 : 1).
