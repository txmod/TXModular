TXV 3D Model

Properties: DrawSource 3DModelAsset

See also: TXV System


Introduction:

This module can load 3D Model files (using the 'Open Asset Import Library' version 2).
If the file contains an animation, it can be played through (manually or automatically).

Note: 3D Model assets are added directly to the TXV System module.


Controls:


Main Button

modelAsset - index (def: 0) - 3D Model Asset.

createLightsFromModel - switch with modulation (def: 0).

loopType - index with modulation (def: 3, range: 1 : 3) options: [0]_No Looping, [1]_Forwards & Backwards Looping, [2]_Normal Looping.

play - switch with modulation (def: 0).

pause - switch with modulation (def: 0).

goToStart - switch with modulation (def: 0).

animationPosition - float with modulation (def: 0, range: 0 : 1).


Position Button

drawPosX - float with modulation (def: 0.5, range: -20 : 20).

drawPosY - float with modulation (def: 0.5, range: -20 : 20).

drawPosZ - float with modulation (def: 0, range: -20 : 20).

adjustPosToSceneCentre - switch (def: 0).


Scale Button

scaleX - float with modulation (def: 1, range: -10 : 10).

scaleY - float with modulation (def: 1, range: -10 : 10).

scaleZ - float with modulation (def: 1, range: -10 : 10).

useScaleXForScaleY - switch (def: 0).

useScaleXForScaleZ - switch (def: 0).


Rotate Button

rotateX - float with modulation (def: 0, range: -360 : 360).

rotateY - float with modulation (def: 0, range: -360 : 360).

rotateZ - float with modulation (def: 0, range: -360 : 360).

rotateMultiply - float with modulation (def: 1, range: 0 : 10).


Draw Parameters Button

drawActive - switch with modulation (def: 1).

maxWidthHeightRule - index (def: 0, range: 0 : 5) options: [0]_Use Screen Width & Height, [1]_Square - Use Max of Screen Width & Height, [2]_Square - Use Min of Screen Width & Height, [3]_Square - Use Screen Width, [4]_Square - Use Screen Height, [5]_Use Custom Size.

customMaxWidth - integer (def: 1024, range: 1 : 4096).

customMaxHeight - integer (def: 768, range: 1 : 4096).

customMaxDepth - integer (def: 1024, range: 1 : 4096).
