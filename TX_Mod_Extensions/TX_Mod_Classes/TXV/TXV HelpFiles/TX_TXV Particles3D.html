TXV Particles3D

Properties: DrawSource ImageAsset

See also: TXV System


Introduction:

This module draws 3D particles with the same parameters as TXV Shape 3D including options for material properties, image-wrapping, and shape distortion.

Automatic or manual triggering creates new particles, each with a fixed lifetime.

Particles can move in various ways and change size or colour during their lifetimes.

Several parameters have an associated curve (for example, width and widthCurve, height and heightCurve).
The curve is used to scale the parameter over it's lifetime.
By default the curves are set to maximum so the parameter remains unchanged, but they can be edited to make a parameter grow or shrink during the particle's lifetime.

For best results, Particles3D should be drawn on a TXV Scene (or other 3D draw destination module) to add lighting and camera control.
