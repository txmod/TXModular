TXV Transform3D

Properties: DrawSource DrawDestination

See also: TXV System


Introduction:

This module is a 3D Draw Destination that draws anything drawn to it after applying a 3D "transform" which includes position shift, scale and rotate in any order.


Controls:


Transform & Shift Button

transformOrder - index with modulation (def: 0, range: 0 : 5) options: [0]_Shift Rotate Scale, [1]_Shift Scale Rotate, [2]_Rotate Scale Shift, [3]_Rotate Shift Scale, [4]_Scale Shift Rotate, [5]_Scale Rotate Shift.

shiftX - float with modulation (def: 0, range: -10 : 10).

shiftY - float with modulation (def: 0, range: -10 : 10).

shiftZ - float with modulation (def: 0, range: -10 : 10).


Scale Button

scaleX - float with modulation (def: 1, range: -10 : 10).

scaleY - float with modulation (def: 1, range: -10 : 10).

scaleZ - float with modulation (def: 1, range: -10 : 10).

useScaleXForScaleY - switch (def: 0).

useScaleXForScaleZ - switch (def: 0).


Rotate Button

rotateX - float with modulation (def: 0, range: -360 : 360).

rotateY - float with modulation (def: 0, range: -360 : 360).

rotateZ - float with modulation (def: 0, range: -360 : 360).

rotateMultiply - float with modulation (def: 1, range: 0 : 10).

anchorX - float with modulation (def: 0.5, range: -10 : 10).

anchorY - float with modulation (def: 0.5, range: -10 : 10).

anchorZ - float with modulation (def: 0, range: -10 : 10).


Draw Parameters Button

drawActive - switch with modulation (def: 1).

maxWidthHeightRule - index (def: 0, range: 0 : 5) options: [0]_Use Screen Width & Height, [1]_Square - Use Max of Screen Width & Height, [2]_Square - Use Min of Screen Width & Height, [3]_Square - Use Screen Width, [4]_Square - Use Screen Height, [5]_Use Custom Size.

maxDepthRule - index (def: 0, range: 0 : 4) options: [0]_Use Screen Width, [1]_Use Screen Height, [2]_Use Max of Screen Width & Height, [3]_Use Min of Screen Width & Height, [4]_Use Custom Size.

customMaxWidth - integer (def: 1024, range: 1 : 4096).

customMaxHeight - integer (def: 768, range: 1 : 4096).

customMaxDepth - integer (def: 1024, range: 1 : 4096).

drawLayersRule - index with modulation (def: 0, range: 0 : 26) options: [0]_All layers except Layer 20, [1]_Layer 1, [2]_Layer 2, [3]_Layer 3, [4]_Layer 4, [5]_Layer 5, [6]_Layer 6, [7]_Layer 7, [8]_Layer 8, [9]_Layer 9, [10]_Layer 10, [11]_Layer 11, [12]_Layer 12, [13]_Layer 13, [14]_Layer 14, [15]_Layer 15, [16]_Layer 16, [17]_Layer 17, [18]_Layer 18, [19]_Layer 19, [20]_Layer 20, [21]_Layers 1 - 5, [22]_Layers 6 - 10, [23]_Layers 11 - 15, [24]_Layers 16 - 20, [25]_Layers 1 - 10, [26]_Layers 11 - 20.
