TXV Draw Connection


See also: TXV System & TXV Drawing In Layers


Introduction:

This module connects a Draw Source module to a particular draw layer in a Draw Destination module (which all have 20 draw layers).
Whenever the Draw Destination module draws that particular layer, the Draw Source module will be drawn.

To draw to the main window, set the Draw Destination to the TXV System module.


Controls:

source - Source module which will be drawn.

destination - Destination module will be drawn to.

layerNo - Layer number (1-20) to be drawn to.

active - switch with modulation (def: On).
