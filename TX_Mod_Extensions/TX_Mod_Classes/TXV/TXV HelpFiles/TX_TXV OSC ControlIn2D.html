TXV OSC ControlIn2D

Properties: ModulationSource

See also: TXV System


Introduction:

This module listens for Open Sound Control (OSC) network messages sent to a particular OSC address.

If the first 2 message arguments are numbers (float or integer), they will be normalised (mapped from the range inputMin:inputMax to the range 0:1) and output as 2 modulation signals.


Controls:

active - switch with modulation (def: 1).

oscAddress - Text.

input1Min - float with modulation (def: 0, range: -1000000 : 1000000).

input1Max - float with modulation (def: 1, range: -1000000 : 1000000).

input2Min - float with modulation (def: 0, range: -1000000 : 1000000).

input2Max - float with modulation (def: 1, range: -1000000 : 1000000).

smoothTime - float with modulation (def: 0, range: 0 : 10).

linearSmoothing - switch with modulation (def: 0).


Modulation Outputs:

out1

out2


Modulation Inputs:

active

input1Min

input1Max

input2Min

input2Max

smoothTime

linearSmoothing
