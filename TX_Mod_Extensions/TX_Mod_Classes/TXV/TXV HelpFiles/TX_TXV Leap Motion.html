TXV Leap Motion

Properties: ModulationSource

See also: TXV System


Introduction:

Note: Broken module! - not currently working.

This module connects to a Leap Motion sensor for sensing left and right hand positions and outputs modulation signals for various hand and finger measurements.


Controls:


Main Button

active - switch with modulation (def: 1).

handMode - index with modulation (def: 0, range: 0 : 2) options: [0]_2 hands, separate spaces, [1]_2 hands, shared space, [2]_1 hand.

swapHands - switch with modulation (def: 0).

getFingerData - switch with modulation (def: 0).

stabilizePalmPosition - switch with modulation (def: 1).

velocityScaling - float with modulation (def: 1, range: 0.01 : 10).

palmPositionCentreBias - float with modulation (def: 0, range: 0 : 1).

palmRotationCentreBias - float with modulation (def: 0, range: 0 : 1).


Curves Button

grabCurve - drawable curve (size: 256 points).

pinchCurve - drawable curve (size: 256 points).


Smoothing Button

positionSmoothTime - float with modulation (def: 0, range: 0 : 10).

rotationSmoothTime - float with modulation (def: 0, range: 0 : 10).

velocitySmoothTime - float with modulation (def: 0, range: 0 : 10).


Interaction Box Button

XAxisMinMM - integer (def: -250, range: -1000 : 1000).

XAxisMaxMM - integer (def: 250, range: -1000 : 1000).

YAxisMinMM - integer (def: 200, range: 90 : 1000).

YAxisMaxMM - integer (def: 500, range: 90 : 1000).

ZAxisMinMM - integer (def: -150, range: -1000 : 1000).

ZAxisMaxMM - integer (def: 150, range: -1000 : 1000).


Send OSC to SC Button

sendLeftHandDataToSC - switch (def: 0).

sendRightHandDataToSC - switch (def: 0).

sendPositionToSC - switch (def: 1).

sendPinchGrabToSC - switch (def: 1).

sendRotationToSC - switch (def: 1).

sendVelocityToSC - switch (def: 0).

sendFingerDataToSC - switch (def: 0).
