TXV LFO 3D

Properties: ModulationSource

See also: TXV System


Introduction:

This module is a 3D "Low Frequency Oscillator" (LFO) which outputs a 3 modulation signals based on a preset waveform or custom curves.


Controls:


Main Button

lfoType - index with modulation (def: 1, range: 0 : 18) options: [0]_No Pattern, [1]_Circles, [2]_Spirals, [3]_Spiral Sine, [4]_Spiral Tri, [5]_Lines, [6]_Diagonal Lines, [7]_Grid, [8]_Sine Wave, [9]_Zig Zag, [10]_Random, [11]_Random Quantised, [12]_Random Smoothed, [13]_Double Random, [14]_Double Random Quantised, [15]_Double Random Smoothed, [16]_Trefoil Knot, [17]_Custom Curve, [18]_Custom Curve - Mirrored.

randomSmoothType - index with modulation (def: 23, range: 0 : 23) options: [0]_Cubic In, [1]_Cubic Out, [2]_Cubic In & Out, [3]_Elastic In, [4]_Elastic Out, [5]_Elastic In & Out, [6]_Exponential In, [7]_Exponential Out, [8]_Exponential In & Out, [9]_Linear In (default), [10]_Linear Out, [11]_Linear In & Out, [12]_Quadratic In, [13]_Quadratic Out, [14]_Quadratic In & Out, [15]_Quartic In, [16]_Quartic Out, [17]_Quartic In & Out, [18]_Quintic In, [19]_Quintic Out, [20]_Quintic In & Out, [21]_Sine In, [22]_Sine Out, [23]_Sine In & Out.

noLevels - integer with modulation (def: 1, range: 1 : 20).

circularDirectionOut - switch with modulation (def: 1).


Adjust X,Y,Z

reverseX - switch with modulation (def: 0).

reverseY - switch with modulation (def: 0).

reverseZ - switch with modulation (def: 0).

swapXAndY - switch with modulation (def: 0).

scaleX - float with modulation (def: 1, range: 0 : 1).

scaleY - float with modulation (def: 1, range: 0 : 1).

scaleZ - float with modulation (def: 1, range: 0 : 1).


BPM, Cycle, Phase Button

bpm - float with modulation (def: 120, range: 0.001 : 1000).

useMasterBpm - switch with modulation (def: 1).

cycleTimeBeats - float with modulation (def: 16, range: 1 : 1000).

cycleTimeDivisor - float with modulation (def: 1, range: 1 : 1000).

phaseOffset - float with modulation (def: 0, range: 0 : 1).

frozen - switch with modulation (def: 0).

resetClock - switch with modulation (def: 0).


Custom Curve X Button

customCurveX - drawable curve (size: 256 points).

customCurveX2 - drawable curve (size: 256 points).

customCurveXMorph - float with modulation (def: 0, range: 0 : 1).


Custom Curve Y Button

customCurveY - drawable curve (size: 256 points).

customCurveY2 - drawable curve (size: 256 points).

customCurveYMorph - float with modulation (def: 0, range: 0 : 1).


Custom Curve Z Button

customCurveZ - drawable curve (size: 256 points).

customCurveZ2 - drawable curve (size: 256 points).

customCurveZMorph - float with modulation (def: 0, range: 0 : 1).
