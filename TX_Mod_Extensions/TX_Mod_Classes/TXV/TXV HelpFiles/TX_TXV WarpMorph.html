TXV WarpMorph

Properties: ModulationSource

See also: TXV System


Introduction:

This module is for "mapping" or changing the behaviour of a modulation signal.

The input value is mapped using a morph between 2 preset or custom transfer curves, with optional quantization.

The horizontal (left-right) axis of the curve represents the range of values of the input signal (based on the input range).

The vertical (up-down) axis of the curve represents the range of values of the output signal (based on the output range).

The default curve is a linear curve which will have no effect on the input.


Controls:


Main Button

inputValue - float with modulation (def: 0, range: 0 : 1).

warpType - index with modulation (def: 0, range: 0 : 15) options: [0]_Linear, [1]_Exponential, [2]_Minus Exponential, [3]_Curve 5, [4]_Curve 4, [5]_Curve 3, [6]_Curve 2, [7]_Curve Minus 2, [8]_Curve Minus 3, [9]_Curve Minus 4, [10]_Curve Minus 5, [11]_Phase Shift 90, [12]_Phase Shift 180, [13]_Phase Shift 270, [14]_Custom Curve 1, [15]_Custom Curve 2.

warpType2 - index with modulation (def: 0, range: 0 : 15) options: [0]_Linear, [1]_Exponential, [2]_Minus Exponential, [3]_Curve 5, [4]_Curve 4, [5]_Curve 3, [6]_Curve 2, [7]_Curve Minus 2, [8]_Curve Minus 3, [9]_Curve Minus 4, [10]_Curve Minus 5, [11]_Phase Shift 90, [12]_Phase Shift 180, [13]_Phase Shift 270, [14]_Custom Curve 1, [15]_Custom Curve 2.

warpTypeMorph - float with modulation (def: 0, range: 0 : 1).

inputMin - float with modulation (def: 0, range: 0 : 1).

inputMax - float with modulation (def: 1, range: 0 : 1).

outputMin - float with modulation (def: 0, range: 0 : 1).

outputMax - float with modulation (def: 1, range: 0 : 1).


Quantize Button

useQuantize - switch with modulation (def: 0).

quantizeSteps - integer with modulation (def: 4, range: 2 : 128).


Custom Curves Button

customCurve - drawable curve (size: 256 points).

customCurve2 - drawable curve (size: 256 points).
