TXV Shape 2D

Properties: DrawSource

See also: TXV System


Introduction:

This module draws a 2D shape selected from various presets. The shape can be drawn as an outline, filled or both.

The preset shape names show which parameters are relevent for that shape.


Controls:


Shape Button

shapeType - index with modulation (def: 0, range: 0 : 10) options: [0]_Circle (width, circleResolution), [1]_Ellipse (width, height, circleResolution), [2]_Square (width), [3]_Rectangle (width, height), [4]_Round Rectangle (width, height, roundness, circleResolution), [5]_Star (width, height, circleResolution), [6]_Triangle 1 (Isosceles) (width, height), [7]_Triangle 2 (right) (width, height), [8]_Diamond (width, height), [9]_Crescent (width, height, thickness, circleResolution), [10]_Ring (width, thickness, circleResolution).

lineWidth - integer with modulation (def: 2, range: 0 : 10).

fillShape - switch with modulation (def: 1).

width - float with modulation (def: 0.05, range: 0 : 1).

height - float with modulation (def: 0.1, range: 0 : 1).

useMaxWidthToScaleHeight - switch (def: 0).

roundness - float with modulation (def: 0.2, range: 0 : 1).

thickness - float with modulation (def: 0.2, range: 0 : 1).

circleResolution - integer with modulation (def: 32, range: 3 : 1024).


Fill Color Button

colorHue - float with modulation (def: 0.5, range: 0 : 1).

colorHueRotate - float with modulation (def: 0, range: 0 : 1).

colorSaturation - float with modulation (def: 0.6, range: 0 : 1).

colorBrightness - float with modulation (def: 0.5, range: 0 : 1).

colorAlpha - float with modulation (def: 1, range: 0 : 1).

colorHSBAsRGB - switch with modulation (def: 0).


Outline Color Button

useFillColorForLineColor - switch with modulation (def: 0).

swapFillColorAndLineColor - switch with modulation (def: 0).

lineColorHue - float with modulation (def: 0.5, range: 0 : 1).

lineColorHueRotate - float with modulation (def: 0, range: 0 : 1).

lineColorSaturation - float with modulation (def: 0.6, range: 0 : 1).

lineColorBrightness - float with modulation (def: 0.5, range: 0 : 1).

lineColorAlpha - float with modulation (def: 1, range: 0 : 1).

lineColorHSBAsRGB - switch with modulation (def: 0).


Draw :: Active & Max Size Button

drawActive - switch with modulation (def: 1).

maxWidthHeightRule - index (def: 0, range: 0 : 5) options: [0]_Use Screen Width & Height, [1]_Square - Use Max of Screen Width & Height, [2]_Square - Use Min of Screen Width & Height, [3]_Square - Use Screen Width, [4]_Square - Use Screen Height, [5]_Use Custom Size.

customMaxWidth - integer (def: 1024, range: 1 : 4096).

customMaxHeight - integer (def: 768, range: 1 : 4096).


Draw :: Scale Button

scaleX - float with modulation (def: 1, range: -10 : 10).

scaleY - float with modulation (def: 1, range: -10 : 10).

useScaleXForScaleY - switch (def: 0).


Draw :: Position, Anchor, Rotate Button

positionX - float with modulation (def: 0.5, range: 0 : 1).

positionY - float with modulation (def: 0.5, range: 0 : 1).

rotate - float with modulation (def: 0, range: -360 : 360).

rotateMultiply - float with modulation (def: 1, range: 0 : 10).

anchorX - float with modulation (def: 0.5, range: 0 : 1).

anchorY - float with modulation (def: 0.5, range: 0 : 1).
