TXV Curves

Properties: DrawSource

See also: TXV System & TX_Points Curve Editor


Introduction:

This module creates multiple curves, with each curve created by connecting points together. Optionally, the curves can be filled, so there are separate line and fill colours.

Depending on the type of curve used, there are also control points which can be used to shape the curve. Curve types include: Automatic curve, Straight line, Bezier curve (with 1 or 2 control points).

Variations between multiple curves are created by interpollating between start and end curves - points, line and fill colours all have optional end values.

Position and color spread curves are used to define where the parameter values for each particular curve will be inbetween their start and end values.


Controls:


Global Button

curveType - index with modulation (def: 0, range: 0 : 4) options: [0]_Automatic curve, [1]_Straight line, [2]_Bezier curve using control point 1, [3]_Bezier curve using control point 1, smoothed, [4]_Bezier curve using control points 1 & 2.

numPoints - integer with modulation (def: 8, range: 2 : 8).

numCurves - integer with modulation (def: 12, range: 0 : 1000).

ifSingleIgnoreEndValues - switch with modulation (def: 0). This switch only matters if numCurves is set to 1 so a single curve is being displayed. Normally, the curve would be a mix between the start and end curves, but if this switch is on, only the start curve is used.
This is useful if you just need to draw a single curve - if you were for example modulating the number of curves from 1 to many, it would look more natural if this switch was off.

lineWidth - integer with modulation (def: 4, range: 0 : 10).

autoClose - switch with modulation (def: 0).

fillShape - switch with modulation (def: 0).

windingMode - index with modulation (def: 0, range: 0 : 1) options: [0]_Odd - allows holes (default), [1]_Non-Zero - outline, no holes, autoClose.

curveResolution - integer with modulation (def: 64, range: 2 : 1024).

reverseCurveOrder - switch with modulation (def: 0).

drawControlPoints - switch with modulation (def: 0).
