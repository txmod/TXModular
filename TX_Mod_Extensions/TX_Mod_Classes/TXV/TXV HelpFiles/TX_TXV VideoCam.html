TXV VideoCam

Properties: DrawSource

See also: TXV System


Introduction:

This module draws an image taken from a video camera.

Note - When the sample width and height are maximum (which they are by default) the sample area is the whole image, in which case samplePosX and samplePosY will have no effect. But when the sample area is smaller than the whole image, samplePosX and samplePosY move the sample area around within the whole image.


Controls:


Camera & Sample Area Button

videoCamActive - switch (def: 1).

videoDeviceIndex - integer (def: 0, range: 0 : 10).

cameraViewWidth - integer (def: 640, range: 10 : 5000).

cameraViewHeight - integer (def: 480, range: 10 : 5000).

samplePosX - float with modulation (def: 0, range: 0 : 1).

samplePosY - float with modulation (def: 0, range: 0 : 1).

sampleWidth - float with modulation (def: 1, range: 0 : 1).

sampleHeight - float with modulation (def: 1, range: 0 : 1).


Draw :: Active & Alpha Button

drawActive - switch with modulation (def: 1).

alphaThreshold - float (def: 0.03, range: 0 : 0.05).

alpha - float with modulation (def: 1, range: 0 : 1).


Draw :: Size Button

maxWidthHeightRule - index (def: 0, range: 0 : 5) options: [0]_Use Screen Width & Height, [1]_Square - Use Max of Screen Width & Height, [2]_Square - Use Min of Screen Width & Height, [3]_Square - Use Screen Width, [4]_Square - Use Screen Height, [5]_Use Custom Size.

customMaxWidth - integer (def: 1024, range: 1 : 4096).

customMaxHeight - integer (def: 768, range: 1 : 4096).

customMaxDepth - integer (def: 1024, range: 1 : 4096).

useSampleSizeForDrawSize - switch (def: 0).

constrainToEdges - switch (def: 0).

drawWidth - float with modulation (def: 1, range: 0 : 1).

drawHeight - float with modulation (def: 1, range: 0 : 1).


Draw :: Scale Button

useScaleXForScaleY - switch (def: 0).

scaleX - float with modulation (def: 1, range: -10 : 10).

scaleY - float with modulation (def: 1, range: -10 : 10).


Draw :: Position, Anchor & Rotate Button

useSamplePosForDrawPos - switch (def: 0).

drawPosX - float with modulation (def: 0.5, range: 0 : 1).

drawPosY - float with modulation (def: 0.5, range: 0 : 1).

anchorX - float with modulation (def: 0.5, range: 0 : 1).

anchorY - float with modulation (def: 0.5, range: 0 : 1).

rotate - float with modulation (def: 0, range: -360 : 360).

rotateMultiply - float with modulation (def: 1, range: 0 : 10).
