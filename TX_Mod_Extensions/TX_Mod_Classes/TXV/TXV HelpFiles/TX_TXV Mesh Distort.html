TXV Mesh Distort

Properties: DrawSource DrawDestination Renders

See also: TXV System


Introduction:

Note: Unfinished module!

This module is an unfinished experimental module that renders a 3D scene with 3 lights and a camera, adding position distortion to anything drawn to it.

Four waveforms, including Perlin noise, can be used to distort the X/Y/Z position of every point in every mesh drawn.

Distortion can be applied before or after the transform matrix that adds any scaling, rotation or position offset (using the switch distortAfterMatrix).

Bugs/ Issues:
If distortAfterMatrix is on, Z-axis distortion makes holes in meshes.
Light and camera position parameters are measured in absolute pixels rather than units relative to the screen size, so if the screen size changes, the positioning will be different.
