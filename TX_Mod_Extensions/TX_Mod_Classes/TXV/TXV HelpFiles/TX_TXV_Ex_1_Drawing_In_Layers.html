TXV Ex 1 Drawing In Layers


See also: TXV System & TXV Drawing In Layers


Introduction

This help file uses an example file "Ex_1_Drawing_In_Layers" found in the folder "TXV_Examples".

Please open the example in the TX Modular system.

In order to start the TXV software, you can use the “Start TXV” button if you are using the standalone TX Modular (for MacOS) which includes TXV inside it. If you are using another platform, you will need to have a TXV build ready for your platform and open it separately.

Once TXV has been started, press the button “Rebuild TXV System” which will send all the data from the TX Modular to TXV.

You should then see a blue-green background with a red triangle on it, and a yellow oblong over the red triangle.


TXV Modules Used

If you click on the button “Modules + Channels”, you can see in the left-hand column all the TXV modules used. Apart from the TXV System itself, there is a Gradient, two Shape 2Ds, a Transform and four DrawConnections.

The Gradient creates the blue-green background. The two Shape 2Ds are the red triangle and yellow oblong.

The Transform module is used to group together the two Shape 2Ds, which allows them to be “transformed” by shifting them along the X or Y axis, scaling them or rotating them.

The 4 DrawConnections can be understood by opening the TXV System module and clicking on the button “Draw Connections”.

This shows that the Gradient is drawn first to layer 1 of the TXV System (i.e. the main TXV window). Then the two Shape 2Ds are drawn to layers 1 & 2 of the Transform. Finally, the Transform is drawn to layer 2 of the TXV System.

Because layers are drawn in numerical order, the Gradient is drawn first on the main window. Then the Transform is drawn which itself draws the two Shape 2Ds, with the red triangle drawn before the yellow oblong.

One thing to note is that if you click on the button "Draw & Background" in the TXV System, you will see a parameter "drawLayersRule". This selects which layers will actually be drawn on the main window. By default, it is set to "All layers except Layer 20". Keep the default setting for this example.


Changing layers and other parameters

If you click on the button “Run Interface”, you will see some GUI widgets which allow you to easily change how things look.

First, try setting “Draw Connection [2] - layerNo” to 3. This will mean that the red triangle is now drawn after the yellow oblong and will appear in front of it.

Next, change the rotation of the triangle and oblong and they will rotate relative to one another.

Next change “Transform [1] - rotateZ”. This will add further rotation to the two Shape 2Ds treating them as a group because they are both drawn onto the Transform [1].

Finally change “Transform [1] - shiftX” and you will see both Shape 2Ds move left or right.
