TXV Assets


See also: TXV System


Assets are used by many modules and are added directly to the TXV System module. Each module in TXV has a set of properties which include the type of any assets they use - image, movie, svg, cube, 3D model, font
(TXV List Of All Modules shows the properties for all modules.)

Assets represent files of the following types:

image - images are used by many modules in TXV including TXV Image and all Image FX modules.

movie - used by TXV Movie

svg - used by TXV SVG

cube - used by TXV LUT

3D model - used by TXV 3D Model

font - used by TXV Text


Note: An alternative way to use images is by using TXV Slide Show to display images that are in a folder. This allows lots of images to be used without having to add them all as Image Assets.
