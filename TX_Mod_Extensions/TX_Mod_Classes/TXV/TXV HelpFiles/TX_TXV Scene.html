TXV Scene

Properties: DrawSource DrawDestination

See also: TXV System


Introduction:

This module is a 3D Draw Destination that renders an image from anything drawn to it.

3 lights and 2 cameras can be used to create the scene captured.


Controls:


Global Button

useLighting - switch with modulation (def: 1).

smoothLighting - switch (def: 1).

drawLights - switch (def: 0).

cameraSelect - index with modulation (def: 0, range: 0 : 2) options: [0]_Front View, [1]_Camera 1, [2]_Camera 2.


Light 1 :: Type Button

light1Active - switch with modulation (def: 1).

light1Type - index with modulation (def: 0, range: 0 : 2) options: [0]_Point, [1]_Directional, [2]_Spot.

light1SpotConcentration - integer with modulation (def: 64, range: 0 : 128).

light1SpotlightAngle - float with modulation (def: 0, range: 0 : 90).


Light 1 :: Position, Orientation Button

light1PositionX - float with modulation (def: 0, range: -10 : 10).

light1PositionY - float with modulation (def: 0, range: -10 : 10).

light1PositionZ - float with modulation (def: 0, range: -10 : 10).

light1OrientationY - float with modulation (def: 0, range: -360 : 360).

light1OrientationZ - float with modulation (def: 0, range: -360 : 360).

light1DiffuseColorBrightness - float with modulation (def: 0.4, range: 0 : 1).

light1DiffuseColorHSBAsRGB - switch with modulation (def: 0).

light1AmbientColorHueRotate - float with modulation (def: 0, range: 0 : 1).

maxDepthRule - index (def: 0, range: 0 : 4) options: [0]_Use Screen Width, [1]_Use Screen Height, [2]_Use Max of Screen Width & Height, [3]_Use Min of Screen Width & Height, [4]_Use Custom Size.

customMaxHeight - integer (def: 768, range: 1 : 4096).

drawLayersRule - index with modulation (def: 0, range: 0 : 26) options: [0]_All layers except Layer 20, [1]_Layer 1, [2]_Layer 2, [3]_Layer 3, [4]_Layer 4, [5]_Layer 5, [6]_Layer 6, [7]_Layer 7, [8]_Layer 8, [9]_Layer 9, [10]_Layer 10, [11]_Layer 11, [12]_Layer 12, [13]_Layer 13, [14]_Layer 14, [15]_Layer 15, [16]_Layer 16, [17]_Layer 17, [18]_Layer 18, [19]_Layer 19, [20]_Layer 20, [21]_Layers 1 - 5, [22]_Layers 6 - 10, [23]_Layers 11 - 15, [24]_Layers 16 - 20, [25]_Layers 1 - 10, [26]_Layers 11 - 20.
