TXV Feedback

Properties: DrawSource DrawDestination Renders

See also: TXV System


Introduction:

This module is a Draw Destination which renders an image by drawing the previous frame behind the current one with optional shift, scale and rotate.


Controls:


Freeze, Alpha, Brightness Button

freezeFeedback - switch with modulation (def: 0).

feedbackAlpha - float with modulation (def: 0.5, range: 0 : 1).

feedbackBrightness - float with modulation (def: 1, range: 0 : 1).


Shift & Scale Button

feedbackShiftX - float with modulation (def: 0, range: -100 : 100).

quantiseFeedbackShiftX - switch with modulation (def: 0).

feedbackShiftY - float with modulation (def: 0, range: -100 : 100).

quantiseFeedbackShiftY - switch with modulation (def: 0).

feedbackScaleX - float with modulation (def: 1, range: -10 : 10).

feedbackScaleY - float with modulation (def: 1, range: -10 : 10).

useFeedbackScaleXForScaleY - switch (def: 0).


Anchor & Rotate Button

feedbackAnchorX - float with modulation (def: 0.5, range: 0 : 1).

quantiseFeedbackAnchorX - switch with modulation (def: 0).

feedbackAnchorY - float with modulation (def: 0.5, range: 0 : 1).

quantiseFeedbackAnchorY - switch with modulation (def: 0).

feedbackRotate - float with modulation (def: 0, range: -360 : 360).

feedbackRotateMultiply - float with modulation (def: 1, range: 0 : 10).


Clear & Render :: Clear Button

clearNow - switch with modulation (def: 0).

clearColorHue - float with modulation (def: 0.5, range: 0 : 1).

clearColorSaturation - float with modulation (def: 0.6, range: 0 : 1).

clearColorBrightness - float with modulation (def: 0, range: 0 : 1).

clearColorAlpha - float with modulation (def: 1, range: 0 : 1).

clearColorHSBAsRGB - switch (def: 0).


Clear & Render :: Render Button

drawLayersRule - index with modulation (def: 0, range: 0 : 26) options: [0]_All layers except Layer 20, [1]_Layer 1, [2]_Layer 2, [3]_Layer 3, [4]_Layer 4, [5]_Layer 5, [6]_Layer 6, [7]_Layer 7, [8]_Layer 8, [9]_Layer 9, [10]_Layer 10, [11]_Layer 11, [12]_Layer 12, [13]_Layer 13, [14]_Layer 14, [15]_Layer 15, [16]_Layer 16, [17]_Layer 17, [18]_Layer 18, [19]_Layer 19, [20]_Layer 20, [21]_Layers 1 - 5, [22]_Layers 6 - 10, [23]_Layers 11 - 15, [24]_Layers 16 - 20, [25]_Layers 1 - 10, [26]_Layers 11 - 20.

renderContinuosly - switch with modulation (def: 1).

renderNow - switch with modulation (def: 0).


Draw :: Active, Size, Alpha Button

drawActive - switch with modulation (def: 1).

maxWidthHeightRule - index (def: 0, range: 0 : 5) options: [0]_Use Screen Width & Height, [1]_Square - Use Max of Screen Width & Height, [2]_Square - Use Min of Screen Width & Height, [3]_Square - Use Screen Width, [4]_Square - Use Screen Height, [5]_Use Custom Size.

customMaxWidth - integer (def: 1024, range: 1 : 4096).

customMaxHeight - integer (def: 768, range: 1 : 4096).

drawAlpha - float with modulation (def: 1, range: 0 : 1).

drawWidth - float with modulation (def: 1, range: 0 : 1).

drawHeight - float with modulation (def: 1, range: 0 : 1).


Draw :: Scale Button

scaleX - float with modulation (def: 1, range: -10 : 10).

scaleY - float with modulation (def: 1, range: -10 : 10).

useScaleXForScaleY - switch (def: 0).


Draw :: Position, Anchor, Rotate Button

drawPosX - float with modulation (def: 0.5, range: 0 : 1).

drawPosY - float with modulation (def: 0.5, range: 0 : 1).

anchorX - float with modulation (def: 0.5, range: 0 : 1).

anchorY - float with modulation (def: 0.5, range: 0 : 1).

rotate - float with modulation (def: 0, range: -360 : 360).

rotateMultiply - float with modulation (def: 1, range: 0 : 10).
