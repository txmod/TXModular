TXV Perlin

Properties: ModulationSource

See also: TXV System


Introduction:

This module outputs Perlin noise (a kind of smoothed random noise) as a modulation signal.


Controls:

perlinType - index with modulation (def: 0, range: 0 : 1) options: [0]_Normal, [1]_Quantised.

speed - float with modulation (def: 1, range: 0.001 : 100).

useMasterBpm - switch with modulation (def: 1).

speedDivisor - float with modulation (def: 1, range: 0.01 : 1000).

frozen - switch with modulation (def: 0).


Modulation Outputs:

out


Modulation Inputs:

perlinType

speed

useMasterBpm

speedDivisor

frozen
