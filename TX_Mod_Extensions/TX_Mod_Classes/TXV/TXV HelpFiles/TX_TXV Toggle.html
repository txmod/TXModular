TXV Toggle

Properties: ModulationSource

See also: TXV System


Introduction:

This module outputs a switched modulation signal which is either 0 or 1.

When the input value passes the trigger threshold, the output will toggle to it's opposite value - 0 becomes 1, or 1 becomes 0.


Controls:

inputValue - float with modulation (def: 0, range: 0 : 1).

triggerThreshold - float with modulation (def: 0, range: 0 : 1).

resetToZero - switch with modulation (def: 0).


Modulation Outputs:

out


Modulation Inputs:

inputValue

triggerThreshold
