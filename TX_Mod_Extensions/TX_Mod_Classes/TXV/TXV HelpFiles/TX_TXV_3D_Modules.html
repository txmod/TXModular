Extra Consideration When Drawing In 3D

3D Draw sources need to be drawn to a TXV Scene for their material image and colors to work properly

TXV Scene includes 3 lights with various colors. See TXV 3D Lights And Materials for more details.

The alternative is draw them to a TXV EnvMap. However, EnvMap will ignore the shape's material and use its own base color with a simulated lighting effect, as well as simulated reflections from an environment image, to create the output image.
