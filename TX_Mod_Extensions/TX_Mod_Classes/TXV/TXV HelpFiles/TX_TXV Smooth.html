TXV Smooth

Properties: ModulationSource

See also: TXV System


Introduction:

This module outputs a smoothed signal. By modulating the targetValue with an input signal, the output will be a smoothed version of the input.


Controls:

smoothType - index with modulation (def: 0, range: 0 : 1) options: [0]_Exponential, [1]_Linear.

targetValue - float with modulation (def: 0, range: 0 : 1).

smoothTime - float with modulation (def: 1, range: 0 : 60).

circularSmoothing - switch with modulation (def: 0).


Modulation Outputs:

out


Modulation Inputs:

smoothType

targetValue

smoothTime

circularSmoothing
