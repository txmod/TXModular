See also: TXV System


Introduction:

This module generates a pattern chosen from a list of presets.

Note: these patterns are created by using modified versions of GLSL shader code downloaded from www.shadertoy.com and github.com/patriciogonzalezvivo/sNodes.

Most of the patterns can change over time, with manual or automatic control of the speed of movement.

Each pattern has particular parameters that can be changed (such as point X & Y, zoom, BPM & colours 1-4). The pattern name shows which paramenters are used.


Controls:


Generator Button

generatorType - index with modulation (def: 2, range: 0 : 21) options: [0]_70s Pattern- uses zoom, [1]_Caramel Clouds, [2]_Clouds, [3]_Cosmos, [4]_Dark Cloud, [5]_Dense Bubbles - uses color1, [6]_Dense Cloud - uses point, zoom, color1, [7]_Fire-Ball - uses point, zoom, color1, [8]_Flow - uses zoom, [9]_Fog - uses zoom, color1-4, [10]_Fog 2 - uses zoom, color1-4, [11]_Hypnotic Color, [12]_Mountains Texture - uses zoom, color1, [13]_Noise - uses zoom, color1-4, [14]_Particles, [15]_Pattern 1 - uses zoom, color1, [16]_Pattern 2 - uses zoom, color1, [17]_Snow, [18]_Space Warp, [19]_Voronoi - uses zoom, color1-3, [20]_Voronoi Static - uses point, zoom, color1, [21]_Waves Cascade.

pointY - float with modulation (def: 0.5, range: 0 : 1000).

zoom - float with modulation (def: 1, range: 0 : 100).
