TXV Choosing Colours


See also: TXV System


Introduction

Colours are used in many TXV modules and the controls are all very similar.

Colours can be chosen using Hue, Saturation, Brightness and Alpha (transparency) channels.

Alternatively, all colours have a switch marked HSBAsRGB, which treats the Hue, Saturation, Brightness channels as Red, Green & Blue channels.

In some modules, there is an extra colour control: HueRotation. This can be used to offset the hue, for example if you need to select a particular range of hues.

Note HueRotation will have no effect if HSBAsRGB is switched on.

In 3D Draw Source modules which use a material for their colours, the Alpha value will automatically be set to 1 (fully opaque). This is because transparency has no effect in 3D (since depth testing, which decides which 3D objects will displayed, overrides transparency).
