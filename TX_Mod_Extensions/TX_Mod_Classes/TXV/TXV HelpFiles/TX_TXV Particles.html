TXV Particles

Properties: DrawSource ImageAsset

See also: TXV System


Introduction:

This module draws particles which are 2D shapes based on masked rectangles taken from an image.

Automatic or manual triggering creates new particles, each with a fixed lifetime.

Particles can move in various ways and change size or colour during their lifetimes.

Several parameters have an associated curve (for example, the brush size has width and widthCurve, height and heightCurve).
The curve is used to scale the parameter over it's lifetime.
By default the curves are set to maximum so the parameter remains unchanged, but they can be edited to make a parameter grow or shrink during the particle's lifetime.
