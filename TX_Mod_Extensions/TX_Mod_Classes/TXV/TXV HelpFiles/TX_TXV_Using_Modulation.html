TXV Using Modulation


See also: TXV System


Introduction

Within TXV, some modules are Modulation Sources including LFOs, Step Sequencers and TXV System which has various mouse property modulation outputs. (In the module properties at the top of each module's help file, Modulation Sources are shown).

See TXV List Of All Modules & TXV Intro To All Modules for the properties of all modules, especially the group Modulation Sources.


Internal vs. External Modulation

Modulation works slightly differently in TXV compared to the TX Modular.

Internal modulation is where a TXV modulation source modulates another TXV module's parameter using a TXV Node module to make the connection. For example, an LFO might be modulating the vertical position of a 2D shape on the screen.

The Node sets the amount of modulation and can also invert the signal.

In order for Internal modulation to work, the parameter to be modulated should have "modulation mix" set above zero (it is zero by default). It should also have "use external modulation" switched off which is the default value.

External modulation is where a TX Modular modulation source modulates a TXV module parameter using a Channel. This allows, for example, Audio Analysis signals in the TX Modular to change TXV parameters for "visualiser"-type effects.

In order for External modulation to work, the parameter to be modulated should have "use external modulation" switched on.


Note: Modulation with multiple sources is also calculated differently in TXV compared to the TX Modular.

In TXV, if multiple sources modulate the same destination, their combined signals will automatically be scaled to the appropriate range of values in the destination parameter.

So for example, 3 different modulation sources added together with their modulation Nodes at maximum modulation will automatically be scaled to the destination parameter range, and will not "hit the buffers" at the min & max values.

(This auto-scaling doesn't happen in the TX Modular unless the modulation sources are first mixed into a Mix Control 8-1 or Matrix C 8x8 module with Auto-Scale switched on).

There is one case which is different: if the TXV destination parameter is a switch with multiple modulation sources connected to it, there is no auto-scaling so that any one of the individual modulation sources are able to turn the switch on.
