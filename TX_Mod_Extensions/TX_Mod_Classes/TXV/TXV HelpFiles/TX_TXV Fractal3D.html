TXV Fractal3D

Properties: DrawSource DrawDestination

See also: TXV System


Introduction:

This module is a 3D Draw Destination which draws multiple transformed copies of what is drawn to it using a fractal branching model.

Up to 10 branches can be used, each with up to 10 branching levels. Each branch can draw a specific draw layer (from 20 layers) and add scaling, rotation and position offsets.


Controls:


Main Button

drawActive - switch with modulation (def: 1).

maxWidthHeightRule - index (def: 0, range: 0 : 5) options: [0]_Use Screen Width & Height, [1]_Square - Use Max of Screen Width & Height, [2]_Square - Use Min of Screen Width & Height, [3]_Square - Use Screen Width, [4]_Square - Use Screen Height, [5]_Use Custom Size.

maxDepthRule - index (def: 0, range: 0 : 4) options: [0]_Use Screen Width, [1]_Use Screen Height, [2]_Use Max of Screen Width & Height, [3]_Use Min of Screen Width & Height, [4]_Use Custom Size.

customMaxWidth - integer (def: 1024, range: 1 : 4096).

customMaxHeight - integer (def: 768, range: 1 : 4096).

customMaxDepth - integer (def: 1024, range: 1 : 4096).

fractalTreeLevels - integer with modulation (def: 2, range: 1 : 10).

fractalTreeBranches - integer with modulation (def: 2, range: 1 : 10).

cellWidth - float with modulation (def: 0.2, range: 0 : 1).

cellHeight - float with modulation (def: 0.2, range: 0 : 1).

cellDepth - float with modulation (def: 0.2, range: 0 : 1).

randomSeed - integer with modulation (def: 100, range: 0 : 10000).


Branch 1 :: Options & Probability Button

branch01_branchIfTopLevel - switch with modulation (def: 1).

branch01_drawBottomLevelOnly - switch with modulation (def: 0).

branch01_branchRule - index with modulation (def: 0, range: 0 : 4) options: [0]_Always branch, [1]_Branch if Level is Greater than Comparator, [2]_Branch if Level is Less than Comparator, [3]_Branch if Level is Equal to Comparator, [4]_Branch if Level is Not Equal to Comparator.

branch01_branchRuleComparator - integer with modulation (def: 1, range: 1 : 10).

branch01_drawLayersRule - index with modulation (def: 0, range: 0 : 26) options: [0]_All layers except Layer 20, [1]_Layer 1, [2]_Layer 2, [3]_Layer 3, [4]_Layer 4, [5]_Layer 5, [6]_Layer 6, [7]_Layer 7, [8]_Layer 8, [9]_Layer 9, [10]_Layer 10, [11]_Layer 11, [12]_Layer 12, [13]_Layer 13, [14]_Layer 14, [15]_Layer 15, [16]_Layer 16, [17]_Layer 17, [18]_Layer 18, [19]_Layer 19, [20]_Layer 20, [21]_Layers 1 - 5, [22]_Layers 6 - 10, [23]_Layers 11 - 15, [24]_Layers 16 - 20, [25]_Layers 1 - 10, [26]_Layers 11 - 20.

branch01_branchProbability - float with modulation (def: 1, range: 0 : 1).

branch01_subBranchProbability - float with modulation (def: 1, range: 0 : 1).


Branch 1 :: Offset & Anchor Button

branch01_applyOffsetLast - switch with modulation (def: 1).

branch01_offsetX - float with modulation (def: 0, range: -10 : 10).

branch01_offsetY - float with modulation (def: 0, range: -10 : 10).

branch01_offsetZ - float with modulation (def: 0, range: -10 : 10).

branch01_anchorX - float with modulation (def: 0.5, range: 0 : 1).

branch01_anchorY - float with modulation (def: 0.5, range: 0 : 1).

branch01_anchorZ - float with modulation (def: 0, range: 0 : 1).


Branch 1 :: Scale Button

branch01_scaleX - float with modulation (def: 1, range: 0 : 2).

branch01_scaleXLevelDelta - float with modulation (def: 0, range: -1 : 1).

branch01_scaleXRandom - float with modulation (def: 0, range: 0 : 1).

branch01_scaleY - float with modulation (def: 1, range: 0 : 2).

branch01_scaleYLevelDelta - float with modulation (def: 0, range: -1 : 1).

branch01_scaleYRandom - float with modulation (def: 0, range: 0 : 1).

branch01_scaleZ - float with modulation (def: 1, range: 0 : 2).

branch01_scaleZLevelDelta - float with modulation (def: 0, range: -1 : 1).

branch01_scaleZRandom - float with modulation (def: 0, range: 0 : 1).


Branch 1 :: Rotate Button

branch01_rotateAxisX - float with modulation (def: 0, range: -1 : 1).

branch01_rotateAxisY - float with modulation (def: 0, range: -1 : 1).

branch01_rotateAxisZ - float with modulation (def: 1, range: -1 : 1).

branch01_angle - float with modulation (def: 0, range: -720 : 720).

branch01_angleLevelDelta - float with modulation (def: 0, range: -180 : 180).

branch01_angleRandom - float with modulation (def: 0, range: 0 : 1).


Branches 2 - 10 are the same as Branch 1
