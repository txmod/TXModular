TXV LFO

Properties: ModulationSource

See also: TXV System


Introduction:

This module is a "Low Frequency Oscillator" (LFO) which outputs a modulation signal from a preset or custom waveform curve.


Controls:


Main Button

lfoType - index with modulation (def: 0, range: 0 : 17) options: [0]_Sine, [1]_Cosine, [2]_Square, [3]_Pulse, [4]_Triangle, [5]_Saw, [6]_Saw Reversed, [7]_Random, [8]_Random Quantised, [9]_Random Smoothed, [10]_Double Random, [11]_Double Random Quantised, [12]_Double Random Smoothed, [13]_Trefoil Knot X, [14]_Trefoil Knot Y, [15]_Trefoil Knot Z, [16]_Custom Curve, [17]_Custom Curve - Mirrored.

randomSmoothType - index with modulation (def: 23, range: 0 : 23) options: [0]_Cubic In, [1]_Cubic Out, [2]_Cubic In & Out, [3]_Elastic In, [4]_Elastic Out, [5]_Elastic In & Out, [6]_Exponential In, [7]_Exponential Out, [8]_Exponential In & Out, [9]_Linear In (default), [10]_Linear Out, [11]_Linear In & Out, [12]_Quadratic In, [13]_Quadratic Out, [14]_Quadratic In & Out, [15]_Quartic In, [16]_Quartic Out, [17]_Quartic In & Out, [18]_Quintic In, [19]_Quintic Out, [20]_Quintic In & Out, [21]_Sine In, [22]_Sine Out, [23]_Sine In & Out.

pulseWidth - float with modulation (def: 0.5, range: 0 : 1).


BPM, Cycle, Phase Button

bpm - float with modulation (def: 120, range: 0.001 : 1000).

useMasterBpm - switch with modulation (def: 1).

cycleTimeBeats - float with modulation (def: 1, range: 0.01 : 1000).

cycleTimeDivisor - float with modulation (def: 1, range: 0.01 : 1000).

phaseOffset - float with modulation (def: 0, range: 0 : 1).

frozen - switch with modulation (def: 0).

resetClock - switch with modulation (def: 0).


Custom Curve Button

customCurve - drawable curve (size: 256 points).

customCurve2 - drawable curve (size: 256 points).

customCurveMorph - float with modulation (def: 0, range: 0 : 1).
