TXV Color Sampler

Properties: ModulationSource ImageAsset

See also: TXV System


Introduction:

This module reads the color of 4 sample positions from an image, and outputs separate modulation signals for each R/G/B/H/S/B/A color channel.


Controls:


Image Button

sourceImageAsset - index (def: 0) - Image Asset.

extSourceImageModule - index (def: 0) - TXV module to import image from.

useExternalSourceImage - switch (def: 0).


Sample 1 Button

sample1Active - switch with modulation (def: 1).

sample1PosX - float with modulation (def: 0, range: 0 : 1).

sample1PosY - float with modulation (def: 0, range: 0 : 1).

color1SmoothTime - float with modulation (def: 0, range: 0 : 60).


Sample 2 Button

sample2Active - switch with modulation (def: 1).

sample2PosX - float with modulation (def: 0, range: 0 : 1).

sample2PosY - float with modulation (def: 0, range: 0 : 1).

color2SmoothTime - float with modulation (def: 0, range: 0 : 60).


Sample 3 Button

sample3Active - switch with modulation (def: 1).

sample3PosX - float with modulation (def: 0, range: 0 : 1).

sample3PosY - float with modulation (def: 0, range: 0 : 1).

color3SmoothTime - float with modulation (def: 0, range: 0 : 60).


Sample 4 Button

sample4Active - switch with modulation (def: 1).

sample4PosX - float with modulation (def: 0, range: 0 : 1).

sample4PosY - float with modulation (def: 0, range: 0 : 1).

color4SmoothTime - float with modulation (def: 0, range: 0 : 60).
